# 1. 목적

본 절차서는 M&C Electronics VINA의 생산 및 서비스 제공이 관리된 조건(Controlled Conditions) 하에서 수행되도록 하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.5.1 요구사항에 따른 생산/서비스 관리 조건 수립
- 제품/서비스의 적합성을 보장하기 위한 관리 체계 확립
- 특수공정의 검증 및 관리
- 항공우주 산업 생산 관리 요구사항 충족

---

본 절차서는 M&C Electronics VINA의 모든 생산/제조 활동 및 관련 서비스 제공에 적용한다.

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 관리 조건 (Controlled Conditions) | 제품/서비스의 적합성을 보장하기 위해 관리되는 생산 환경, 절차, 자원의 조건 |
| 특수공정 (Special Process) | 결과 출력을 후속 검사/시험으로 완전히 검증할 수 없는 생산 공정 |
| 공정 승인 (Process Approval) | 특수공정이 일관되게 적합한 결과를 생산할 수 있음을 입증하는 활동 |
| 작업표준서 (Work Instruction) | 작업자가 공정을 수행하는 데 필요한 상세 지침 |
| 공정능력 (Process Capability) | 공정이 규격 한계 내에서 제품을 생산할 수 있는 통계적 능력 (Cpk) |
| 생산 계획 (Production Plan) | 제품 실현을 위한 공정 순서, 자원, 일정의 계획 |
| 관리 계획서 (Control Plan) | 공정별 관리 항목, 관리 방법, 측정 빈도를 규정한 문서 |
| NADCAP | 항공우주 특수공정 인가 프로그램 (National Aerospace and Defense Contractors Accreditation Program) |

# 6. 생산 관리 체계

## 6.1 생산 계획

| 계획 항목 | 내용 |
|----------|------|
| 공정 순서 (Routing) | 원자재 투입부터 완제품까지의 공정 흐름 |
| 작업 배분 | 공정별 작업자 배치 |
| 일정 계획 | 착수일, 공정별 소요시간, 완료 예정일 |
| 자원 계획 | 설비, 공구, 자재, 치공구 |
| 검사 계획 | 공정 내 검사 포인트, 검사 방법, 빈도 |

## 6.2 생산 실행 흐름

```
자재 입고 (수입검사 합격)
    │
    ▼
자재 불출 ──→ 로트/배치 추적성 부여 (QP-812)
    │
    ▼
공정 1 (가공/성형 등) ──→ 자주검사 + 공정검사
    │
    ▼
공정 2 (특수공정 해당 시) ──→ 공정 파라미터 관리
    │
    ▼
공정 N (조립/배선 등) ──→ 자주검사 + 공정검사
    │
    ▼
최종검사 (QP-817) ──→ 합격 판정
    │
    ▼
포장/출하 (QP-814)
```

# 7. 관리 조건 (Controlled Conditions)

AS9100D 8.5.1에 따라 다음 관리 조건을 수립하고 유지한다:

| 관리 조건 | 내용 | 관련 절차/문서 |
|----------|------|--------------|
| **a) 문서화된 정보** | 제품 특성, 수행할 활동, 달성할 결과를 규정하는 문서 | 도면, 사양서, 작업표준서(WI) |
| **b) 모니터링/측정 자원** | 적합한 측정장비의 가용성 및 사용 | QP-702 |
| **c) 모니터링/측정 활동** | 적절한 단계에서의 검사/시험 | 관리 계획서, QP-817 |
| **d) 인프라/업무환경** | 프로세스 운영에 적합한 시설/환경 | QP-701 |
| **e) 자격 있는 인원** | 필요 역량을 보유한 작업자 배치 | QP-703 |
| **f) 특수공정 검증/재검증** | 특수공정의 승인 및 주기적 재검증 | 8절 참조 |
| **g) 인적 오류 방지** | Poka-Yoke, 체크리스트, 자동 검출 등 | 작업표준서, PFMEA |
| **h) 출시/인도/인도 후 활동** | 제품 출시 기준, 인도 조건 | QP-817, QP-815 |

## 7.1 작업표준서 관리

| 항목 | 기준 |
|------|------|
| 작성 기준 | 작업자가 이해할 수 있는 수준으로 상세 기술 |
| 포함 내용 | 공정명, 사용 장비/공구, 작업 순서, 관리 기준, 검사 방법, 안전 주의사항 |
| 비치 | 작업 현장에 최신 승인본 비치 |
| 교육 | 신규 또는 변경 시 작업자 교육 실시 |

## 7.2 인적 오류 방지 (Human Error Prevention)

AS9100D 8.5.1 g)에 따라:

| 방법 | 적용 |
|------|------|
| Poka-Yoke (방오 장치) | 조립 방향 오류, 부품 누락 방지 |
| 체크리스트 | 작업 순서, 검사 항목 확인 |
| 시각적 관리 | 색상 코딩, 라벨, 경고 표시 |
| 자동 검출 | 센서, 비전 검사, 자동 정지 |
| 2인 확인 (Dual Verification) | 안전 핵심 작업 시 |

# 8. 특수공정 관리

## 8.1 특수공정 정의

결과 출력을 후속 모니터링 또는 측정으로 완전히 검증할 수 없는 공정:

| 특수공정 | 예시 |
|---------|------|
| 열처리 | 경화, 어닐링, 응력 제거 |
| 표면처리/도금 | 양극 산화, 도금, 화성 피막 |
| 용접 | TIG, MIG, 저항 용접, 납땜 |
| NDT (비파괴검사) | UT, RT, MT, PT, ET |
| 접착/밀봉 | 구조 접착, 실링 |
| 도장 | 프라이머, 탑코트 |

## 8.2 특수공정 승인 (Process Qualification)

| 단계 | 활동 | 책임자 |
|------|------|--------|
| 1 | 공정 사양서/절차서 수립 | 기술팀장 |
| 2 | 장비/환경 적합성 확인 | 생산팀장 |
| 3 | 작업자 자격 인증 | 품질팀장 (QP-703 연계) |
| 4 | 공정 시험 (Process Qualification Test) | 기술/품질팀 |
| 5 | 시험 결과 평가 및 공정 승인 | QMR |
| 6 | 공정 파라미터 기준 확정 | 기술팀장 |
| 7 | 정기 재검증 (연 1회 또는 변경 시) | 품질팀장 |

## 8.3 특수공정 관리 항목

| 관리 항목 | 내용 |
|----------|------|
| 공정 파라미터 | 온도, 시간, 전류, 압력 등 — 공정 중 모니터링/기록 |
| 작업자 자격 | 자격 인증 유효기간 관리 (연 1회 재인증) |
| 장비 상태 | 교정, 정기 점검, 유지보수 |
| 자재/소모품 | 약품, 가스, 용접봉 등의 유효기간/규격 관리 |
| 환경 조건 | 온도, 습도, 청정도 (해당 시) |
| 시험편 (Test Coupon) | 대표 시험편으로 공정 결과 검증 (해당 시) |

## 8.4 NADCAP 인가 (해당 시)

고객이 NADCAP 인가를 요구하는 특수공정의 경우:
- NADCAP 인가 취득 및 유지
- NADCAP 감사 대응
- 인가 범위 내 공정만 수행
- 외주 특수공정: NADCAP 인가 공급자 사용

# 9. 생산 검증 및 확인

## 9.1 공정 내 검사

| 검사 유형 | 실시자 | 주기 |
|----------|--------|------|

| 항목 | 기준 |
|------|------|

# 10. 항공우주 특수 요구사항

## 10.1 생산 공정 문서화

AS9100D는 다음을 생산 공정 문서에 포함하도록 요구한다:

| 항목 | 내용 |
|------|------|
| 승인된 생산 데이터 | 도면, BOM, 공정 흐름도, 작업 지시서 |
| 공구/NC 프로그램 목록 | 공정별 사용 공구/프로그램 식별 |
| 검사 계획 | 검사 포인트, 방법, 합부 기준 |
| 특수 관리 항목 | 핵심 특성, 안전 핵심 특성의 관리 방법 |

## 10.2 생산 공정 변경 관리

공정 변경(4M 변경) 시:
- 변경 전 리스크 평가 (PFMEA 갱신)
- 고객 승인 필요 여부 확인 (고객 계약 조건)
- 변경 후 FAI 재실시 여부 결정 (QP-811 연계)
- QP-816 변경관리 절차에 따른 처리

## 10.3 FOD 방지

생산 전 과정에서 FOD(이물질 손상) 방지 활동:
- 작업 전후 청결 확인
- 공구/부품 수량 관리 (Tool Accountability)
- 개구부 보호 (Cover/Plug)
- QP-819 FOD 방지 절차와 연계
