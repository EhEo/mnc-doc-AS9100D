# 1. 목적

본 절차서는 M&C Electronics VINA의 품질경영시스템(QMS)이 AS9100D 요구사항 및 조직의 자체 요구사항에 적합한지를 체계적으로 검증하기 위한 내부심사 절차를 규정한다.

본 절차서의 목적:
- AS9100D 9.2 요구사항에 따른 내부심사 프로그램의 수립 및 운영
- QMS의 효과적 실행 및 유지 여부 확인
- 개선 기회 및 부적합 사항의 식별
- 인증심사(Stage 1/2) 대비 자체 점검 체계 확립

---

# 2. 적용 범위

본 절차서는 M&C Electronics VINA의 QMS 전체 프로세스 및 모든 부서에 적용한다.

| 구분 | 심사 대상 |
|------|----------|
| 시스템 심사 | AS9100D 전 조항(4~10) 대비 QMS 적합성 |
| 프로세스 심사 | 핵심 프로세스의 효과성 및 성과 |

| 공급자 심사 | 외부공급자의 QMS 적합성 (QP-808 연계) |

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 내부심사 (Internal Audit) | 조직 자체적으로 QMS의 적합성과 효과성을 검증하는 체계적이고 독립적인 활동 |
| 심사 프로그램 (Audit Program) | 특정 기간에 계획된 심사의 전체 집합으로, 목적, 범위, 일정, 자원을 포함 |
| 심사 계획 (Audit Plan) | 개별 심사의 활동 및 일정을 기술한 문서 |
| 심사 기준 (Audit Criteria) | 심사 증거를 비교 평가하는 기준 (규격, 절차서, 법규 등) |
| 심사 증거 (Audit Evidence) | 심사 기준과 관련되고 검증 가능한 기록, 사실, 관찰 결과 |
| 심사 발견사항 (Audit Finding) | 심사 증거를 심사 기준과 비교하여 평가한 결과 |
| 부적합 (Nonconformity) | 요구사항의 불충족 — Major(중대) 또는 Minor(경미)로 분류 |
| 관찰사항 (Observation/OFI) | 부적합은 아니나 개선이 권고되는 사항 |
| 수석심사원 (Lead Auditor) | 심사팀을 이끌고 심사 전체를 관리하는 심사원 |
| 심사원 (Auditor) | 심사를 수행할 자격을 갖춘 인원 |
| PEAR | AS9101의 프로세스 효과성 평가 보고서 (Process Effectiveness Assessment Report) |

# 5. 책임과 권한

| 역할 | 책임 |
|------|------|
| **대표이사** | 내부심사 프로그램 승인; 심사 결과에 대한 경영검토 참여; 시정조치를 위한 자원 제공 |
| **품질책임자 (QMR)** | 내부심사 프로그램 총괄 관리; 수석심사원/심사원 임명; 심사 결과 경영검토 보고; 시정조치 이행 감독 |
| **수석심사원** | 심사 계획 수립; 심사팀 구성 및 업무 배분; 심사 실시 주관; 심사 보고서 작성; 시정조치 유효성 확인 |
| **심사원** | 배정된 영역의 심사 실시; 심사 증거 수집 및 기록; 발견사항 보고; 심사 기밀 유지 |
| **피심사 부서장** | 심사 협조 (인원, 기록, 시설 접근 제공); 부적합 사항에 대한 시정조치 이행; 시정조치 완료 보고 |

# 6. 심사 프로그램 관리

## 6.1 연간 심사 프로그램

QMR은 매년 12월에 익년도 내부심사 프로그램을 수립하여 대표이사의 승인을 받는다.

| 고려 사항 | 내용 |
|----------|------|
| 심사 주기 | AS9100D 전 조항을 최소 연 1회 심사 (인증 주기 3년 내 전 조항 커버) |
| 리스크 기반 | 고위험 프로세스, 이전 부적합 다발 영역 → 심사 빈도/깊이 증가 |
| 변경 사항 | 프로세스 변경, 조직 변경, 고객 불만 영역 → 추가 심사 |
| 외부심사 결과 | 인증기관 심사에서 지적된 영역 → 후속 심사 강화 |
| 자원 | 심사원 가용성, 생산 일정과의 조율 |

## 6.2 심사 유형

> **참고**: 기존 ISO 9001/IATF 16949 체계(ISO.02.05.06, Rev.08)에서는 QSA(시스템 심사), QPA(공정 심사), Product Audit(제품 심사) 3종을 운영하고 있다. AS9100D 전환 시 이 3종 체계를 유지하면서 항공우주 요구사항을 보강한다.

### 6.2.1 심사 유형 및 주기

| 심사 유형 | 기존 명칭 | 목적 | 주기 | 체크리스트 |
|----------|---------|------|------|----------|
| **시스템 심사 (QSA)** | QSA (Quality System Audit) | AS9100D 전 조항 대비 QMS 적합성/효과성 평가 | **연 1회 이상** | F-903-03 (AS9101 PEAR 포함) |
| **공정 심사 (QPA)** | QPA (Quality Process Audit) | 생산 공정의 절차 준수, 파라미터 관리, 작업 환경 확인 | **주 1회** (기존 체계 유지) | F-903-08 QPA 체크시트 |
| **제품 심사 (Product Audit)** | Product Audit | 완성품의 원자재, 성능, 치수 적합성 확인 | **연 1회** | F-903-09 제품심사 체크시트 |
| 추가 심사 | - | 중대 부적합, 고객 불만, 프로세스 변경 시 | 필요 시 | 해당 유형 체크리스트 |
| 후속 심사 | - | 시정조치 유효성 확인 | 시정조치 완료 후 | F-903-07 |
| 사전 심사 (Pre-audit) | - | 인증심사 대비 모의 심사 | 인증심사 4~6주 전 | F-903-03 |

### 6.2.2 AS9100D 전환 시 추가된 심사 요구사항

| 기존 (ISO 9001/IATF) | AS9100D 추가 요구 |
|---------------------|-----------------|
| 시스템 심사 연 1회 | 프로세스 **효과성** 평가 추가 (AS9101 PEAR) |
| 심사 기록 보존 3년 | **5년**으로 상향 |
| 심사원 자격: 내부교육 이수 | AS9100D/항공우주 심사 교육 추가 이수 필요 |
| VDA 6.3 공정 심사 참조 | AS9101 기반 항공우주 프로세스 평가로 전환 |

## 6.3 심사 프로그램 모니터링

| 지표 | 목표 |
|------|------|
| 계획 대비 실행률 | ≥ 90% |
| 부적합 시정조치 완료율 | 100% (기한 내) |
| 심사원 자격 유지율 | 100% |
| 전 조항 커버리지 | 연간 100% (AS9100D 4~10) |

# 7. 심사 계획 및 준비

## 7.1 심사 계획 수립

수석심사원은 개별 심사 실시 최소 **2주 전**에 심사 계획을 수립한다.

| 계획 포함 사항 | 내용 |
|--------------|------|
| 심사 목적 | QMS 적합성/효과성 확인, 특정 개선 확인 등 |
| 심사 범위 | 대상 부서, AS9100D 조항, 프로세스 |
| 심사 기준 | AS9100D, 관련 절차서, 고객 요구사항, 법적 요구사항 |
| 심사 일정 | 날짜, 시간, 장소 |
| 심사팀 구성 | 수석심사원, 심사원, 관찰자(Trainee) |
| 심사 방법 | 인터뷰, 문서/기록 검토, 현장 관찰, 샘플링 기법 |

## 7.2 심사원 독립성

- 심사원은 자신의 업무 영역을 심사할 수 없다 (독립성 보장)
- 소규모 조직의 경우, 교차 심사(Cross-Audit) 방식으로 독립성 확보
- 심사원 배정 시 이해충돌(Conflict of Interest) 여부 확인

## 7.3 심사 준비

| 준비 활동 | 내용 |
|----------|------|
| 문서 사전 검토 | 해당 부서 관련 절차서, 이전 심사 보고서, CAR 현황 검토 |
| 체크리스트 작성 | AS9100D 조항별 심사 질문 목록 작성 (AS9101 PEAR 참조) |
| 사전 통보 | 피심사 부서에 심사 계획 통보 (최소 1주 전) |
| 심사팀 회의 | 심사 범위, 역할 분담, 주의사항 공유 |

# 8. 심사 실시

## 8.1 심사 프로세스

```
시작 회의 (Opening Meeting)
    │
    ▼
현장 심사 실시 (On-site Audit)
    │  ├── 문서/기록 검토
    │  ├── 인터뷰
    │  ├── 현장 관찰
    │  └── 샘플링
    │
    ▼
심사 증거 수집 및 평가
    │
    ▼
발견사항 분류 (Major NC / Minor NC / OFI)
    │
    ▼
심사팀 내부 회의
    │
    ▼
종료 회의 (Closing Meeting)
```

## 8.2 시작 회의 (Opening Meeting)

| 항목 | 내용 |
|------|------|
| 참석자 | 수석심사원, 심사팀, 피심사 부서장 및 관련 인원 |
| 진행 내용 | 심사 목적/범위 확인; 심사 일정 안내; 심사 방법 설명; 부적합 분류 기준 설명; 기밀 유지 확인 |

## 8.3 현장 심사

| 심사 방법 | 설명 |
|----------|------|
| 문서/기록 검토 | 절차서 준수 여부, 기록의 완전성/정확성 확인 |
| 인터뷰 | 업무 담당자와의 면담을 통한 프로세스 이해도 및 실행 확인 |
| 현장 관찰 | 작업 현장에서 절차 준수, 장비 상태, 환경 조건 직접 확인 |
| 샘플링 | 기록/제품에서 대표 샘플을 추출하여 적합성 검증 |
| 추적 심사 (Tracing) | 특정 주문/로트를 처음부터 끝까지 추적하여 프로세스 연결 확인 |

## 8.4 발견사항 분류

| 분류 | 정의 | 예시 |
|------|------|------|
| **Major 부적합** | 시스템적 실패 또는 요구사항의 완전한 불충족; 제품/서비스 품질에 직접 영향 | 절차서 미수립, 교정 미실시, FAI 미수행 |
| **Minor 부적합** | 요구사항의 부분적 불충족; 단발적 또는 경미한 일탈 | 기록 누락(단건), 서명 미비, 경미한 절차 미준수 |
| **OFI (관찰사항)** | 부적합은 아니나 개선 시 효과적인 사항 | 프로세스 효율화 기회, 모범 사례 확산 가능 |

## 8.5 종료 회의 (Closing Meeting)

| 항목 | 내용 |
|------|------|
| 참석자 | 시작 회의와 동일 |
| 진행 내용 | 심사 결과 요약 발표; 발견사항(NC/OFI) 설명; 시정조치 일정 합의; 감사 표시 |
| 피심사 부서 | 발견사항에 대한 이견 표시 기회; 시정조치 기한 합의 |

# 9. 심사 보고 및 후속 조치

## 9.1 심사 보고서

수석심사원은 심사 종료 후 **5영업일 이내**에 내부심사 보고서를 작성하여 QMR에게 제출한다.

| 보고서 포함 사항 | 내용 |
|----------------|------|
| 심사 개요 | 심사 목적, 범위, 일정, 심사팀 |
| 심사 기준 | 적용된 규격, 절차서 |
| 발견사항 요약 | Major NC, Minor NC, OFI 건수 및 내용 |
| 발견사항 상세 | 각 건별 조항, 증거, 분류, 해당 부서 |
| 긍정적 발견사항 | 우수 사례, 모범 실행 |
| 종합 평가 | QMS 전체 적합성/효과성 평가 |

## 9.2 시정조치

| 분류 | 시정조치 기한 | 후속 조치 |
|------|------------|----------|
| Major 부적합 | 발행일로부터 **30일 이내** | 시정조치 완료 후 후속 심사로 유효성 확인 |
| Minor 부적합 | 발행일로부터 **60일 이내** | 서류 검증 또는 후속 심사로 확인 |
| OFI | 권고 사항 (기한 없음) | 차기 심사 시 개선 여부 확인 |

시정조치는 QP-1001 (부적합/시정조치 관리)에 따라 CAR을 발행하여 관리한다.

## 9.3 시정조치 유효성 확인

| 확인 방법 | 적용 |
|----------|------|
| 서류 검증 | Minor NC의 경우, 시정조치 증거 서류 검토로 완료 가능 |
| 후속 심사 | Major NC의 경우, 현장 방문하여 시정조치의 효과적 실행 확인 |
| 차기 정기 심사 | 시정조치 유지 여부를 차기 정기 심사에서 재확인 |

수석심사원이 유효성을 확인하고 서명 시 시정조치가 종결된다.

# 10. 심사원 자격 관리

## 10.1 심사원 자격 요건

| 구분 | 요건 |
|------|------|
| **수석심사원** | ISO 19011 또는 AS9100 심사원 과정 이수; 내부심사 리더 경험 3회 이상; AS9100D 요구사항 심화 교육 이수; QMR 임명 |
| **심사원** | ISO 19011 기반 내부심사원 교육 이수 (최소 16시간); 내부심사 참관 경험 1회 이상; 담당 심사 영역의 업무 이해 |
| **심사원 후보 (Trainee)** | 내부심사원 교육 이수; 자격 심사원 지도하에 심사 참관 2회 이상; 참관 후 평가 합격 시 심사원 자격 부여 |

## 10.2 심사원 자격 유지

| 항목 | 기준 |
|------|------|
| 심사 실적 | 연 1회 이상 내부심사 참여 |
| 보수 교육 | 연 1회 이상 AS9100D/심사 기법 관련 교육 참가 |
| 자격 재평가 | 2년마다 QMR이 심사원 역량 재평가 |
| 자격 박탈 | 2년간 심사 미참여 또는 심각한 심사 윤리 위반 시 |

## 10.3 심사원 명부

품질팀장은 심사원 명부를 유지하며, 다음 정보를 포함한다:
- 성명, 소속 부서
- 자격 등급 (수석심사원/심사원/Trainee)
- 교육 이수 이력
- 심사 수행 이력
- 전문 영역 (심사 가능 프로세스)
- 자격 유효기간

# 11. 항공우주 특수 요구사항

## 11.1 프로세스 효과성 평가 (AS9100D 9.2.2.1)

AS9100D는 내부심사 시 단순 적합성뿐 아니라 프로세스의 **효과성**을 평가하도록 요구한다.

| 평가 항목 | 내용 |
|----------|------|
| 프로세스 성과 | 프로세스 목표(KPI) 달성 여부 |
| 리스크 관리 | 식별된 리스크에 대한 관리 활동의 효과성 |
| 고객 만족 | 프로세스가 고객 요구사항 충족에 기여하는 정도 |
| 지속적 개선 | 개선 활동의 실행 및 성과 |

AS9101 PEAR (Process Effectiveness Assessment Report) 체크리스트를 내부심사 시 참고하여 프로세스 효과성을 체계적으로 평가한다.

## 11.2 제품 안전 심사

AS9100D 8.1.4 (제품 안전)와 관련하여:
- 안전 관련 프로세스는 매년 반드시 심사 대상에 포함
- 안전 요구사항의 식별, 관리, 보고 체계 확인
- 안전 관련 부적합의 적시 보고 및 처리 확인

## 11.3 특수공정 심사

AS9100D 8.5.1과 관련하여 특수공정에 대한 심사 시:
- 공정 승인 상태 확인
- 작업자 자격 유효성 확인
- 공정 파라미터 관리 기록 확인
- 장비/환경 조건 적합성 확인

## 11.4 위조 부품 방지 심사

AS9100D 8.1.5와 관련하여:
- 위조 부품 방지 절차의 실행 여부 확인
- 공급자 인증서, CoC의 진위 확인 프로세스 심사
- 의심/발견된 위조 부품 보고 체계 확인

## 11.5 인증심사 대비

| 활동 | 시기 | 목적 |
|------|------|------|
| 사전 심사 (Pre-audit) | 인증심사 4~6주 전 | 인증심사와 동일 범위/방법으로 모의 심사 실시 |
| GAP 분석 | 인증심사 8주 전 | AS9100D 전 조항 대비 문서/실행 수준 점검 |
| 시정조치 완료 확인 | 인증심사 2주 전 | 모든 내부심사 CAR 종결 확인 |
