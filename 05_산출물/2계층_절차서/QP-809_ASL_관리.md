# 1. 목적

본 절차서는 M&C Electronics VINA의 인가된 공급자 목록(ASL, Approved Supplier List)의 작성, 유지, 변경 및 폐기에 관한 전 생애주기 관리 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.4.1 Note 요구사항에 따른 ASL 수립 및 체계적 관리
- 공급자의 품질 능력에 기반한 승인/분류/재평가 체계 확립
- 고객 지정 공급자(Customer-Directed Source) 및 고객 승인 공급자(Customer-Approved Source) 관리
- 특수공정 공급자에 대한 별도 인증(NADCAP 등) 요건 관리
- 위조부품 유입 방지를 위한 공급자 통제 체계 구축
- **CAR-009 (ASL 미작성, 공급자 평가/선정/재평가 이력 없음) 시정조치 반영**

> **CAR-009 연계**: 갭 분석에서 "승인 공급업체 목록(ASL) 미작성, 공급자 평가/선정/재평가 이력 없음"으로 Major 부적합이 식별됨. Is-Is Not 분석 결과, 공급자 선정이 가격/납기 중심으로만 이루어져 왔고 품질 기반 평가 체계가 미구축된 것이 근본 원인으로 확인됨. 본 절차서는 QP-808(외부공급자 관리)과 연계하여 ASL 생애주기 전반을 규정한다.

> **참고**: 공급자 초기 선정, 평가 기준, 재평가 방법 등 외부공급자 관리의 일반 사항은 QP-808에서 규정한다. 본 절차서는 QP-808의 평가 결과에 기반한 **ASL 등재, 상태 관리, 변경/제거**에 집중한다.

# 2. 적용 범위

본 절차서는 M&C Electronics VINA에 제품, 부품, 원자재, 외주 공정 및 서비스를 공급하는 모든 외부공급자의 ASL 관리에 적용한다.

| 구분 | 적용 대상 | 비고 |
|------|----------|------|
| 원자재 공급자 | 금속, 수지, 전자부품 등 직접재 공급자 | 항공우주 규격 재료 인증(Mill Cert.) 필수 |
| 부품 공급자 | 구매 부품, 표준품(Fastener 등) 공급자 | 항공우주 등급 요건 확인 |
| 외주 가공 공급자 | 기계가공, 판금 등 외주 제조 업체 | 공정 능력 검증 필요 |
| 특수공정 공급자 | 도금, 열처리, 비파괴검사(NDT), 용접, 도장 등 | NADCAP 등 별도 인증 요구 (§11 참조) |
| 교정 서비스 | 측정장비 교정 업체 | ISO/IEC 17025 인정 필요 |
| 물류/운송 | 항공우주 제품 운송 업체 | 포장/운송 요건 충족 여부 |

**적용 제외**:
- 사무용품, 시설 유지보수 등 제품 품질에 영향을 미치지 않는 간접 서비스 공급자
- 단, 품질책임자(QMR)가 품질 영향이 있다고 판단하는 경우 ASL 관리 대상에 포함

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| ASL (Approved Supplier List) | 인가된 공급자 목록 — M&C Electronics VINA가 품질 능력을 평가하여 승인한 공급자의 공식 목록. 구매 발주는 ASL에 등재된 공급자에게만 허용됨 |
| QVL (Qualified Vendor List) | 적격 공급업체 목록 — 고객 또는 산업 규격에 의해 자격이 인정된 공급자 목록. ASL의 상위 개념으로 활용되기도 함 |
| Approved (승인) | ASL 등재 상태 중 최상위 등급. 초기 평가 및 시범 공급을 통과하여 제한 없이 발주 가능한 공급자 |
| Conditional (조건부 승인) | 특정 조건(품목 한정, 수량 제한, 추가 검사 등) 하에서만 발주가 허용되는 공급자. 시범 공급 단계 또는 시정조치 이행 중인 공급자 |
| Probation (관찰) | 품질/납기 성과 저하로 인해 경고 상태에 놓인 공급자. 개선 기한 내 미달 시 Suspended 또는 Removed로 전환 |
| Suspended (일시 정지) | 중대 품질 사고, 인증 실효 등의 사유로 신규 발주가 일시 중단된 공급자. 기존 발주(In-progress)는 리스크 평가 후 결정 |
| Removed (제거/삭제) | ASL에서 완전 삭제된 공급자. 해당 공급자에 대한 모든 발주 금지. 재등재 시 초기 평가부터 재수행 |
| Customer-Directed Source | 고객이 반드시 사용하도록 지정한 공급자. 당사 평가 결과와 무관하게 사용 의무가 있으나, 당사 ASL에도 등재하여 관리 |
| Customer-Approved Source | 고객이 사전 승인한 공급자 목록(고객 QVL 등)에 포함된 공급자. 당사 ASL 등재 시 고객 승인 상태를 확인·유지 |
| OEM 공급원 (OCM/OEM) | 원제조사(Original Component/Equipment Manufacturer). 부품의 최초 제조원 |
| 인가 유통업체 (Authorized Distributor) | OCM/OEM이 공식 인가한 유통 경로. 위조부품 리스크 최소화를 위한 우선 조달원 |
| 특수공정 (Special Process) | 결과물을 후속 모니터링 또는 측정으로 검증할 수 없어, 사전 자격 인정이 필요한 공정 (예: 용접, 도금, 열처리, NDT, 도장) |
| NADCAP | National Aerospace and Defense Contractors Accreditation Program — 항공우주 특수공정 국제 인증 프로그램 |
| Flow-down Requirements | 고객으로부터 당사로, 당사에서 공급자로 전달(유동)해야 하는 품질/기술/규제 요구사항 |

| **품질팀장** | 공급자 품질 능력 평가(QP-808 연계); 수입검사 데이터 기반 공급자 성과 분석; 공급자 감사(on-site audit) 수행; 부적합 발생 시 공급자 시정조치 관리; 특수공정 인증 현황 모니터링 |

| **구매팀 담당자** | ASL 데이터베이스 유지/갱신; 공급자 문서(인증서, 시험성적서 등) 수집/관리; ASL 등재 신청서 작성; 공급자 연락 창구 |
| **영업팀장** | 고객 지정 공급자(Customer-Directed Source) 정보 전달; 고객 QVL/ASL 요구사항 확인 및 전달 |

### 5.1 ASL 관련 의사결정 권한 매트릭스

| 의사결정 사항 | 기안 | 검토 | 승인 |

# 6. ASL 등재 기준 및 절차

## 6.1 ASL 등재 프로세스 개요

ASL 등재는 다음 3단계로 진행한다:

```
[1단계: 초기 평가]  →  [2단계: 시범 공급]  →  [3단계: 정식 등재]
   (QP-808 연계)          (Conditional)          (Approved)
```

## 6.2 1단계: 초기 평가

QP-808에 따른 외부공급자 초기 평가를 수행한다. ASL 등재를 위한 최소 요건은 다음과 같다:

### 6.2.1 필수 평가 항목

| 평가 항목 | 최소 요건 | 평가 방법 |
|----------|----------|----------|
| 품질경영시스템 인증 | ISO 9001 이상 (항공우주: AS9100D 우선) | 인증서 사본 확인 |
| 사업 안정성 | 최근 2년 이상 사업 영위, 재정 건전성 | 사업자등록증, 재무제표 |
| 기술 능력 | 해당 품목/공정 수행 능력 보유 | 설비 목록, 기술 자격증, 실적 |
| 품질 이력 | 타 고객 대상 품질 성과 자료 | 부적합률, 고객 불만 이력 |
| 납기 이력 | 납기 준수율 90% 이상 (자사/타사 실적) | 납기 실적 데이터 |
| 위조부품 방지 | 위조부품 방지 체계 보유 (§12 참조) | 자가 점검 체크리스트 또는 현장 확인 |
| 규제 준수 | 수출 규제, 환경 규제 등 법적 요건 준수 | 관련 인허가/인증서 |

### 6.2.2 초기 평가 수행

1. **구매팀**: 공급자 기본 정보 및 자격 서류 수집 → QF-809-01 (ASL 등재 신청서) 작성
2. **품질팀**: QP-808에 따른 품질 능력 평가 수행
   - 서면 평가(Documentation Review) 필수
   - 현장 감사(On-site Audit): 아래 조건 중 1개 이상 해당 시 필수
     - 항공우주 안전 관련 부품(Safety Critical Part) 공급자
     - 특수공정 수행 공급자
     - 연간 구매액 상위 20% 이내 공급자
     - 과거 품질 문제 이력이 있는 공급자
3. **기술팀**: 기술적 적합성 평가 (필요 시)
4. **품질팀장**: 종합 평가 결과 검토 → 등재 권고 의견 작성

### 6.2.3 초기 평가 점수 기준

| 종합 평가 점수 | 판정 | 후속 조치 |
|-------------|------|----------|
| 80점 이상 | 적합 (시범 공급 진행) | 2단계 시범 공급 실시 |
| 60~79점 | 조건부 적합 | 시정조치 요구 후 재평가; 재평가 통과 시 시범 공급 |
| 60점 미만 | 부적합 | 등재 불가; 6개월 후 재신청 가능 |

## 6.3 2단계: 시범 공급 (Conditional 상태)

초기 평가를 통과한 공급자에 대해 시범 공급(Trial Period)을 실시한다.

### 6.3.1 시범 공급 조건

| 항목 | 기준 |
|------|------|
| 기간 | 최초 납품일로부터 6개월 또는 3회 납품 중 먼저 도래하는 시점 |
| ASL 상태 | **Conditional** |
| 수입검사 | 전수 검사(Full Inspection) 적용 |
| 발주 한도 | 단일 품목, 1차 공급원으로만 사용 불가(2nd Source로 운용) |
| 모니터링 | 월 1회 품질/납기 성과 평가 |

### 6.3.2 시범 공급 판정

시범 공급 기간 종료 후 아래 기준에 따라 판정한다:

| 판정 기준 | Approved 전환 | Conditional 연장 | 등재 취소 |
|----------|-------------|-----------------|----------|
| 수입검사 합격률 | ≥ 98% | 95~97% | < 95% |
| 납기 준수율 | ≥ 95% | 90~94% | < 90% |
| 시정조치 대응 | 요청 후 10영업일 내 회신 | 10~20영업일 | 미회신 또는 20영업일 초과 |
| 중대 품질 사고 | 0건 | 0건 | 1건 이상 |

- **Conditional 연장**: 최대 1회(추가 6개월)까지 허용. 2차 연장 후에도 미달 시 등재 취소
- **등재 취소**: ASL에서 Removed 처리, 공급자에 서면 통보

## 6.4 3단계: 정식 등재 (Approved 상태)

시범 공급 판정을 통과한 공급자를 정식 Approved 상태로 전환한다.

1. **품질팀장**: 시범 공급 평가 결과 종합 → 정식 등재 추천서 작성
2. **구매팀장**: ASL 등재 변경 기안 (Conditional → Approved)
3. **QMR**: 최종 승인
4. **구매팀 담당자**: ASL 데이터베이스 갱신 → 공급자에 승인 통보서 발송

# 7. ASL 상태 분류

## 7.1 상태 정의 및 운영 기준

| 상태 | 코드 | 발주 가능 여부 | 수입검사 수준 | 설명 |
|------|------|-------------|-------------|------|
| **Approved** | A | 제한 없이 발주 가능 | 정상 샘플링 검사 | 모든 평가 기준 충족, 성과 양호 |
| **Conditional** | C | 조건부 발주 가능 | 전수 검사 또는 강화 검사 | 시범 공급 중이거나 특정 조건 부과 상태 |
| **Probation** | P | 기존 발주만 허용 (신규 발주 제한) | 전수 검사 | 성과 저하 경고 상태, 개선 기한 부여 |
| **Suspended** | S | 신규 발주 중단 | 해당 없음 (발주 중단) | 중대 사유 발생, 기존 발주는 리스크 평가 후 결정 |
| **Removed** | R | 모든 발주 금지 | 해당 없음 | ASL에서 완전 삭제, 재등재 시 초기 평가부터 재수행 |

## 7.2 상태 전환 매트릭스

```
                    정상 운영
                       │
                   ┌───┴───┐
                   │       │
              Approved  Conditional
               (A)       (C)
                │         │
                ├─────────┤
                │         │
           성과 저하    미달 지속
                │         │
                ▼         ▼
            Probation ←───┘
               (P)
                │
          ┌─────┴─────┐
          │           │
       개선 완료   미개선/중대 사고
          │           │
          ▼           ▼
      Approved    Suspended
       (A)          (S)
                     │
               ┌─────┴─────┐
               │           │
            시정 완료    시정 불가
               │           │
               ▼           ▼
           Conditional   Removed
            (C)           (R)
```

## 7.3 상태 전환 사유

### 7.3.1 Approved → Probation 전환 사유
- 정기 재평가에서 종합 점수 70점 미만
- 분기 수입검사 합격률 95% 미만
- 분기 납기 준수율 90% 미만
- 동일 유형 부적합 반복 발생 (분기 내 3건 이상)
- 시정조치 요청에 대한 기한 내 미이행

### 7.3.2 Approved/Conditional/Probation → Suspended 전환 사유
- 안전 관련 부적합(Safety-Related Nonconformity) 발생
- 품질경영시스템 인증 실효/정지
- NADCAP 등 특수공정 인증 만료/정지 (특수공정 공급자의 경우)
- 위조부품/비인가 부품 공급 의심 또는 확인
- 고객으로부터의 공급자 사용 중단 요청
- 심각한 법규 위반 (수출통제, 환경규제 등)
- 공급자의 비협조적 태도 (감사 거부, 시정조치 완전 무시 등)

### 7.3.3 → Removed 전환 사유
- Suspended 상태에서 60일 이내 시정조치 미완료
- 위조부품 공급 확인 (즉시 Removed)
- 공급자 폐업/파산
- 고객으로부터의 영구 사용 금지 지시
- QMR 및 대표이사 승인에 의한 거래 종료 결정

# 8. ASL 유지 관리

## 8.1 정기 재평가

### 8.1.1 재평가 주기

| ASL 상태 | 재평가 주기 | 평가 방법 |
|---------|----------|----------|
| Approved | 12개월 | 서면 평가 + 성과 데이터 분석 |
| Approved (안전 관련 부품) | 12개월 | 서면 평가 + 현장 감사 |
| Conditional | 6개월 | 서면 평가 + 성과 데이터 분석 |
| Probation | 3개월 | 서면 평가 + 성과 데이터 분석 + 현장 감사 (필요 시) |

### 8.1.2 재평가 항목

| 평가 영역 | 평가 항목 | 배점 |
|----------|----------|------|
| 품질 (40%) | 수입검사 합격률, 공정 내 불량률, 고객 불만 연계 부적합, 시정조치 이행도 | 40점 |
| 납기 (25%) | 납기 준수율, 긴급 대응 능력, 납기 지연 사전 통보 여부 | 25점 |
| 기술 (15%) | 기술 지원 능력, 설계 변경 대응, 공정 개선 활동 | 15점 |
| 경영 (10%) | QMS 인증 유지, 재정 안정성, 사업 연속성 | 10점 |
| 협력 (10%) | 의사소통 적시성, 감사 협조, 개선 활동 참여 | 10점 |

### 8.1.3 재평가 판정 기준

| 종합 점수 | 등급 | ASL 조치 |
|----------|------|---------|
| 85점 이상 | A (우수) | Approved 유지, 수입검사 완화 검토 가능 |
| 70~84점 | B (양호) | Approved 유지, 개선 권고 사항 통보 |
| 60~69점 | C (미흡) | Probation 전환, 시정조치 요구 |
| 60점 미만 | D (부적합) | Suspended 전환, 대체 공급원 확보 착수 |

## 8.2 성과 모니터링

### 8.2.1 월간 모니터링 항목

구매팀은 다음 항목을 월간 집계하여 품질팀장에게 보고한다:

| 모니터링 항목 | 데이터 소스 | 산출 방법 |
|-------------|-----------|----------|
| 수입검사 합격률 | 수입검사 기록 (QF-809-03) | (합격 로트 수 / 총 검사 로트 수) × 100% |
| 납기 준수율 | 구매 발주 및 입고 기록 | (기한 내 입고 건수 / 총 발주 건수) × 100% |
| 부적합 건수 | 부적합 보고서 (QP-818) | 공급자별 월간 부적합 발생 건수 |
| 시정조치 대응률 | CAR 기록 (QP-1001) | (기한 내 회신 건수 / 총 CAR 발행 건수) × 100% |

### 8.2.2 분기 성과 리뷰

품질팀장 주관으로 분기 1회 공급자 성과 리뷰를 실시한다:
- 참석: 구매팀장, 품질팀장, 기술팀장 (필요 시)
- 주요 내용:
  - 공급자별 분기 성과 종합 (품질/납기/대응)
  - ASL 상태 변경 필요 공급자 식별
  - 리스크 공급자 관리 대책
  - 고객 피드백 반영 사항
- 결과: QF-809-04 (공급자 분기 성과 리뷰 보고서) 작성 → QMR 보고

### 8.2.3 인증서 유효기간 관리

구매팀 담당자는 ASL 등재 공급자의 다음 인증서 유효기간을 추적 관리한다:
- ISO 9001 / AS9100D 인증서
- NADCAP 인증서 (특수공정 공급자)
- ISO/IEC 17025 인정서 (교정 업체)
- 기타 고객 요구 인증서

인증서 만료 **90일 전** 공급자에 갱신 통보, **30일 전** 미갱신 시 품질팀장에 보고하여 ASL 상태 변경 검토.

# 9. ASL 변경 관리

## 9.1 ASL 변경 유형

| 변경 유형 | 설명 | 승인 권한 |
|----------|------|----------|
| 상태 등급 변경 (상향) | Conditional → Approved, Probation → Approved | QMR |
| 상태 등급 변경 (하향) | Approved → Probation, 등 | QMR |
| Suspended 처리 | 신규 발주 중단 | QMR |
| Removed 처리 | ASL 완전 삭제 | QMR + 대표이사 |
| 공급 범위 변경 | 품목/공정 추가 또는 제한 | QMR |
| 공급자 정보 변경 | 상호, 주소, 연락처, 인증서 등 | 구매팀장 |

## 9.2 ASL 변경 절차

1. **변경 기안**: 해당 부서(구매팀 또는 품질팀)에서 QF-809-05 (ASL 변경 요청서) 작성
2. **사유 및 근거 문서화**: 변경 사유, 관련 데이터(성과 기록, 부적합 보고서, 감사 보고서 등) 첨부
3. **검토**: §5.1 권한 매트릭스에 따른 검토자 검토
4. **승인**: §5.1 권한 매트릭스에 따른 승인자 승인
5. **시행**: ASL 데이터베이스 갱신, 관련 부서 통보
6. **공급자 통보**: 상태 변경 시 공급자에 서면 통보 (Probation, Suspended, Removed의 경우 필수)

## 9.3 Removed 공급자 재등재

Removed 상태의 공급자가 재등재를 희망하는 경우:

1. Removed 처리 후 **최소 12개월** 경과 (위조부품 사유의 경우 재등재 불가)
2. 공급자가 근본 원인 분석 및 시정조치 완료 증빙 제출
3. **초기 평가(§6.2)부터 전 과정 재수행**
4. QMR 및 대표이사 공동 승인 필요
5. 재등재 시 **Conditional** 상태에서 시작 (직접 Approved 불가)
6. 시범 공급 기간을 **12개월 또는 6회 납품**으로 확대 적용

## 9.4 긴급 공급자 사용 (비 ASL 공급자)

예외적으로 ASL에 미등재된 공급자로부터 구매가 불가피한 경우:

| 항목 | 요건 |
|------|------|
| 승인 | QMR 서면 승인 필수 (QF-809-06, 비ASL 공급자 긴급 사용 승인서) |
| 사유 | 기존 ASL 공급자 공급 불가, 긴급 고객 요구, 단종품 대체 등 |
| 검사 | 해당 납품분 전수 검사 필수 |
| 기한 | 1회성 발주에 한함. 계속 거래 시 ASL 등재 절차 수행 |
| 기록 | 긴급 사용 사유, 리스크 평가, 검사 결과 기록 보존 |
| 고객 통보 | 안전 관련 부품인 경우 고객 사전 통보 및 승인 필요 |

# 10. 고객 지정 공급자 및 고객 승인 공급자 관리

## 10.1 고객 지정 공급자 (Customer-Directed Source)

고객이 특정 공급자의 사용을 의무화(Directed)한 경우:

### 10.1.1 등재 절차
1. **영업팀**: 계약/발주서에서 고객 지정 공급자 정보 식별 → 구매팀 전달
2. **구매팀**: QF-809-01 (ASL 등재 신청서)에 "고객 지정" 표시하여 등재 요청
3. **품질팀**: 가능한 범위에서 초기 평가 수행 (§6.2)
4. **QMR 승인**: ASL에 등재 (상태: Approved 또는 Conditional)
5. **특이사항**: 초기 평가 결과가 부적합하더라도 고객 지정 공급자는 등재하되, **리스크 기록을 문서화**하고 고객에 통보

### 10.1.2 관리 요건
- ASL에 "Customer-Directed" 표시 필수
- 당사 평가 결과와 무관하게 발주 가능하나, 수입검사는 당사 기준 적용
- 품질 문제 발생 시 고객에 즉시 통보
- 정기 재평가 대상에 포함 (결과는 고객 공유)
- 고객 지정 해제 시 일반 공급자와 동일 기준 적용

## 10.2 고객 승인 공급자 (Customer-Approved Source)

고객이 사전 승인한 공급자 목록(QVL 등)에 포함된 공급자:

### 10.2.1 등재 절차
1. 고객 QVL/ASL 수령 시 해당 공급자 목록 확인
2. 당사 ASL 등재 절차(§6) 수행 — 고객 승인 사실은 가점 부여 가능
3. ASL에 "Customer-Approved" 표시 및 고객 승인 번호/일자 기록

### 10.2.2 관리 요건
- 고객 QVL 갱신 시 당사 ASL과 비교·대조하여 정합성 유지
- 고객 QVL에서 삭제된 공급자는 즉시 ASL 상태 재검토
- 고객 요구 시 당사 공급자 성과 데이터 공유

# 11. 특수공정 공급자 ASL

## 11.1 특수공정 유형

| 특수공정 | 해당 인증/자격 | 비고 |
|---------|-------------|------|
| 도금 (Plating) | NADCAP Chemical Processing | AC7108 기준 |
| 열처리 (Heat Treatment) | NADCAP Heat Treating | AC7102 기준 |
| 비파괴검사 (NDT) | NADCAP Non-Destructive Testing | AC7114 기준 |
| 용접 (Welding) | NADCAP Welding | AC7110 기준 |
| 도장/코팅 (Coating) | NADCAP Coatings | AC7109 기준 |
| 사출성형 (Molding) | 공정 검증(Process Validation) | 자사 또는 고객 기준 |

## 11.2 특수공정 공급자 등재 추가 요건

일반 ASL 등재 요건(§6)에 더하여 다음을 충족해야 한다:

| 추가 요건 | 설명 |
|----------|------|
| NADCAP 인증 보유 | 고객 요구 시 해당 공정 NADCAP 인증 필수. 고객 요구 없는 경우에도 NADCAP 보유를 우선 |
| 공정 인정(Process Approval) | 고객 또는 당사에 의한 공정 인정(PQ, Process Qualification) 완료 |
| 작업자 자격 인정 | 해당 특수공정 수행 인원의 자격 인정 증빙 확인 |
| 시험 설비 | 해당 공정 결과 검증에 필요한 시험/측정 장비 보유 확인 |
| 공정 변경 통보 | 공정, 설비, 재료, 인원 변경 시 사전 통보 의무 계약 체결 |

## 11.3 특수공정 공급자 모니터링

- NADCAP 인증 유효기간 및 갱신 현황 분기별 확인
- NADCAP Merit 또는 18개월 인증 공급자: 연 1회 모니터링
- NADCAP 12개월 인증 또는 Non-NADCAP 공급자: 반기 1회 현장 감사 또는 서면 확인
- 고객으로부터 특수공정 관련 품질 이슈 접수 시 즉시 현장 감사 실시

# 12. 위조부품 방지와 ASL 연계

## 12.1 위조부품 방지 원칙

AS9100D 8.1.4 및 AS6081/AS6174 요구사항에 따라, ASL 관리 전반에 위조부품 방지 원칙을 적용한다:

1. **OEM/OCM 공급원 우선**: 부품 조달 시 원제조사 또는 인가 유통업체를 우선 사용
2. **공급 이력 추적**: 부품의 공급 경로(Supply Chain Traceability) 추적 가능해야 함
3. **비인가 공급원 통제**: OEM/OCM 이외 공급원 사용 시 추가 검증 절차 적용

## 12.2 ASL 등재 시 위조부품 방지 평가

| 평가 항목 | 요건 |
|----------|------|
| 조달 경로 | OEM/OCM 직접 또는 인가 유통업체 여부 확인 |
| 위조부품 방지 절차 | 자체 위조부품 탐지/방지 절차 보유 여부 |
| 입고 검증 | 부품 진위 확인 절차 (Visual, Marking, Test 등) |
| 인증서 관리 | CoC(Certificate of Conformance), Mill Certificate, 시험성적서 관리 체계 |
| 이력 추적 | 부품 원산지 ~ 최종 납품까지 추적 가능 여부 |
| 교육 | 위조부품 인식 교육 실시 여부 |

## 12.3 비인가 공급원 (Unauthorized Source) 사용 통제

OEM/OCM 인가 유통업체가 아닌 공급원(Independent Distributor, Broker 등)으로부터 조달이 불가피한 경우:

1. **QMR 사전 승인** 필수
2. 공급원의 AS6081 인증 보유 확인 (또는 동등 위조부품 방지 체계)
3. 해당 부품에 대한 **추가 수입검사** 실시:
   - 외관 검사(Visual Inspection) — 마킹, 패키지, 날짜 코드 확인
   - X-ray 또는 전기적 시험 (전자부품의 경우)
   - CoC, 시험성적서 진위 확인
4. 검사 결과 기록 보존 (최소 10년 또는 고객 요구 기간)
5. 고객 사전 통보 및 승인 (계약서에 명시된 경우)

## 12.4 위조부품 발견 시 ASL 조치

위조부품 또는 위조 의심 부품이 발견된 경우:

1. **즉시 격리**: 해당 부품 및 동일 로트 전량 격리
2. **ASL 즉시 Suspended**: 해당 공급자 ASL 상태를 즉시 Suspended로 전환
3. **고객 통보**: 24시간 이내 해당 고객 및 관련 당국(해당 시) 통보
4. **조사 실시**: 공급자 공동 조사 또는 독자 조사 수행
5. **최종 판정**:
   - 위조 확인 시: **즉시 Removed** 처리 (영구 거래 중단, 재등재 불가)
   - 위조 의심 해소 시: 시정조치 확인 후 Conditional 복귀 가능
6. **GIDEP 보고**: 해당 시 관련 산업 데이터베이스(GIDEP 등)에 보고
