# 1. 목적

본 절차서는 M&C Electronics VINA에서 고객 또는 외부 공급자가 제공한 소유물(자재, 장비, 도구, 지적재산, 개인정보 등)을 식별, 검증, 보호 및 안전하게 관리하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.5.3 요구사항에 따른 고객/공급자 소유물의 체계적 관리
- 소유물의 분실, 손상, 부적합을 방지하기 위한 보호 체계 수립
- 소유물에 문제 발생 시 고객/공급자에게 즉시 통보하는 체계 확립
- 항공우주 분야 특수 소유물(GFE, GFM, 기술자료 등)의 관리 강화

---

# 2. 적용 범위

본 절차서는 M&C Electronics VINA의 QMS 내에서 취급하는 모든 고객 및 외부 공급자 소유물에 적용한다.

| 구분 | 적용 대상 |
|------|----------|
| 고객 제공 자재 | 원자재, 부품, 반제품 (GFM) |
| 고객 제공 장비 | 시험장비, 치공구, 금형 (GFE) |
| 고객 도면/기술자료 | 도면, 사양서, 3D 모델, 기술 데이터 |
| 지적재산 | 특허, 설계 노하우, 소프트웨어 라이선스 |
| 개인정보 | 고객 담당자 연락처, 개인 데이터 |
| 공급자 제공 도구 | 공급자 소유 검사 게이지, 전용 치공구 |
| 포장재 | 고객 지정 전용 포장재, 반환용 용기 |

**적용 제외**: 당사가 구매하여 자체 소유한 자재/장비는 본 절차서의 적용 범위에 해당하지 않는다.

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 고객 소유물 (Customer Property) | 고객이 당사에 제공한 자재, 장비, 도구, 도면, 지적재산, 개인정보 등 고객에게 소유권이 있는 일체의 물품 및 정보 |
| GFE (Government/Customer Furnished Equipment) | 고객(또는 정부)이 제공한 장비, 치공구, 시험기기 등. 계약에 의해 당사에 대여되며 소유권은 고객에게 있음 |
| GFM (Government/Customer Furnished Material) | 고객(또는 정부)이 제공한 원자재, 부품, 반제품 등. 제품 제조에 사용되며 소유권은 고객에게 있음 |
| Bailment (위탁 보관) | 소유자(위탁자)가 타인(수탁자)에게 물품을 맡기는 법적 관계. 수탁자는 선량한 관리자의 주의의무를 부담 |
| 외부 공급자 소유물 | 외부 공급자가 당사에 제공한 도구, 게이지, 금형 등 공급자에게 소유권이 있는 물품 |
| 지적재산 (Intellectual Property, IP) | 도면, 사양서, 소프트웨어, 특허, 노하우 등 무형의 재산권이 인정되는 정보 자산 |
| 수출통제 대상물 (Export Controlled Item) | ITAR, EAR 등 수출관리 법규에 의해 취급이 규제되는 기술자료, 부품, 장비 |

# 5. 책임과 권한

| 역할 | 책임 |
|------|------|
| **품질책임자 (QMR)** | 고객/공급자 소유물 관리 체계 총괄; 고객 통보 최종 승인; 분쟁 발생 시 대응 총괄 |
| **품질팀장** | 소유물 대장(Register) 유지·관리; 정기 재고 확인(Inventory) 주관; 손상/분실 보고서 검토 |
| **자재/창고 담당자** | 소유물 입고 검수 및 식별 표시; 보관 장소 배정 및 보존; 입출고 기록 유지 |
| **생산부서장** | 공정 중 소유물 취급 관리; 작업자 소유물 취급 교육; 손상/이상 발생 시 즉시 보고 |
| **기술부서장** | 기술자료(도면/사양) 접근 통제; 수출통제 대상 기술자료 관리; 지적재산 보호 |
| **구매 담당자** | 공급자 소유물 관련 계약 조건 확인; 공급자 소유물 반환 조율 |
| **영업 담당자** | 고객 소유물 수령/반환 조율; 고객 통보 창구; 계약 상 소유물 조건 확인 |

# 6. 고객/공급자 소유물 범위

## 6.1 유형 자산 (Tangible Property)

| 분류 | 예시 | 관리 특성 |
|------|------|----------|
| 원자재/부품 (GFM) | 고객 지급 PCB, 커넥터, IC 등 | 입고 수량 확인, 로트 추적, 재고 관리 |
| 장비/치공구 (GFE) | 고객 제공 시험장비, 전용 지그 | 캘리브레이션 관리, 사용 이력 기록 |
| 금형/몰드 | 고객 소유 사출 금형, 프레스 다이 | 사용 횟수 관리, 예방 보전, 보관 조건 유지 |
| 포장재/용기 | 고객 지정 전용 운송 용기, 정전기 방지 트레이 | 파손 관리, 반환 수량 추적 |

## 6.2 무형 자산 (Intangible Property)

| 분류 | 예시 | 관리 특성 |
|------|------|----------|
| 도면/기술자료 | 2D/3D 도면, 사양서, BOM | 접근 통제, 개정 관리, 인쇄 통제 |
| 소프트웨어/라이선스 | 시험 소프트웨어, 프로그래밍 파일 | 라이선스 조건 준수, 복제 방지 |
| 지적재산 | 특허, 설계 노하우, 공정 비밀 | 비밀유지계약(NDA), 접근 제한 |
| 개인정보 | 고객 담당자 연락처, 방문자 정보 | 개인정보보호법 준수, 목적 외 사용 금지 |

## 6.3 공급자 소유물

| 분류 | 예시 | 관리 특성 |
|------|------|----------|
| 검사 게이지 | 공급자 제공 전용 게이지, Go/No-go 게이지 | 캘리브레이션 상태 확인, 반환 시기 관리 |
| 전용 치공구 | 공급자 소유 가공 지그 | 사용 이력, 보관 조건 유지 |

# 7. 입고 및 식별

## 7.1 입고 절차

```
고객/공급자 → 소유물 인도 → 자재/창고 담당자 접수
                                    ↓
                              인도서/송장 확인
                              (품명, 수량, 상태)
                                    ↓
                              외관 검사 및 수량 확인
                                    ↓
                         ┌─── 이상 없음 ───┐
                         ↓                  ↓
                    입고 확인서 발행     이상 발견 시
                    소유물 대장 등록     → 9절 통보 절차
                         ↓
                    식별 표시 부착
                         ↓
                    지정 보관 장소 이동
```

## 7.2 입고 시 확인 사항

| 확인 항목 | 세부 내용 |
|----------|----------|
| 인도 문서 | 인도서(Delivery Note), 패킹리스트(Packing List) 대조 |
| 수량 확인 | 인도서 기재 수량과 실물 수량 일치 확인 |
| 외관 상태 | 운송 중 손상, 오염, 변형 여부 확인 |
| 포장 상태 | 포장 파손, 습기 침투, ESD 보호 상태 확인 |
| 식별 정보 | 품번, 로트번호, 시리얼번호 등 식별 정보 확인 |
| 유효기간 | Shelf Life 해당 품목의 유효기간 확인 |

## 7.3 식별 표시

고객/공급자 소유물에는 다음과 같이 명확하게 식별 표시를 부착한다:

| 표시 항목 | 내용 |
|----------|------|
| 식별 라벨 색상 | **노란색** (고객 소유물), **주황색** (공급자 소유물) |
| 라벨 기재 사항 | "고객 소유물" 또는 "공급자 소유물" 문구, 소유자명, 품번, 수량, 입고일 |
| 보관 장소 표시 | 전용 보관 구역에 "Customer Property" 또는 "Supplier Property" 표시 |
| 기술자료 표시 | "CUSTOMER PROPRIETARY" 또는 "SUPPLIER PROPRIETARY" 워터마크/스탬프 |

## 7.4 소유물 대장 (Property Register) 등록

입고된 소유물은 **고객/공급자 소유물 대장 (QF-813-01)**에 다음 정보를 등록한다:

- 소유자명 (고객/공급자)
- 품명 및 품번
- 수량/수량 단위
- 입고일 및 입고 근거 (계약번호, PO번호)
- 보관 장소
- 상태 (양호/손상/사용 중/반환 완료)
- 책임 담당자

# 8. 검증 및 보관

## 8.1 입고 검증

| 소유물 유형 | 검증 방법 | 검증 기준 |
|------------|----------|----------|
| 원자재/부품 (GFM) | 수입검사 절차 준용 (외관, 수량, CoC 확인) | 고객 지정 사양, 인도서 기재 사양 |
| 장비/치공구 (GFE) | 기능 점검, 캘리브레이션 상태 확인 | 고객 제공 매뉴얼, 교정 성적서 |
| 금형 | 외관 검사, 시사출 검증 (해당 시) | 금형 사양서, 이전 사용 이력 |
| 도면/기술자료 | 개정 상태 확인, 문서 완전성 점검 | 최신 개정 여부, 배포 목록 |

**참고**: 고객/공급자 소유물에 대한 입고 검증은 당사의 수입검사를 대체하지 않는다. 검증은 소유물의 인도 상태를 확인하는 것이며, 최종 제품 적합성 검증은 별도로 수행한다.

## 8.2 보관 관리

| 관리 항목 | 관리 기준 |
|----------|----------|
| 전용 보관 구역 | 당사 자재와 분리된 전용 구역 또는 전용 선반에 보관 |
| 보관 조건 | 소유물 특성에 맞는 온도, 습도, ESD 보호 조건 유지 |
| 접근 통제 | 인가된 담당자만 접근 가능, 기술자료는 잠금 보관 |
| 정기 점검 | **분기별** 재고 확인(Inventory) 및 상태 점검 실시 |
| FIFO 원칙 | GFM의 경우 선입선출 원칙 적용 (유효기간 관리 포함) |

## 8.3 사용 중 관리

- GFM은 생산 투입 시 불출 기록을 남기고, 사용 잔량을 추적한다.
- GFE는 사용 전후 상태를 점검하고, 사용 이력을 기록한다.
- 금형은 사용 횟수(Shot Count)를 관리하고, 예방 보전 주기에 따라 점검한다.
- 기술자료는 최신 개정 상태를 유지하며, 구 개정판은 회수/폐기한다.

# 9. 손상/분실/부적합 통보 절차

## 9.1 이상 발견 시 즉시 조치

고객/공급자 소유물에 손상, 분실, 부적합 또는 사용에 부적절한 상태가 발견된 경우, **즉시** 다음 절차를 수행한다:

```
이상 발견 (발견자)
        ↓
즉시 사용 중지 및 격리
        ↓
부서장에게 보고 (구두 즉시, 서면 24시간 이내)
        ↓
품질팀장에게 통보
        ↓
손상/분실/부적합 보고서 작성 (QF-813-02)
        ↓
품질책임자(QMR) 검토 및 승인
        ↓
고객/공급자에게 즉시 통보 ← AS9100D 8.5.3 필수 요건
        ↓
고객/공급자 지시에 따른 후속 조치
        ↓
기록 완료 및 소유물 대장 상태 업데이트
```

## 9.2 고객/공급자 통보

| 항목 | 기준 |
|------|------|
| 통보 시한 | 이상 발견 후 **24시간 이내** (안전 관련 사항은 즉시) |
| 통보 방법 | 서면 통보 (이메일 또는 공문) — 구두 통보 후 서면 확인 필수 |
| 통보 내용 | 소유물 식별 정보, 이상 내용(손상/분실/부적합), 발견 일시 및 상황, 현재 조치 상태 |
| 통보 책임 | 영업 담당자가 고객에게, 구매 담당자가 공급자에게 통보 (QMR 승인 후) |
| 후속 조치 | 고객/공급자 지시(교환, 수리, 반환, 폐기 등)에 따라 처리 |

## 9.3 보고서 내용 (QF-813-02)

손상/분실/부적합 보고서에는 다음 사항을 기록한다:

- 소유물 식별 정보 (소유자, 품명, 품번, 수량)
- 이상 유형 (손상 / 분실 / 부적합 / 기타)
- 발견 일시, 장소, 발견자
- 이상 내용 상세 설명 (사진 첨부 권장)
- 추정 원인
- 현재 격리/봉쇄 상태
- 고객/공급자 통보 일시 및 방법
- 고객/공급자 지시 사항 및 후속 조치 결과

# 10. 반환 관리

## 10.1 반환 절차

| 단계 | 활동 | 책임 |
|------|------|------|
| 1 | 반환 사유 확인 (계약 종료, 고객 요청, 프로젝트 완료 등) | 영업/구매 담당자 |
| 2 | 반환 대상 소유물 목록 작성 및 상태 확인 | 자재/창고 담당자 |
| 3 | 소유물 상태 점검 (손상 여부, 수량 일치 확인) | 품질팀 |
| 4 | 반환 포장 (소유물 특성에 적합한 포장 방법 적용) | 자재/창고 담당자 |
| 5 | 반환 인도서 작성 및 고객/공급자 확인 | 영업/구매 담당자 |
| 6 | 소유물 대장 상태 업데이트 ("반환 완료") | 품질팀 |

## 10.2 반환 시 확인 사항

- 소유물 대장과 실물의 수량 일치 확인
- 반환 시 소유물 상태 기록 (사진 촬영 권장)
- 고객/공급자의 반환 확인서(Receipt Acknowledgment) 수령
- 관련 기술자료(도면, 사양서)의 반환 또는 폐기 확인

## 10.3 미반환 소유물 관리

- 장기 보관(6개월 이상 미사용) 소유물은 고객/공급자에게 보관 지속 여부를 확인한다.
- 고객/공급자와 연락이 불가한 경우, 서면 통보(등기우편 등) 후 합리적 기간 동안 보관한다.
- 폐기 요청 시 고객/공급자의 서면 승인 후 폐기하며, 폐기 기록을 유지한다.

# 11. 항공우주 특수 요구사항

## 11.1 GFE/GFM 관리

항공우주 고객이 제공하는 GFE/GFM에 대해 다음 추가 요건을 적용한다:

| 항목 | 요구사항 |
|------|----------|
| 입고 검증 강화 | CoC(적합성 인증서), 자재 인증서(Material Certificate) 확인 필수 |
| 추적성 | GFM 사용 로트별 추적성 유지, 최종 제품까지 연결 |
| 잔여 자재 관리 | GFM 잔여 수량 정확히 관리, 고객 지시에 따라 반환 또는 폐기 |
| GFE 교정 관리 | 고객 제공 장비의 교정 상태 확인, 교정 만료 시 사용 금지 및 고객 통보 |
| 기록 보존 기간 | 계약에 명시된 기간 또는 최소 **10년** (항공우주 일반 요건) |

## 11.2 도면/기술자료 보호

| 관리 항목 | 세부 요건 |
|----------|----------|
| 접근 통제 | Need-to-Know 원칙 적용, 접근 권한자 목록 유지 |
| 보관 | 물리적 자료: 잠금 캐비닛, 전자 자료: 암호화 저장 및 접근 로그 |
| 인쇄 통제 | 인쇄물에 "CUSTOMER PROPRIETARY" 표시, 인쇄 부수 관리, 불필요 인쇄물 파쇄 |
| 개정 관리 | 고객 발행 최신본만 사용, 구 개정판 즉시 회수·폐기 |
| 복제 금지 | 고객 서면 승인 없이 복제, 전송, 제3자 공유 금지 |
| 전자 매체 | USB, 이메일 전송 시 암호화 필수, 전송 기록 유지 |

## 11.3 수출통제 대상물 관리

고객이 제공한 소유물 중 ITAR, EAR 등 수출통제 규정 적용 대상에 대해:

- 수출통제 분류 번호(ECCN) 또는 USML 카테고리를 확인하고 기록한다.
- 수출통제 대상 소유물에는 **"EXPORT CONTROLLED"** 표시를 추가한다.
- 해당 국가의 수출통제 법규를 준수하며, 인가되지 않은 국적의 인원에게 접근을 허용하지 않는다.
- 수출통제 관련 의문 사항은 고객에게 확인 후 처리한다.
- 수출통제 교육을 관련 담당자에게 실시한다.

## 11.4 지적재산(IP) 보호

- 고객과 체결한 비밀유지계약(NDA)의 조건을 준수한다.
- 고객의 설계 노하우, 공정 비밀 등을 당사 제품 개발에 무단 활용하지 않는다.
- 프로젝트 종료 시 고객 지적재산 관련 자료를 반환하거나 고객 지시에 따라 파기한다.
- 지적재산 침해 의심 사례 발견 시 QMR에게 즉시 보고한다.
