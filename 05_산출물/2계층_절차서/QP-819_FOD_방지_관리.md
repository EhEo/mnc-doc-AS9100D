# 1. 목적

본 절차서는 M&C Electronics VINA의 항공우주 제품 제조, 검사, 보관 및 운송 전 과정에서 FOD(Foreign Object Debris/Damage, 이물질 손상)를 예방하고 관리하기 위한 체계적 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.5.1 항공우주 산업 요건에 따른 FOD 방지 프로그램 수립 및 운영
- 제품, 부품, 어셈블리에 대한 이물질 오염 및 손상 예방
- FOD 통제 구역(FOD Critical Zone) 지정 및 관리 기준 확립
- 공구 관리(Tool Accountability), 부품 수량 관리(Hardware Accountability) 체계 운영
- FOD 인식 교육 및 전 임직원 예방 문화 정착
- **CAR-011 (FOD 관리 프로그램 부재, FOD 통제 구역 미지정, 교육 미실시) 시정조치 반영**

> **CAR-011 연계**: 갭 분석에서 "FOD(이물질) 관리 프로그램 없음, FOD 통제 구역 미지정, 교육 미실시"로 Major 부적합이 식별됨. 본 절차서는 Fishbone(6M) 분석을 통해 도출된 근본 원인(항공우주 FOD 요구사항 인식 부재, 고객 요구사항 검토 미흡)에 대한 시정조치로 제정됨.

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| FOD (Foreign Object Debris) | 제품, 시스템 또는 장비에 잠재적 손상을 유발할 수 있는 이물질(금속 조각, 와이어 잔재, 볼트/너트, 공구, 개인 물품, 포장재 파편, 먼지 등) |
| FOD (Foreign Object Damage) | 이물질(Foreign Object Debris)에 의해 제품, 시스템 또는 장비에 발생한 손상 |
| FOD 구역 (FOD Area) | FOD 방지 활동이 적용되는 일반 작업 구역으로, 기본 청결 기준과 FOD 예방 조치가 시행되는 영역 |
| FOD Critical Zone | FOD 발생 시 제품 안전성 또는 기능에 직접적 영향을 미칠 수 있는 고위험 구역. 강화된 통제 기준 적용 (예: 정밀 조립 라인, 클린룸, 개구부 작업 구역) |
| Tool Accountability (공구 수량 관리) | 작업 전후 공구 수량을 확인하여 공구가 제품 내부에 유실되지 않았음을 보장하는 활동 |
| Hardware Accountability (부품 수량 관리) | 작업에 투입되는 볼트, 너트, 와셔, 리벳 등 소형 부품의 수량을 작업 전후 확인하여 미사용 부품이 제품 내에 잔류하지 않음을 보장하는 활동 |
| FOD Walk-down | 작업 구역을 체계적으로 순회하며 이물질 유무를 검사하는 활동 |
| 개구부 보호 (Opening Protection) | 제품의 개구부(구멍, 튜브, 커넥터 등)를 커버, 플러그, 캡, 테이프 등으로 밀봉하여 이물질 유입을 방지하는 활동 |
| FOD 태그 (FOD Tag) | 개구부 보호물(커버/플러그)에 부착하는 식별 표식으로, 보호 일자, 담당자, 제거 조건 등을 기재 |

# 6. FOD 방지 프로그램

## 6.1 FOD 방지 정책

M&C Electronics VINA는 항공우주 제품의 안전성과 품질을 보장하기 위하여 다음과 같은 FOD 방지 정책을 수립하고 이행한다:

- 모든 항공우주 제품의 제조, 검사, 보관, 운송 과정에서 FOD가 발생하지 않도록 예방 활동을 우선한다
- FOD 방지 활동은 전 임직원의 책임이며, 경영진은 이를 위한 자원과 환경을 제공한다
- FOD 발생 시 즉시 보고하며, 은폐 또는 방치 시 징계 대상이 된다
- FOD 방지 프로그램의 유효성을 정기적으로 평가하고 개선한다

## 6.2 FOD 인식 교육

| 교육 구분 | 대상 | 주기 | 내용 |
|----------|------|------|------|
| 신규 입사자 교육 | 전 신규 입사자 | 입사 시 (배치 전 필수) | FOD 정의, 위험성, 사례, 기본 규정 |
| 정기 교육 | 전 임직원 | 연 1회 이상 | FOD 방지 절차, 사례 공유, 규정 변경사항 |
| 보수 교육 | FOD 발생 관련자 | FOD 발생 시 | 발생 사례 분석, 재발 방지 대책 |
| 방문자/외주 교육 | 방문자, 외주 인력 | 입장 시 (간이) | FOD 구역 규정, 개인 물품 관리, 금지사항 |

## 6.3 FOD 구역 지정

### 6.3.1 FOD 구역 분류

| 등급 | 구역 | 통제 수준 | 표시 |
|------|------|----------|------|
| **Level 1 — FOD Critical Zone** | 정밀 조립 라인, 클린룸, 개구부 작업 구역, 최종 검사 구역 | 강화 통제 (출입 제한, 개인 물품 전면 금지, 작업 전후 FOD walk-down 필수) | 적색 표지판 + 바닥 적색 라인 |
| **Level 2 — FOD Controlled Area** | 일반 생산 라인, CNC 가공, 프레스, 납땜 구역 | 기본 통제 (개인 물품 제한, 정기 FOD 검사, 작업 후 청소 필수) | 황색 표지판 + 바닥 황색 라인 |
| **Level 3 — FOD Awareness Area** | 원자재 창고, 완제품 창고, 포장 구역, 시험실 | 인식 수준 (기본 청결 유지, 정기 순회 검사) | 청색 표지판 |

### 6.3.2 FOD 구역 표시 요건

- 각 FOD 구역 입구에 FOD 등급 표지판 설치
- 바닥 라인 테이프(Level 1: 적색, Level 2: 황색)로 구역 경계 표시
- FOD 구역 내 "FOD 예방 수칙" 게시물 부착
- FOD 구역도(Site Map)를 사내 게시 및 교육 시 활용

## 6.4 청결 기준

| 구역 등급 | 청소 주기 | 청결 기준 |
|----------|----------|----------|
| Level 1 | 작업 시작 전/후, 교대 시, 작업 중단 시 | 육안으로 이물질 미확인; 바닥, 작업대 위 잔재물 없음; 개구부 보호물 정상 |
| Level 2 | 작업 종료 후, 교대 시 | 금속 칩/절삭유 잔재 제거; 작업대/바닥 청소 완료 |
| Level 3 | 일 1회 이상 | 통로/보관 구역 정리정돈 완료; 파손 포장재/먼지 제거 |

# 7. FOD 예방 활동

## 7.1 개인 물품 관리

| 구분 | FOD Critical Zone (Level 1) | FOD Controlled Area (Level 2) | FOD Awareness Area (Level 3) |
|------|---------------------------|------------------------------|------------------------------|
| 휴대폰 | 반입 금지 | 지정 보관함에 보관 | 주의하여 휴대 |
| 음식/음료 | 반입 금지 | 반입 금지 | 지정 장소에서만 |
| 액세서리(반지, 목걸이 등) | 착용 금지 | 착용 금지 | 주의 |
| 주머니 내 물품 | 전량 반출 (투명 주머니 사용) | 낙하 방지 조치 | 주의 |
| 필기구 | 끈 부착 또는 반입 금지 | 끈 부착 권장 | 자유 |

## 7.2 공구 관리 (Tool Accountability)

### 7.2.1 공구 수량 확인 절차

```
┌──────────────────────┐
│ 1. 작업 시작 전      │
│    공구 목록 확인     │
│    (Shadow Board 또는 │
│     공구 체크시트)    │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 2. 공구 수량 기록    │
│    (총 수량, 품명)   │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 3. 작업 수행         │
│    (공구 사용 시     │
│     지정 위치 반납)  │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 4. 작업 완료 후      │
│    공구 수량 재확인   │
└──────────┬───────────┘
           │
     ┌─────┴─────┐
     │           │
     ▼           ▼
┌─────────┐  ┌──────────────────┐
│ 일치    │  │ 불일치           │
│ → 기록  │  │ → 작업 중단      │
│   완료  │  │ → 즉시 수색      │
└─────────┘  │ → FOD 보고서 작성│
             │ → 제품 격리       │
             └──────────────────┘
```

### 7.2.2 공구 관리 요건

- **Shadow Board**: 공구 형상이 표시된 보관판 사용 (빈 자리 즉시 확인 가능)
- **공구 체크시트(F-816-01)**: 작업 시작/종료 시 공구 수량을 기록
- **파손 공구**: 파손 시 전체 조각 수거 확인 후 교체. 파편 미회수 시 FOD 보고서 작성
- **개인 공구**: FOD 구역 내 개인 공구 사용 금지 (회사 지급 공구만 허용)
- **공구 표식**: 각 공구에 일련번호 또는 식별 표시 부착

## 7.3 부품 수량 관리 (Hardware Accountability)

| 절차 | 내용 |
|------|------|
| 작업 전 수량 확인 | 투입 부품(볼트, 너트, 와셔, 리벳, 클립 등) 수량을 BOM 기준으로 확인·기록 |
| 작업 중 관리 | 부품을 분류 트레이에 보관; 바닥 낙하 부품 즉시 수거; 여분 부품 별도 관리 |
| 작업 후 수량 확인 | 설치 수량 + 잔여 수량 = 최초 불출 수량 확인 |
| 불일치 시 | 작업 중단 → 수색 → 미발견 시 FOD 보고서 작성 → 제품 격리 |

## 7.4 개구부 보호 (Cover/Plug)

| 항목 | 요건 |
|------|------|
| 보호 대상 | 튜브, 배관, 커넥터, 포트, 구멍, 덕트 등 이물질 유입 가능한 모든 개구부 |
| 보호 방법 | 전용 캡(Cap), 플러그(Plug), 커버(Cover), 보호 테이프 사용 |
| FOD 태그 | 모든 보호물에 FOD 태그 부착 (일자, 담당자, 제거 조건 기재) |
| 제거 시점 | 후속 작업 직전에만 제거 → 작업 완료 후 즉시 재보호 |
| 보호물 검사 | 보호물 상태(파손, 이탈, 오염 여부) 정기 점검 |
| 보호물 수량 관리 | 설치/제거 시 보호물 수량 확인 (보호물 자체가 FOD가 되지 않도록) |

## 7.5 작업 전후 청소

| 시점 | 활동 | 담당 |
|------|------|------|
| 작업 시작 전 | 작업대/장비/주변 청소; 이전 작업 잔재물 제거 확인; FOD 유무 점검 | 작업자 |
| 작업 중 | 절삭칩/잔재물 수시 제거; 사용 완료 자재 즉시 반납/폐기 | 작업자 |
| 작업 완료 후 | 작업 구역 전체 청소; 공구/부품 수량 확인; FOD 검사 실시 | 작업자 + 반장 |
| 교대 시 | 인수인계 시 FOD 상태 확인; 청소 상태 확인 서명 | 인계자/인수자 |

## 7.6 쓰레기/잔해 관리

- 각 작업 구역에 밀폐형 쓰레기통 배치 (뚜껑 필수)
- 금속 잔재(칩, 버, 절삭분)와 일반 쓰레기 분리 수거
- 쓰레기통은 70% 이상 차기 전에 교체
- 포장재(스티로폼, 비닐, 종이) 즉시 처리 — 현장 방치 금지
- 압축공기(에어건) 사용 시 비산 방지 조치 실시 (FOD Critical Zone 내 에어건 사용 제한)

# 8. FOD 구역 관리

## 8.1 FOD Critical Zone 지정

### 8.1.1 지정 기준

다음 조건에 해당하는 구역을 FOD Critical Zone(Level 1)으로 지정한다:

| 지정 기준 | 예시 |
|----------|------|
| 개구부가 노출된 상태로 작업하는 구역 | 배관 조립, 커넥터 연결 작업대 |
| 항공 안전 핵심 부품(Safety-Critical Item) 취급 구역 | 비행 제어 관련 부품 조립 |
| 밀봉/밀폐 전 최종 조립 구역 | 하우징 조립, 기밀 봉인 작업 |
| 정밀 전자 부품 조립/시험 구역 | PCB 실장, 와이어 하네스 제작 |
| 고객이 FOD Critical Zone 지정을 요구한 구역 | 고객 사양서에 명시된 경우 |

### 8.1.2 지정 절차

1. FOD 코디네이터(품질팀장)가 현장 평가 실시
2. FOD 리스크 수준에 따라 구역 등급(Level 1/2/3) 결정
3. QMR 승인 후 구역 지정 공포
4. 구역 표시(표지판, 바닥 라인) 설치
5. 해당 구역 작업자 대상 특별 교육 실시

## 8.2 출입 통제

| 구역 | 출입 통제 기준 |
|------|--------------|
| Level 1 (Critical Zone) | 출입 로그 작성 필수; 승인된 작업자만 출입; 방문자 반드시 동반 출입; 입장 전 개인 물품 점검; 출입 시 FOD 점검 (신발 이물질 등) |
| Level 2 (Controlled Area) | 작업 관련자만 출입; 방문자 동반 출입 권장; 음식/음료 반입 금지 |
| Level 3 (Awareness Area) | 자유 출입; FOD 인식 표지판 확인 |

## 8.3 환경 관리

| 관리 항목 | 기준 |
|----------|------|
| 조명 | 이물질 육안 식별이 가능한 충분한 조도 확보 (작업면 최소 500 lux) |
| 바닥 | 균열, 파손 없는 매끈한 바닥재; 청소 용이한 재질; 정기 점검 |
| 작업대 | 매끈한 표면; 가장자리 턱(lip) 설치로 부품 낙하 방지 |
| 환기 | 먼지/입자 유입을 최소화하는 양압 또는 필터링 시스템 (Level 1) |
| 배수구 | 격자 커버 설치; 부품 유실 방지 망 적용 |

# 9. FOD 검사

## 9.1 FOD Walk-down

| 항목 | 내용 |
|------|------|
| 정의 | 지정된 경로를 따라 체계적으로 작업 구역을 순회하며 이물질 유무를 검사하는 활동 |
| 주기 | Level 1: 작업 시작 전/후 매회, Level 2: 일 1회, Level 3: 주 1회 |
| 수행자 | FOD 코디네이터가 지정한 검사원 또는 해당 구역 반장 |
| 방법 | 정해진 Walk-down 경로를 따라 육안 검사; 필요 시 거울, 손전등, 내시경 활용 |
| 기록 | F-816-01 FOD 검사 기록서에 검사 일시, 구역, 결과, 검사원 서명 기록 |

## 9.2 작업 전후 검사

| 시점 | 검사 내용 |
|------|----------|
| 작업 시작 전 | 작업대/장비 위 이물질 유무; 개구부 보호물 상태; 공구/부품 수량 |
| 작업 완료 후 | 제품 내외부 이물질 유무; 개구부 보호 재적용; 공구/부품 수량 일치; 청소 완료 상태 |
| 밀봉/봉인 전 | 밀봉 대상 내부 최종 FOD 검사 (필수 — 품질 검사원 확인 서명) |

# 10. FOD 교육/인식

## 10.1 연간 교육 계획

| 교육 항목 | 대상 | 주기 | 시간 | 교육 내용 |
|----------|------|------|------|----------|
| FOD 기초 교육 | 전 임직원 | 연 1회 | 2시간 | FOD 정의/위험성, 방지 절차, 구역 규정, 개인 책임, 보고 절차 |
| FOD 심화 교육 | FOD 구역 작업자 | 연 1회 | 4시간 | Tool/Hardware Accountability, 개구부 보호, Walk-down, 사례 분석 |
| FOD 코디네이터 교육 | FOD 코디네이터 | 연 1회 | 8시간 | FOD 프로그램 관리, 감사 기법, NAS 412/SAE AS9146 요건, 원인 분석 |

## 10.2 신규 입사자 교육

- 배치 전 필수 이수 (미이수 시 FOD 구역 출입 불가)
- 교육 내용: FOD 정의, 위험성, 실제 사고 사례(사진/동영상), 기본 규정, 보고 절차
- 교육 후 이해도 확인 (퀴즈 또는 실습)
- 교육 이수 기록: QP-703(역량/교육훈련 관리) 연계 관리

## 10.3 FOD 사례 공유

| 활동 | 주기 | 내용 |
|------|------|------|
| FOD 사례 게시 | 발생 시 즉시 + 월간 | 사내 발생 사례 및 산업계 사고 사례를 게시판에 공유 |
| FOD 교훈 미팅 | 분기 1회 | 분기 내 FOD 발생 현황 리뷰, 교훈 도출, 개선 사항 논의 |
| FOD 우수 사례 포상 | 연 1회 | FOD 방지 우수 부서/개인 포상 (인식 제고) |

## 10.4 FOD 인식 캠페인

- FOD 방지 포스터를 각 생산 구역, 출입구, 휴게실에 게시
- "Clean As You Go" 슬로건 적용
- FOD 인식의 달(연 1회) 지정 운영
- FOD 자가 점검 카드 배포 (작업자 휴대용)
