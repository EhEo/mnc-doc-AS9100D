# 1. 목적

본 절차서는 M&C Electronics VINA에서 프로세스의 출력(원자재, 재공품, 완제품)을 의도된 용도에 적합한 상태로 보존하기 위한 식별, 취급, 포장, 보관 및 운송에 관한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.5.4 요구사항에 따른 제품 보존 체계 수립
- 생산 전 과정(입고~출하)에 걸친 제품 품질 열화 방지
- 항공우주 부품의 특수 보존 요건(ESD, 습기, 온도, 수명 제한 등) 관리
- FOD(이물질 손상) 방지 활동과 연계한 보존 관리 강화

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 보존 (Preservation) | 프로세스의 출력을 의도된 용도에 적합한 상태로 유지하기 위한 활동. 식별, 취급, 오염관리, 포장, 보관, 전달 또는 운송, 보호를 포함 |
| FIFO (First In, First Out) | 선입선출 — 먼저 입고된 자재를 먼저 출고하는 재고 관리 원칙 |
| Shelf Life (유효기간) | 제조자가 지정한 자재/부품의 사용 가능 기간. 유효기간 초과 시 사용 금지 |
| ESD (Electrostatic Discharge) | 정전기 방전 — 전자 부품/조립품에 손상을 줄 수 있는 정전기 에너지의 방출 |
| MSD (Moisture Sensitive Device) | 습기 민감 소자 — 습기 흡수 시 리플로우 솔더링 중 손상 가능한 부품 |
| MSL (Moisture Sensitivity Level) | 습기 민감도 등급 — IPC/JEDEC J-STD-020에 따른 1~6등급 분류 |
| Floor Life | MSD가 방습 포장 개봉 후 대기 노출 가능한 최대 허용 시간 |
| Desiccant (건조제) | 방습 포장 내 습기를 흡수하기 위해 사용하는 물질 (실리카겔 등) |
| HIC (Humidity Indicator Card) | 습도 표시 카드 — 방습 포장 내 습도 상태를 표시하는 카드 |
| FOD (Foreign Object Debris/Damage) | 이물질 잔류물/손상 — 제품에 잠재적 손상을 줄 수 있는 외래 물질 |

# 5. 책임과 권한

| 역할 | 책임 |
|------|------|
| **품질책임자 (QMR)** | 보존 관리 체계 총괄; 특수 보존 요건 적합성 승인; 보존 관련 고객 불만 대응 |
| **품질팀장** | 보존 기준 수립 및 유지; 보존 상태 정기 점검 주관; 부적합 보존 상태 시정 지시 |
| **자재/창고 담당자** | 자재 입출고 관리; FIFO 실행; Shelf Life 관리; 보관 환경(온습도) 모니터링 |
| **생산부서장** | 공정 중 제품 취급 관리; 작업자 보존 교육; ESD/FOD 관리 실행 |
| **포장 담당자** | 포장 사양서에 따른 포장 수행; 포장재 관리; 특수 포장 요건 실행 |
| **물류/운송 담당자** | 운송 중 제품 보호; 운송 조건 확인; 운송 손상 보고 |
| **기술부서장** | 특수 보존 조건 설정; 포장 사양서 승인; 자재별 보관 조건 결정 |

# 6. 보존 활동 범위

제품 보존은 다음 5대 활동으로 구성된다:

| 보존 활동 | 내용 | 적용 단계 |
|----------|------|----------|
| **식별 (Identification)** | 제품/자재의 상태, 검사 결과, 추적성 정보를 명확히 표시 | 전 과정 |
| **취급 (Handling)** | 제품/자재를 안전하게 들어올리고, 이동하고, 배치하는 활동 | 전 과정 |
| **포장 (Packaging)** | 제품/자재를 운송 및 보관 중 손상으로부터 보호하는 활동 | 입고, 공정 간, 출하 |
| **보관 (Storage)** | 적절한 환경 조건에서 제품/자재를 안전하게 저장하는 활동 | 입고 후 ~ 출하 전 |
| **운송 (Transportation)** | 제품/자재를 손상 없이 목적지까지 전달하는 활동 | 사내 이동, 출하 운송 |

# 7. 입고 보존

## 7.1 수입검사 후 보관 절차

```
자재/부품 입고
      ↓
수입검사 (QP-810 준용)
      ↓
 ┌─── 합격 ───┐        ┌─── 불합격 ───┐
 ↓             ↓        ↓               ↓
합격 라벨 부착   부적합 처리 (QP-818)
 ↓
보관 조건 확인 (일반/특수)
 ↓
 ┌─── 일반 보관 ───┐    ┌─── 특수 보관 ───┐
 ↓                  ↓    ↓                  ↓
일반 창고 입고       방습 보관 / ESD 보관
 ↓                  온도 관리 보관
FIFO 위치 배정      건조 캐비닛 등
 ↓
재고 시스템 등록
```

## 7.2 입고 자재 보관 기준

| 자재 유형 | 보관 조건 | 보관 장소 |
|----------|----------|----------|
| 일반 전자부품 | 온도 15~30°C, 습도 40~70%RH | 일반 부품 창고 |
| 습기 민감 부품 (MSD) | 방습 포장 유지 또는 건조 캐비닛 (<10%RH) | 건조 보관실 |
| ESD 민감 부품 | ESD 보호 포장 유지, 정전기 방지 선반 | ESD 보호 구역 |
| PCB (Bare Board) | 진공 포장 유지, 온도 15~30°C, 습도 <70%RH | 일반/건조 보관실 |
| 화학물질 (솔더, 플럭스 등) | 제조사 지정 온도 조건, 환기 설비 | 화학물질 전용 보관소 |
| 금속 자재 | 방청 처리 유지, 습도 관리 | 일반 자재 창고 |

## 7.3 FIFO 관리

- 모든 자재는 FIFO 원칙에 따라 불출한다.
- 입고일 또는 제조일을 기준으로 선입선출 관리한다.
- 보관 선반에 FIFO 방향 표시(화살표)를 부착한다.
- Shelf Life가 있는 자재는 유효기간 만료일을 우선 기준으로 출고 순서를 결정한다.
- 월 1회 FIFO 준수 상태를 점검하고, 위반 사항은 시정 조치한다.

## 7.4 Shelf Life (유효기간) 관리

| 관리 항목 | 기준 |
|----------|------|
| 유효기간 식별 | 자재 라벨에 제조일, 유효기간 만료일 표시 |
| 유효기간 경고 | 만료 **3개월 전** 경고 표시(노란색 라벨) 부착 |
| 만료 자재 격리 | 유효기간 만료 자재는 즉시 격리, "사용 금지" 표시 |
| 재검사/연장 | 제조사 기준에 따라 재검사 후 유효기간 연장 가능 (기술부서장 승인) |
| 만료 자재 처분 | 재검사 불합격 시 폐기 또는 반환 처리 |

# 8. 공정 중 보존

## 8.1 공정 간 취급

| 관리 항목 | 기준 |
|----------|------|
| 운반 용기 | 전용 운반 트레이, ESD 보호 용기 사용 |
| 적재 높이 | 제품 특성에 따른 최대 적재 단수 준수 |
| 이동 시 주의 | 낙하, 충격, 진동 방지 — 조심 취급 |
| 공정 대기 | 지정된 대기 장소에 보관, 바닥 직접 적재 금지 |
| 식별 유지 | 공정 간 이동 시 로트 식별, 검사 상태 표시 유지 |

## 8.2 ESD 관리 (공정 중)

| 관리 항목 | 기준 |
|----------|------|
| ESD 보호 구역 | 조립/검사 구역을 EPA(ESD Protected Area)로 지정 |
| 작업자 보호구 | 정전기 방지 손목 스트랩, 정전기 방지 의복/신발 착용 |
| 작업대 | 접지된 정전기 방지 매트 사용 |
| 운반 | ESD 보호 용기/트레이로 이동 |
| 접지 확인 | 접지 모니터 정기 점검, 손목 스트랩 일일 테스트 |
| 교육 | ESD 인식 교육 연 1회 이상, 신규 작업자 배치 전 필수 |

## 8.3 FOD 방지 (공정 중)

- 작업 전후 FOD 체크를 실시한다 (QP-819 연계).
- 작업대 및 제품 주변을 청결하게 유지한다.
- 공구 관리(Tool Control) — 사용 전후 공구 수량 확인.
- 음식물, 음료를 작업 구역에 반입하지 않는다.
- 개인 소지품(반지, 시계 등)을 작업 구역에서 착용하지 않는다.

## 8.4 공정 중 보관

| 상황 | 관리 기준 |
|------|----------|
| 교대 간 대기 | 진행 중인 작업물은 보호 덮개를 씌우고, 상태 표시 카드 부착 |
| 24시간 이상 대기 | MSD의 경우 방습 포장 또는 건조 캐비닛 보관 |
| 불합격 대기 | QP-818에 따라 격리 구역에 보관 |
| 재작업 대기 | 재작업 지시서와 함께 지정 구역에 보관 |

# 9. 완제품 보존 및 출하 포장

## 9.1 완제품 보관

| 관리 항목 | 기준 |
|----------|------|
| 보관 장소 | 완제품 전용 보관 구역, 일반 자재와 분리 |
| 보관 조건 | 온도 15~30°C, 습도 40~70%RH (특수 조건은 별도 지정) |
| 식별 | 최종검사 합격 라벨, 출하 대기 상태 표시 |
| 적재 | 제품 사양에 따른 최대 적재 단수 준수, 바닥 직접 적재 금지 |
| 선입선출 | FIFO 원칙 적용 |

## 9.2 출하 포장 절차

```
최종검사 합격
      ↓
포장 사양서 확인 (고객 요구사항 포함)
      ↓
내부 포장 (제품 보호)
- ESD 보호 포장 (해당 시)
- 완충재 적용
- 건조제/HIC 삽입 (해당 시)
      ↓
외부 포장 (운송 보호)
- 골판지 박스 또는 전용 용기
- 충격/진동 완충
- 방수 처리 (해당 시)
      ↓
포장 라벨 부착
- 품번, 수량, 로트번호
- 취급 주의 표시 (Fragile, ESD, This Side Up 등)
- 고객 요구 라벨
      ↓
포장 검사 (포장 상태, 라벨 정확성 확인)
      ↓
출하 대기 구역 이동
```

## 9.3 포장 사양

| 포장 유형 | 적용 대상 | 포장 구성 |
|----------|----------|----------|
| 표준 포장 | 일반 전자 조립품 | ESD 봉투 → 완충재 → 골판지 박스 |
| 방습 포장 | MSD, PCB 등 습기 민감 제품 | 방습 봉투(MBB) + 건조제 + HIC → 완충재 → 골판지 박스 |
| 강화 포장 | 중량물, 정밀 기기 | 전용 폼 삽입물 → 이중 박스 → 목재 크레이트 (해당 시) |
| 고객 지정 포장 | 고객 포장 사양이 있는 경우 | 고객 포장 사양서에 따라 수행 |

# 10. 특수 보존 요건

## 10.1 ESD 민감 부품 보존

| 단계 | ESD 보존 요건 |
|------|-------------|
| 입고 | 원래 ESD 보호 포장 유지, 불필요한 개봉 금지 |
| 보관 | ESD 보호 봉투 또는 전도성 용기에 보관 |
| 불출 | ESD 보호 용기로 이동, EPA 내에서만 개봉 |
| 공정 | EPA 내 작업, 접지 확인 후 취급 |
| 포장 | ESD 보호 봉투 → 전도성 폼 → 외부 포장 |

## 10.2 습기 민감 소자 (MSD) 보존

| MSL 등급 | Floor Life (30°C/60%RH) | 관리 기준 |
|----------|------------------------|----------|
| MSL 1 | 무제한 | 특별 관리 불필요 |
| MSL 2 | 1년 | 방습 포장 개봉 후 1년 내 사용 |
| MSL 2a | 4주 | 개봉 후 4주 내 사용 |
| MSL 3 | 168시간 (7일) | 개봉 후 7일 내 사용, 경과 시 베이킹 |
| MSL 4 | 72시간 | 개봉 후 72시간 내 사용 |
| MSL 5 | 48시간 | 개봉 후 48시간 내 사용 |
| MSL 5a | 24시간 | 개봉 후 24시간 내 사용 |
| MSL 6 | 리플로우 전 필수 베이킹 | 사용 직전 베이킹 필수 |

**MSD 관리 규칙**:
- 방습 포장(MBB) 개봉 시 개봉 일시를 라벨에 기록한다.
- Floor Life 경과 시 IPC/JEDEC J-STD-033에 따라 베이킹(Baking)을 실시한다.
- 베이킹 조건: 부품 유형에 따라 125°C/24hr 또는 40°C/192hr 등 적용.
- 건조 캐비닛(<10%RH)에 보관 시 Floor Life 시계가 정지된다.
- HIC(습도 표시 카드)가 변색된 경우 해당 자재를 사용 전 검증한다.

# 11. 항공우주 특수 요구사항

## 11.1 수명 제한 부품 (Life-Limited Parts) 관리

| 관리 항목 | 요구사항 |
|----------|----------|
| 식별 | 수명 제한 부품에 수명 정보(제조일, 최대 수명, 잔여 수명)를 표시 |
| 추적성 | 수명 제한 부품은 개별 시리얼번호로 추적 관리 (QP-812 연계) |
| 보관 | 수명 소진 방지를 위한 보관 조건 최적화 |
| 사용 기한 | 잔여 수명이 부족한 부품은 생산 투입 전 기술부서 확인 |
| 폐기 | 수명 만료 부품은 즉시 격리 및 폐기 (재사용 불가 표시) |

## 11.2 특수 포장 요건

항공우주 제품의 포장 시 다음 추가 요건을 적용한다:

| 항목 | 요구사항 |
|------|----------|
| 고객 포장 사양 | 고객이 지정한 포장 사양서(Packaging Specification)가 있는 경우 해당 사양 우선 적용 |
| 군수/방산 포장 | MIL-STD-2073-1 등 해당 군사 규격 적용 (해당 시) |
| 클린룸 포장 | 클린룸 환경에서 사용되는 부품은 클린룸 내 또는 동등 환경에서 포장 |
| 이중 포장 | 항공우주 정밀 부품은 내부 보호 포장 + 외부 운송 포장의 이중 구조 적용 |
| 포장 검증 | 신규 포장 사양 적용 시 진동/낙하 시험 등 포장 검증 수행 (해당 시) |

## 11.3 FOD 방지 연계 (QP-819)

제품 보존의 모든 단계에서 FOD 방지 활동을 통합 적용한다:

- **입고 시**: 포장 개봉 시 이물질 유입 방지, 포장재 잔해물 즉시 제거
- **보관 시**: 보관 구역 청결 유지, 덮개/뚜껑 사용으로 이물질 유입 차단
- **공정 중**: 작업 전후 FOD 점검, 공구 관리(Tool Control)
- **포장 시**: 포장 전 제품 및 포장재 이물질 점검, 클린 포장 환경 유지
- **운송 시**: 운송 용기 내부 청결 확인, 밀봉 처리

## 11.4 보관 환경 모니터링

| 모니터링 항목 | 기준 | 빈도 | 기록 |
|-------------|------|------|------|
| 일반 창고 온습도 | 15~30°C, 40~70%RH | 1일 2회 (오전/오후) | 온습도 기록지 |
| 건조 캐비닛 습도 | <10%RH | 1일 1회 | 캐비닛 습도 기록 |
| 냉장 보관고 온도 | 0~10°C | 1일 2회 | 냉장고 온도 기록 |
| ESD 보호 구역 | 접지 저항, 표면 저항 | 주 1회 | ESD 점검 기록 |
| 청결도 | 육안 점검 | 1일 1회 | FOD 점검 체크리스트 |

**일탈 시 조치**: 기준 초과 시 즉시 시설관리팀에 통보하고, 해당 구역 보관 자재의 영향도를 평가한다. 영향이 우려되는 자재는 격리 후 기술부서의 사용 가부 판정을 받는다.
