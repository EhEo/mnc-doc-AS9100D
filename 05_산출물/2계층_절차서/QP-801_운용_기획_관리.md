# 1. 목적

본 절차서는 M&C Electronics VINA의 제품 실현(운용)에 필요한 프로세스를 기획, 실행 및 관리하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.1 요구사항에 따른 운용 기획 프로세스 수립
- 제품/서비스의 적합성을 보장하기 위한 체계적인 생산 계획 수립
- 프로세스 자원(인원, 설비, 자재, 환경)의 효율적 결정 및 배분
- ERP 시스템을 활용한 생산 오더 관리 및 추적
- 항공우주 산업 운용 기획 요구사항 충족

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 운용 기획 (Operations Planning) | 제품/서비스 요구사항 충족을 위한 프로세스의 기획 및 수립 활동 |
| 제품 실현 계획 (Product Realization Plan) | 특정 제품/프로젝트에 대한 품질, 공정, 자원, 일정의 종합 계획 |
| 관리 계획서 (Control Plan) | 공정별 관리 항목, 관리 방법, 검사 빈도, 합부 기준을 규정한 문서 |
| 생산 오더 (Production Order) | ERP 시스템에서 발행하는 생산 지시 단위. 자재, 공정, 수량, 납기 포함 |
| 공정 흐름도 (Process Flow Diagram) | 원자재 투입부터 완제품 출하까지의 공정 순서를 도식화한 문서 |
| ERP (Enterprise Resource Planning) | 생산, 자재, 품질, 재고를 통합 관리하는 전사적 자원 관리 시스템 |
| BOM (Bill of Materials) | 제품 생산에 필요한 자재/부품 목록 (부품번호, 수량, 사양 포함) |
| Routing | 제품 생산을 위한 공정 순서, 작업장, 표준 시간의 정의 |
| 외주 프로세스 (Outsourced Process) | 외부 조직이 수행하지만, 당사 QMS 관리 범위에 포함되는 프로세스 |
| WBS (Work Breakdown Structure) | 프로젝트 범위를 관리 가능한 작업 단위로 분해한 구조 |

# 6. 운용 기획 프로세스

## 6.1 제품 실현 계획 (Product Realization Planning)

AS9100D 8.1에 따라 제품/서비스 제공에 필요한 프로세스를 기획할 때 다음을 결정한다:

| 기획 항목 | 결정 사항 | 관련 문서 |
|----------|----------|----------|
| **a) 품질목표 및 요구사항** | 제품/서비스에 대한 품질 기준 및 고객 요구사항 | QP-805, QP-806 |
| **b) 프로세스 수립** | 공정 흐름, 작업 순서, 공정 조건 | Routing, 공정 흐름도 |
| **c) 자원 결정** | 인원, 설비, 금형/치공구, 자재, 작업 환경 | BOM, 설비 목록 |
| **d) 문서화된 정보** | 작업표준서, 도면, 사양서, 관리 계획서 | WI-xxx, Control Plan |
| **e) 검증/확인 활동** | 검사 포인트, 시험 항목, 합부 기준 | 관리 계획서, QP-810 |
| **f) 제품 합격 기준** | 인수 기준, 출시 판정 기준 | 고객 사양, 도면 |
| **g) 기록 요구사항** | 생산 이력, 검사 기록, 추적성 기록 | QP-706, QP-812 |

## 6.3 자원 결정

### 6.3.1 인적 자원

| 항목 | 내용 |
|------|------|
| 작업자 역량 | 해당 공정 수행에 필요한 자격/역량 보유 여부 확인 (QP-703 연계) |
| 인원 배치 | 생산 물량, 공정 복잡도, 납기에 따른 적정 인원 배치 |
| 특수공정 자격 | 특수공정(용접, NDT 등) 수행 인원의 자격 인증 유효 여부 |

| 항목 | 내용 |
|------|------|
| 설비 가용성 | 생산 계획 대비 설비 가용 용량 확인 (ERP 설비 부하 분석) |

| 항목 | 내용 |

# 7. 관리 계획서 (Control Plan)

## 7.1 관리 계획서 작성

관리 계획서는 제품 실현의 각 공정에서 관리해야 할 항목, 방법, 빈도를 규정한다.

| 포함 항목 | 내용 |
|----------|------|
| 공정 번호/명 | 공정 흐름도에 따른 공정 식별 |
| 관리 특성 | 제품 특성 (치수, 외관, 기능) 및 공정 특성 (온도, 압력, 시간) |
| 특성 등급 | 일반(G), 핵심(K), 안전핵심(SC) 구분 |
| 규격/공차 | 도면/사양서에 따른 합부 기준 |
| 관리 방법 | 측정 장비, 측정 방법, 샘플 크기 |
| 관리 빈도 | 전수/샘플링, 주기 (매 로트, 매 시간, 매일 등) |
| 반응 계획 | 이상 발생 시 대응 절차 (정지, 격리, 보고 등) |

## 7.2 관리 계획서 유형

| 유형 | 적용 시점 | 설명 |
|------|----------|------|
| 시작 (Prototype) | 시작품 제작 | 초기 공정 관리, 전수 검사 위주 |
| 양산 준비 (Pre-launch) | FAI/양산 승인 전 | 강화된 관리 빈도, 공정 능력 확인 |
| 양산 (Production) | 양산 이후 | 정상 관리 빈도, SPC 적용 |

## 7.3 관리 계획서 변경

관리 계획서 변경이 필요한 경우:
- 4M 변경 (인원, 설비, 자재, 방법)
- 고객 요구사항 변경
- 부적합 발생에 따른 관리 강화
- 공정 능력 평가 결과에 따른 조정

변경 시 QP-803 형상관리 절차에 따라 검토/승인 후 적용한다.

# 8. 작업 지시/생산 오더 관리

## 8.1 ERP 생산 오더 관리

당사는 ERP 시스템을 통해 생산 오더를 관리한다.

| 단계 | 활동 | 담당 | ERP 기능 |
|------|------|------|----------|
| 1 | 생산 오더 생성 | 생산계획 담당 | 수주 → MRP 전개 → 생산 오더 자동/수동 생성 |
| 2 | BOM/Routing 확인 | 기술팀 | BOM 전개, 공정 순서 확인 |
| 3 | 자재 가용성 확인 | 생산계획 담당 | 재고 조회, 부족 자재 구매 의뢰 연동 |
| 4 | 오더 확정 (Release) | 생산계획 담당 | 오더 상태: 계획 → 확정 |
| 5 | 작업 지시서 출력 | 생산계획 담당 | 작업 지시서 + 자재 불출 지시서 출력 |
| 6 | 자재 불출 | 자재팀 | ERP 불출 처리 (로트 추적 연동, QP-812) |
| 7 | 생산 실행 | 생산팀 | 공정별 작업 수행, ERP 실적 입력 |
| 8 | 공정 검사 | 품질팀 | 검사 결과 ERP 입력 (합격/부적합) |
| 9 | 오더 완료 | 생산계획 담당 | 전 공정 완료 확인, ERP 오더 마감 |

## 8.2 작업 지시서 포함 사항

작업 지시서(생산 오더 기반)에는 다음 정보를 포함한다:

| 항목 | 내용 |
|------|------|
| 생산 오더 번호 | ERP 자동 채번 |
| 품번/품명 | 제품 식별 정보 |
| 생산 수량 | 주문 수량, 양품 목표 수량 |
| 납기 | 고객 요구 납기, 사내 목표 납기 |
| BOM | 투입 자재/부품 목록 및 수량 |
| Routing | 공정 순서, 작업장, 표준 시간 |
| 도면/사양 참조 | 적용 도면 번호 및 개정 번호 |
| 특기 사항 | 고객 특별 요구, 관리 강화 항목, 안전 핵심 특성 표기 |

## 8.3 생산 일정 관리

| 활동 | 주기 | 담당 |
|------|------|------|

# 9. 외주 프로세스 관리

## 9.1 외주 프로세스 식별

외부 조직이 수행하지만 당사 QMS 관리 범위에 포함되는 프로세스:

| 외주 유형 | 예시 | 관리 수준 |
|----------|------|----------|
| 특수공정 외주 | 열처리, 도금, NDT | 높음 — 공정 승인, NADCAP 요구 |
| 가공 외주 | CNC 가공, 판금 | 중간 — 초도/정기 검사 |
| 시험 외주 | 환경 시험, 신뢰성 시험 | 높음 — 시험소 인정 확인 |
| 표면처리 외주 | 도장, 양극 산화 | 높음 — 공정 사양 준수 확인 |

## 9.2 외주 프로세스 기획

| 기획 항목 | 내용 |
|----------|------|
| 외주 범위 | 외주 대상 공정/작업의 명확한 범위 정의 |
| 기술 요구사항 | 도면, 사양서, 공정 조건의 외주 업체 전달 |
| 품질 요구사항 | 검사 기준, 합부 판정 기준, 기록 요구사항 |
| 납기 관리 | 외주 리드타임을 반영한 생산 일정 수립 (ERP 연동) |
| 승인 공급업체 | ASL(Approved Supplier List) 등재 업체에만 발주 (QP-808 연계) |

## 9.3 외주 프로세스 관리 방법

| 관리 방법 | 내용 |
|----------|------|
| 수입검사 | 외주 완료품의 수입검사 실시 |
| 공정 감사 | 핵심 외주 업체에 대한 정기/수시 공정 감사 |
| 성과 평가 | 외주 업체 품질/납기/서비스 정기 평가 |
| 기록 수집 | 외주 공정 기록(열처리 차트, 도금 기록 등) 수집/보관 |

# 10. 항공우주 특수 요구사항

## 10.1 운영 리스크 관리 연계 (AS9100D 8.1.1 → QP-802)

운용 기획 시 다음의 운영 리스크를 평가하고 관리 조치를 수립한다:

| 리스크 영역 | 평가 항목 | 관리 절차 |
|------------|----------|----------|
| 신규 제품/공정 | 설계/공정 변경에 따른 리스크 | QP-802 (DFMEA/PFMEA) |
| 공급망 리스크 | 자재 조달, 공급자 품질, 납기 | QP-802, QP-808 |
| 생산 리스크 | 공정 능력, 설비 고장, 인적 오류 | QP-802 (PFMEA) |
| 납기 리스크 | 일정 지연, 자원 부족 | ERP 모니터링 |

> **참고**: AS9100D 8.1.1 "운영 리스크 관리"의 상세 절차는 QP-802에서 별도 규정한다. 본 절차서에서는 운용 기획 단계에서의 리스크 평가 연계를 다룬다.

## 10.2 형상관리 연계 (AS9100D 8.1.3 → QP-803)

| 적용 사항 | 내용 |
|----------|------|
| 형상 식별 | 제품/문서의 형상 항목 식별 및 기준선(Baseline) 수립 |
| 형상 변경 | CCB(형상관리위원회) 승인을 통한 변경 관리 |
| 생산 기획 반영 | 승인된 형상(도면/BOM/사양)에 기반한 생산 계획 수립 |

## 10.3 제품 안전 연계 (AS9100D 8.1.4 → QP-804)

| 적용 사항 | 내용 |
|----------|------|
| 안전 핵심 특성 | 안전 핵심 항목(Safety Critical Items)의 생산 기획 시 특별 관리 반영 |
| 관리 계획서 반영 | 안전 핵심 특성에 대한 강화된 검사/관리 빈도 적용 |
| 추적성 | 안전 핵심 부품의 완전한 추적성 보장 (QP-812 연계) |

## 10.4 핵심 특성 관리 (AS9100D 8.1.5)

제품/공정의 핵심 특성(Key Characteristics)에 대해:

| 관리 항목 | 내용 |
|----------|------|
| 식별 | 도면/사양서에서 핵심 특성(KC) 및 안전 핵심 특성(SC) 식별 |
| 관리 계획 | Control Plan에 KC/SC 표기 및 관리 방법 명시 |
| SPC 적용 | KC/SC에 대한 SPC(통계적 공정관리) 적용 (Cpk ≥ 1.33, SC: ≥ 1.67) |
| 작업 지시 | 작업 지시서에 KC/SC 항목 명시 및 작업자 주의 환기 |

## 10.5 변경 관리

운용 기획 관련 변경(공정, 자재, 설비, 일정 등) 발생 시:
- 변경 전 리스크 평가 실시 (QP-802 연계)
- 고객 승인 필요 여부 확인 (고객 계약 조건 검토)
- 형상관리 절차에 따른 승인 (QP-803 연계)
- 관리 계획서 갱신 (해당 시)
- 변경 후 FAI 재실시 여부 결정 (QP-811 연계)
