# 1. 목적

본 절차서는 M&C Electronics VINA에서 제품의 최종검사(OQC: Outgoing Quality Control) 및 출시 관리에 관한 절차를 규정한다.

> 참고: 기존 OQC 검사 절차(QM.02.04.04, Rev.05)를 기반으로 AS9100D 8.6 항공우주 요구사항을 확장한다.

본 절차서의 목적:
- AS9100D 8.6 요구사항에 따른 제품/서비스의 출시 전 최종 적합성 확인
- 계획된 검사 기준을 만족하는 제품만 고객에게 인도되도록 보장
- 합격/불합격 판정 및 출시 승인의 체계적 의사결정
- 고객별 요구사항 및 항공우주 고유 요건의 체계적 이행
- 검사 기록의 완전한 추적성(Traceability) 확보

# 2. 적용 범위

# 5. 책임과 권한

| 역할 | 책임 |
|------|------|
| **OQC 검사원** | 검사 정보 접수, 검사 준비, 샘플링/전수 검사 수행, 검사 성적서 발행, PASS/FAIL 스탬프 날인 |
| **OQC 부책임자(AM)** | 검사 성적서 검토(Check), 검사 결과 확인, 재검사 판정 |
| **QC 관리자(QC MGR)** | 검사 성적서 최종 승인(Approve), 출시 최종 결정, 특채 승인 참여 |
| **생산팀** | 검사 의뢰, 검사 정보(LOT No., 수량, 사양 등) 제공, 부적합 시 재작업 수행 |
| **영업팀** | 고객 요구사항 전달, 출하 일정 조정 |
| **품질책임자(QMR)** | 위임검사원 지정/해제, 검사 체계 총괄 |

### 승인 체계 (기존 OQC 프로세스 기반)

```
OQC 검사원 (Issues/작성) → OQC AM (Checks/검토) → QC MGR (Approves/승인)
```

# 6. 최종검사 프로세스

> 본 프로세스는 기존 OQC 검사 절차(QM.02.04.04)의 흐름을 기반으로 AS9100D 8.6 요구사항을 확장한 것이다.

```
┌─────────────────┐
│ ① 검사 정보 접수  │  ← 생산팀으로부터 LOT 정보, 수량, 사양 수령
└────────┬────────┘
         ▼
┌─────────────────┐
│ ② 검사 준비      │  ← 검사 기준서, 도면, 게이지, 측정장비 준비
└────────┬────────┘
         ▼
┌─────────────────┐
│ ③ 검사 수행      │  ← HQS 기준 샘플링 또는 전수검사 수행
└────────┬────────┘
         ▼
    ┌────┴────┐
    │ 판정    │
    └────┬────┘
    ┌────┴────┐
   OK        NG
    │         │
    ▼         ▼
┌────────┐  ┌──────────────────┐
│④-1 합격 │  │④-2 불합격 통보     │ ← 관련 부서(생산, 기술)에 통보
│ 처리   │  └────────┬─────────┘
└───┬────┘           ▼
    │        ┌──────────────────┐
    │        │⑤ 100% 전수 선별   │ ← 전수 선별(Sorting) 수행

    │        ┌──────────────────┐

    │        └────────┬─────────┘
    │                 ▼
    ▼                 ▼
┌─────────────────────────────┐
```

## 6.2 단계별 세부 절차

### 6.2.1 검사 정보 접수 (① Receive Info)

- 생산팀은 최종검사 의뢰 시 다음 정보를 OQC에 전달한다:
  - 제품명, 제품번호(Part No.)
  - LOT 번호 및 생산 수량
  - 적용 도면 및 사양서 번호/개정번호
  - 공정검사 완료 확인
  - 고객 주문번호(P/O No.)
- OQC 검사원은 전달된 정보의 완전성을 확인한 후 검사를 개시한다.

### 6.2.2 검사 준비 (② Prepare Inspection)

- 해당 제품의 최신 도면, 검사 기준서, 고객 사양서를 확인한다.
- 필요한 측정장비, 게이지, 지그의 교정 상태를 확인한다 (QP-702 참조).
- 검사 환경(조명, 온습도 등)이 규정 조건을 만족하는지 확인한다.
- 항공우주 제품의 경우 위임검사원 자격 여부를 확인한다.

### 6.2.3 검사 수행 (③ Inspect)

- HQS 샘플링 테이블에 따른 샘플링 검사 또는 전수검사를 수행한다 (7장 참조).
- 치수, 외관, 기능, 성능 등 검사 항목별로 합부를 판정한다.
- 검사 데이터를 검사 성적서에 기록한다.

### 6.2.4 합격 처리 (④-1 OK → Report + Archive)

- 합격 시 제품에 **PASS 스탬프**를 날인한다.
- 검사 성적서를 발행한다: OQC 검사원(작성) → OQC AM(검토) → QC MGR(승인).
- 합격 기록을 보관(Archive)한다.
- 출하 가능 상태로 전환한다.

### 6.2.5 불합격 처리 (④-2 NG → Inform Related Dept)

- 불합격 시 제품에 **FAIL 스탬프**를 날인한다.
- 관련 부서(생산팀, 기술팀 등)에 불합격 내용을 즉시 통보한다.
- 부적합 내용을 기록하고 QP-818 (부적합 출력 관리)에 따라 처리를 개시한다.

### 6.2.6 100% 전수 선별 (⑤ 100% Sorting)

- 불합격 LOT에 대해 전수 선별(Sorting)을 수행한다.
- 양품과 불량품을 분리하고, 불량품은 격리 보관한다.
- 선별 수량 및 불량 유형을 기록한다.

### 6.2.7 재검사 (⑥ Re-inspect)

- 전수 선별 완료 후, 양품에 대해 재검사를 수행한다.
- 재검사 합격 시 PASS 스탬프 날인 및 검사 성적서를 재발행한다.
- 재검사 불합격 시 QP-818에 따라 처분(재작업, 폐기 등)을 결정한다.

### 6.2.8 검사 성적서 발행 및 기록 (⑦ Report)

- 최종 검사 성적서를 발행하고 승인 체계에 따라 결재한다.
- 검사 기록 원본을 보관하고, 고객 요청 시 사본을 제공한다.

# 7. 검사 기준 및 방법

| 조건 | 설명 |

# 8. 합부 판정 및 출시 승인

## 8.1 합격 판정 기준

제품이 다음 조건을 **모두** 만족하는 경우 합격(PASS)으로 판정한다:

1. 모든 검사 항목이 도면/사양서 기준 이내
2. 외관 결함이 합격 한도 이내
3. 기능/성능 시험 결과 합격
4. 식별/추적성 표시가 정확
5. 선행 공정검사 기록이 완료 확인됨
6. (해당 시) FAI 보고서가 승인 완료됨

## 8.2 출시 승인 절차

```
검사 합격 판정
    ↓
OQC 검사원: 검사 성적서 작성 + PASS 스탬프 날인
    ↓
OQC AM: 검사 성적서 검토(Check) + 서명
    ↓
QC MGR: 출시 최종 승인(Approve) + 서명
    ↓
출하 가능 (Release)
```

## 8.3 출시 승인 전 확인 사항

| 확인 항목 | 확인자 | 비고 |
|----------|--------|------|
| 검사 성적서 완전성 | OQC AM | 누락 항목 없음 확인 |
| 공정검사 기록 연계 | OQC AM | 전 공정 합격 확인 |
| 고객 특별 요구사항 | QC MGR | 고객별 추가 요건 확인 |
| CoC(적합성 인증서) 발행 | QC MGR | 항공우주 고객 요구 시 |
| 포장/라벨 적합성 | OQC 검사원 | 출하 전 최종 확인 |

## 8.4 조건부 출시 (특채)

- 검사 기준에 부적합하나 고객 합의하에 출하가 필요한 경우, QP-818의 특채(Concession) 절차에 따른다.
- 항공우주 제품의 특채는 반드시 고객의 서면 승인을 득한 후에만 출시한다.
- 특채 이력은 별도 관리하며, 재발 방지 대책을 수립한다.

# 9. 부적합 처리

## 9.1 최종검사 부적합 발생 시

최종검사에서 부적합(NG)이 발생한 경우, 다음 절차에 따라 처리한다:

1. **즉시 격리**: 부적합 제품에 FAIL 스탬프 날인 후 격리 구역에 보관
2. **관련 부서 통보**: 생산팀, 기술팀에 부적합 내용 즉시 통보
3. **부적합 보고서 발행**: QP-818 (부적합 출력 관리)에 따라 부적합 보고서(NCR) 작성
4. **100% 전수 선별**: 해당 LOT 전량에 대해 선별 수행
5. **재검사**: 선별 후 양품에 대해 재검사 수행
6. **처분 결정**: 재작업, 수리, 특채, 폐기 중 결정 (QP-818 절차)
7. **재발 방지**: 근본 원인 분석 후 시정조치 (QP-1001 연계)

## 9.2 QP-818 연계

| 항목 | QP-817 (본 절차서) | QP-818 (부적합 출력 관리) |
|------|-------------------|--------------------------|
| 부적합 발견 | 최종검사 시 부적합 식별 | 부적합 보고서(NCR) 발행 |
| 격리 | FAIL 스탬프 날인, 물리적 격리 | 격리 구역 관리, 처분 대기 |
| 처분 결정 | QP-818로 이관 | MRB 심의, 처분 결정 |
| 재작업 후 | 재검사 수행 (본 절차서) | 재작업 지시서 발행 |
| 기록 | 검사 성적서, 재검사 기록 | NCR, MRB 의사록, 처분 기록 |

# 10. 항공우주 특수 요구사항

## 10.1 위임검사원 (Delegated Inspection)

- 항공우주 제품의 최종검사는 **위임검사원 자격을 보유한 검사원**만 수행할 수 있다.
- 위임검사원의 자격 요건:
  - OQC 검사 경력 최소 1년 이상
  - AS9100D 인식 교육 이수
  - 해당 제품군 검사 역량 평가 합격
  - 품질책임자(QMR)의 서면 승인
- 위임검사원 목록은 품질팀에서 관리하며, 연 1회 이상 자격을 재평가한다.
- 위임검사원 지정/해제 권한은 QMR에게 있다.

## 10.2 고객별 기록 보존기간

기존 OQC 절차(QM.02.04.04)에서 규정한 고객별 기록 보존기간을 항공우주 요구사항과 병행하여 관리한다:

| 고객 | 기록 보존기간 | 비고 |
|------|-------------|------|
| LG | 30년 | 고객 계약 요구사항 |
| ZF | 15년 | 고객 계약 요구사항 |
| Mando | 15년 | 고객 계약 요구사항 |
| Hosiden | 11년 | 고객 계약 요구사항 |
| Samsung | 3년 | 고객 계약 요구사항 |
| 기타 고객 | 1년 | 사내 기본 기준 |
| **항공우주 고객** | **계약 요구 또는 최소 10년** | **AS9100D 요구사항 (규제/계약 중 긴 기간 적용)** |

> **주의**: 항공우주 제품의 경우, 고객 계약 보존기간과 규제 요구사항 중 더 긴 기간을 적용한다. 항공우주 규제기관 요구 시 보존기간이 제품 수명 전체(Lifetime)로 연장될 수 있다.

## 10.3 FAI 연계

- 다음의 경우 최종검사 전에 FAI(First Article Inspection)를 완료해야 한다:
  - 신규 제품 최초 양산 시
  - 설계 변경 후 최초 생산 시
  - 공정/설비/치공구 변경 후 최초 생산 시
  - 생산 중단 후 재개 시 (고객 요구 또는 사내 기준에 따름)
- FAI는 AS9102에 따라 수행하며, FAI 보고서 승인 후에만 양산 출하를 개시한다.
- FAI 관련 세부 사항은 QP-811 (FAI 관리) 절차를 따른다.

## 10.4 적합성 인증서 (CoC/CofC)

- 항공우주 고객이 요구하는 경우, 출하 시 적합성 인증서(Certificate of Conformance)를 발행한다.
- CoC에는 다음 사항을 포함한다:
  - 제품 식별 정보 (Part No., LOT No., S/N)
  - 적용 규격 및 도면 번호/개정번호
  - 검사 결과 요약
  - 자재 성적서(Mill Certificate) 참조 (해당 시)
  - 승인권자 서명 및 일자

## 10.5 고객/규제기관 검사 입회

- 고객 또는 규제기관(EASA, FAA 등)이 최종검사 입회를 요구하는 경우, 검사 일정을 사전에 통보하고 입회 기회를 보장한다.
- 입회 검사 시 검사 기록에 입회자 정보를 기록한다.
