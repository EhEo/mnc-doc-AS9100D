# 1. 목적

본 절차서는 M&C Electronics VINA에서 제품의 적합성을 판정하기 위해 사용하는 모든 모니터링 및 측정 장비의 관리, 교정, 유지보수에 관한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 7.1.5 요구사항에 따른 측정장비의 체계적 관리
- 측정 결과의 유효성 및 신뢰성 보장
- 측정 불확도(Measurement Uncertainty)의 적절한 관리
- 교정 상태의 추적성(Traceability) 확보
- CAR-012 (측정장비 관리 체계 미흡) 시정조치 반영

---

# 2. 적용 범위

본 절차서는 M&C Electronics VINA의 QMS 내에서 제품/공정의 적합성 판정에 사용되는 모든 모니터링 및 측정 장비에 적용한다.

| 구분 | 적용 대상 |
|------|----------|
| 측정장비 | 캘리퍼스, 마이크로미터, 하이트 게이지, 핀 게이지, 좌표측정기(CMM), 경도계 등 |
| 시험장비 | 인장시험기, 충격시험기, 피로시험기 등 |
| 검사장비 | 비파괴검사(NDT) 장비, 광학검사기, 프로파일 프로젝터 등 |
| 환경측정 | 온·습도계, 청정도 측정기 등 (품질에 영향을 미치는 경우) |
| 공정감시 | 토크렌치, 압력게이지, 온도센서 등 (공정 파라미터 관리용) |
| 소프트웨어 | 측정/검사에 사용되는 소프트웨어 (의도된 용도 적합성 확인) |

**제외**: 일반 사무용 자(눈금 정밀도가 요구되지 않는 경우), 참조용 측정기(품질 판정에 사용하지 않는 경우)

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 교정 (Calibration) | 규정된 조건 하에서 측정장비가 나타내는 값과 측정 표준에 의한 참값 사이의 관계를 확정하는 일련의 작업 |
| 검증 (Verification) | 규정된 요구사항이 충족되었음을 객관적 증거를 제시하여 확인하는 활동 |
| 측정 불확도 (Measurement Uncertainty) | 측정 결과에 부여된 파라미터로서 측정값의 분산을 합리적으로 나타내는 값 |
| 추적성 (Traceability) | 국가 또는 국제 측정 표준에 끊어지지 않는 교정 사슬(unbroken chain)로 연결되는 특성 |
| MSA (Measurement System Analysis) | 측정시스템 분석 — 반복성(Repeatability), 재현성(Reproducibility) 등 측정시스템의 변동 분석 |
| GR&R (Gage Repeatability & Reproducibility) | 측정시스템의 반복성과 재현성을 정량적으로 분석하는 방법 |
| 교정 주기 | 장비의 사용 빈도, 환경, 정밀도 등을 고려하여 설정된 교정 간격 |
| 교정 상태 라벨 | 장비에 부착하여 교정 상태(합격/부적합/사용제한)를 시각적으로 표시하는 라벨 |

# 6. 측정장비 관리 체계

## 6.1 장비 등록 및 식별

### 6.1.1 장비 등록

모든 교정 대상 측정장비는 측정장비 대장(Calibration Master List)에 등록한다.

| 등록 정보 | 내용 |
|----------|------|
| 장비 관리번호 | 고유 식별번호 부여 (예: CAL-2026-001) |
| 장비명/모델명 | 제조사, 모델, 시리얼 번호 |
| 측정 범위 | 장비의 측정 가능 범위 |
| 정밀도/분해능 | 장비의 최소 읽음값 |
| 보관 위치 | 사용 부서 및 보관 장소 |
| 교정 주기 | 설정된 교정 간격 |
| 교정 기관 | 사내 교정 또는 외부 교정기관명 |
| 최근 교정일 / 차기 교정일 | 교정 이력 추적 |

### 6.1.2 장비 식별 (라벨링)

| 교정 상태 | 라벨 색상 | 표시 내용 |
|----------|----------|----------|
| 교정 합격 | 녹색 | 관리번호, 교정일, 차기 교정일, 교정자 |
| 사용 제한 | 황색 | 관리번호, 제한 사유, 사용 가능 범위 |
| 교정 부적합/수리 중 | 적색 | 관리번호, "사용 금지", 부적합 일자 |

## 6.2 교정 주기 설정

### 6.2.1 초기 교정 주기

> **참고**: 아래 교정 주기는 기존 ISO 9001/IATF 16949 절차서(QM.02.03.12, Rev.05)의 실제 운영 기준을 기반으로 하며, AS9100D 항공우주 요구사항을 반영하여 보완한 것이다.

| 분류 | 장비 유형 | 교정 주기 | 교정 방법 | 비고 |
|------|---------|---------|---------|------|
| **측정장비 (Thiết bị đo lường)** | CMM, 2D 측정기, 성분분석기(XRF), 캘리퍼스, 하이트 게이지, 다이얼 게이지, 색차계, 분진농도계, 온도계, 도막두께계, 두께계, 핀 게이지, 조도계, 프로파일 측정기 등 | **12개월** | **외부 교정** (ISO/IEC 17025 공인기관 한정) | 고객 요구 시 단축 |
| **검사 Jig (마모 어려움)** | 평탄도 측정 Jig, 교정 블록 등 | **3개월** | **내부 교정** (마모도 기준 자체 검사) | |
| **검사 Jig (마모 용이)** | 두께 슬라이드, 폭, 홈 깊이/폭 측정 Jig, 탭 게이지 등 | **1개월** | **내부 교정** (마모도 기준 자체 검사) | |
| 시험장비 | 인장시험기, 충격시험기 등 | 12개월 | 외부 교정 | |
| 공정감시용 | 토크렌치, 압력게이지 등 | 6~12개월 | 외부/내부 | 사용 빈도 고려 |

**교정 만료 사전 알림**: 교정 유효기간 잔여 **30일 미만** 시 교정 대상 목록을 작성하여 외부 교정기관에 통보한다. (기존 QM.02.03.12 §5.2.3 준용)

### 6.2.2 교정 주기 조정

다음 조건에 해당하는 경우 교정 주기를 조정한다:

| 조건 | 조정 방향 |
|------|----------|
| 교정 결과 연속 3회 합격 + 편차 미미 | 주기 연장 가능 (최대 24개월) |

| 장비 수리/오버홀 후 | 즉시 재교정 |
| 사용 빈도 증가 | 주기 단축 검토 |
| 고객 요구사항 | 고객 요구 주기 우선 적용 |
| 항공우주 규제 요구 | 규제 요구 주기 우선 적용 |

교정 주기 변경 시 품질팀장 승인 필요.

# 7. 교정 관리

## 7.1 교정 계획

품질팀장은 매년 1월에 연간 교정 계획을 수립한다.

| 계획 포함 사항 | 내용 |
|--------------|------|
| 교정 대상 장비 목록 | 전 장비 리스트 |
| 교정 예정월 | 차기 교정일 기준 |
| 교정 방법 | 사내 교정 / 외부 위탁 |
| 교정 기관 | 공인 교정기관명 |
| 예산 | 외부 교정 비용 |

## 7.2 교정 실행

### 7.2.1 외부 교정

| 단계 | 활동 | 책임자 |
|------|------|--------|
| 1 | 교정 대상 장비 확인 및 반출 준비 | 교정담당자 |
| 2 | 공인 교정기관에 의뢰 (ISO/IEC 17025 인정기관) | 품질팀장 |
| 3 | 교정 성적서 수령 및 검토 | 교정담당자 |
| 4 | 합격 판정 및 라벨 부착 | 교정담당자 |
| 5 | 교정 기록 등록 및 장비 반환 | 교정담당자 |

### 7.2.2 사내 교정

사내 교정은 다음 조건을 충족하는 경우에만 실시:
- 상위 표준기가 공인기관에서 교정되어 있을 것
- 교정 절차서/지침서가 수립되어 있을 것
- 교정 수행자가 교육훈련을 이수하고 자격을 보유할 것
- 교정 환경(온도, 습도 등)이 적합할 것

### 7.2.3 교정 기관 요건

| 요건 | 내용 |
|------|------|
| 인정 기관 | ISO/IEC 17025 인정을 받은 교정기관 우선 |
| 추적성 | 국가/국제 표준에 대한 소급 추적성 보유 |
| 교정 성적서 | 측정 불확도, 교정 조건, 사용 표준기 정보 포함 |
| 정기 평가 | 연 1회 교정기관 적합성 평가 (QP-808 외부공급자 관리 연계) |

## 7.3 교정 합부 판정

| 판정 | 기준 | 후속 조치 |
|------|------|----------|
| 합격 (Pass) | 교정 결과가 허용 공차 이내 | 녹색 라벨 부착, 사용 승인 |
| 조건부 합격 | 일부 범위에서만 합격 | 황색 라벨 부착, 사용 가능 범위 명시 |
| 부적합 (Fail) | 허용 공차 초과 | 적색 라벨 부착, 사용 금지 → 9절 참조 |

# 8. 측정 불확도 관리

## 8.1 측정 불확도 요구사항

AS9100D 7.1.5.2에 따라, 측정 결과의 유효성을 보장하기 위해 측정 불확도를 평가하고 관리한다.

| 항목 | 요구사항 |
|------|---------|
| 불확도 평가 | 제품 합부 판정에 영향을 미치는 측정에 대해 불확도 평가 실시 |
| 판정 기준 반영 | 합부 판정 시 측정 불확도를 고려 (불확도 대비 공차 비율 확인) |
| 문서화 | 불확도 평가 결과를 기록으로 유지 |
| 고객 보고 | 고객 요구 시 측정 불확도를 검사 성적서에 포함 |

## 8.2 측정 불확도 적용 기준

| 구분 | 기준 |
|------|------|
| 일반 제품 측정 | 측정 불확도가 공차의 1/4 이하 → 합부 판정에 영향 없음 |
| 한계 측정값 | 측정값이 공차 한계 ± 불확도 범위 내 → 추가 판정 필요 |
| 항공우주 핵심 특성 | 고객 지정 핵심 특성(Key Characteristics)의 경우 불확도를 반드시 보고 |

## 8.3 MSA (측정시스템 분석)

| 분석 항목 | 기준 | 주기 |
|----------|------|------|
| GR&R (반복성/재현성) | %GR&R ≤ 10%: 합격, 10~30%: 조건부 합격, >30%: 부적합 | 신규 장비 도입 시 + 연 1회 |
| 편의 (Bias) | 기준값 대비 계통오차 평가 | 교정 시 |
| 안정성 (Stability) | 시간 경과에 따른 측정값 변동 | 분기 1회 |
| 선형성 (Linearity) | 측정 범위에 따른 편의 변화 | 신규 장비 도입 시 |

# 9. 부적합 장비 처리

## 9.1 부적합 발견 시 즉시 조치

```
부적합 발견 (교정 부적합, 손상, 봉인 훼손 등)
    │
    ▼
① 즉시 사용 중지 ──→ 적색 "사용 금지" 라벨 부착
    │
    ▼
② 격리 보관 ──→ 부적합 장비 보관 구역으로 이동
    │
    ▼
③ 품질팀장에게 보고 ──→ 부적합 장비 보고서 작성
    │
    ▼
④ 영향 평가 실시 ──→ 해당 장비로 측정한 제품의 유효성 평가
    │
    ▼
⑤ 처분 결정 ──→ 수리/재교정/폐기
    │
    ▼
⑥ 필요 시 고객 통보 ──→ 항공우주 고객 요구사항 확인
```

## 9.2 영향 평가 (Retrospective Analysis)

부적합 장비로 측정된 제품에 대해 다음을 평가한다:

| 평가 항목 | 내용 |
|----------|------|
| 영향 범위 | 최근 합격 교정일 이후 해당 장비로 검사한 모든 제품/로트 식별 |
| 재검사 필요성 | 교정 편차와 제품 공차를 비교하여 재검사 필요 여부 판정 |
| 고객 통보 | 출하된 제품에 영향이 있는 경우 고객에게 즉시 통보 |
| 시정조치 | QP-1001에 따라 CAR 발행 (필요 시) |

## 9.3 장비 처분

| 처분 유형 | 기준 | 조치 |
|----------|------|------|
| 수리 후 재교정 | 수리 가능한 경우 | 수리 완료 → 재교정 → 합격 시 사용 복귀 |
| 등급 변경 | 정밀도 저하 시 | 덜 엄격한 측정 용도로 재분류 |
| 폐기 | 수리 불가 또는 비경제적 | 폐기 기록 작성 → 물리적 폐기 |

# 10. 항공우주 특수 요구사항

## 10.1 교정 추적성

AS9100D는 측정 결과의 추적성을 특별히 강조한다:

| 요구사항 | 관리 방법 |
|---------|----------|
| 국가/국제 표준 소급성 | 모든 교정은 국가/국제 측정 표준에 소급 가능한 표준기로 실시 |
| 교정 사슬 문서화 | 교정 성적서에 사용 표준기의 추적성 정보 포함 확인 |
| 고객 지정 교정 | 고객이 교정기관 또는 방법을 지정한 경우 우선 적용 |

## 10.2 소프트웨어 검증

측정/검사에 사용되는 소프트웨어는 의도된 용도에 적합한지 확인한다:

| 단계 | 활동 |
|------|------|
| 초기 검증 | 도입 시 기지값(Known Value) 입력으로 출력 정확성 확인 |
| 정기 검증 | 연 1회 기지값 시험으로 지속적 적합성 확인 |
| 변경 시 | 소프트웨어 업데이트/변경 시 재검증 |
| 기록 | 검증 결과 기록 유지 |

## 10.3 고객 소유 측정장비

고객이 제공한 측정장비 또는 게이지에 대해:
- 수령 시 상태 검사 및 기록
- 사내 장비와 동일한 교정 관리 적용
- 손상 또는 부적합 발견 시 고객에게 즉시 통보
- 별도 식별 라벨 부착 ("고객 소유")

## 10.4 위임검사원(Delegated Inspector) 장비

고객의 위임검사원이 사용하는 측정장비에 대해:
- 위임검사원 자체 교정 기록의 유효성 확인
- 교정 상태 확인 후 사용 허가
- 부적합 장비 사용 시 즉시 중단 및 보고
