# 1. 목적

본 절차서는 M&C Electronics VINA에서 제품 인도 후 수행하는 활동(보증, 기술지원, 유지보수정보 제공, 스페어파트 공급, 반품 처리 등)에 관한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 8.5.5 요구사항에 따른 인도 후 활동의 체계적 관리
- 고객 불만 및 현장 반품(RMA)의 신속·체계적 처리
- 항공우주 분야 특수 의무(감항성 정보, 서비스 불리틴, 안전 통보 등) 이행
- 인도 후 활동 데이터를 품질 개선에 피드백하는 체계 확립

---

# 2. 적용 범위

본 절차서는 M&C Electronics VINA가 고객에게 인도한 모든 제품에 대한 인도 후 활동에 적용한다.

| 구분 | 적용 대상 |
|------|----------|
| 보증 (Warranty) | 계약에 명시된 보증 기간 내 품질 보증 활동 |
| 기술지원 | 고객의 기술 문의 응대, 설치/사용 지원 |
| 유지보수 정보 | 유지보수 매뉴얼, 정비 지침 제공 |
| 스페어파트 공급 | 교체 부품, 수리 부품 공급 |
| 고객 불만 처리 | 제품 품질 관련 고객 불만 접수 및 처리 |
| 현장 반품 (RMA) | 불량/결함 제품의 반환 접수 및 처리 |
| 감항성/안전 통보 | 항공우주 안전 관련 즉시 통보 (해당 시) |

**적용 제외**: 고객과의 계약에서 인도 후 활동이 명시적으로 제외된 경우에는 해당 항목을 적용하지 않는다. 단, 안전 관련 통보 의무는 계약 조건과 관계없이 적용한다.

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 인도 후 활동 (Post-Delivery Activities) | 제품을 고객에게 인도한 후 수행하는 모든 활동. 보증, 기술지원, 유지보수, 반품 처리, 안전 통보 등을 포함 |
| 보증 (Warranty) | 계약에 의해 제조자가 제품의 품질을 일정 기간 보장하는 약정. 보증 기간 내 결함에 대해 수리, 교환 또는 환불을 제공 |
| RMA (Return Material Authorization) | 반품 승인 — 고객이 불량/결함 제품을 제조자에게 반환하기 위해 사전 승인을 받는 절차 |
| 감항성 (Airworthiness) | 항공기 또는 항공기 부품이 안전하게 비행할 수 있는 상태 |
| 서비스 불리틴 (Service Bulletin, SB) | 제조자가 발행하는 기술 통보로, 제품의 개선, 수정 또는 검사를 권고하는 문서 |
| 감항성 지시 (Airworthiness Directive, AD) | 항공 당국이 발행하는 의무적 시정 조치 지시 |
| 고객 불만 (Customer Complaint) | 고객이 제품 또는 서비스의 품질, 성능, 납기 등에 대해 표명하는 불만족 |
| 필드 불량 (Field Failure) | 고객 현장에서 사용 중 발생한 제품 결함/고장 |
| MTBF (Mean Time Between Failures) | 평균 고장 간격 — 제품의 신뢰성을 나타내는 지표 |
| Root Cause Analysis | 근본 원인 분석 — 불량/결함의 근본적 발생 원인을 규명하는 체계적 분석 |

| **생산부서장** | 재작업/교환품 생산; 불량 원인 관련 공정 개선; 스페어파트 생산 |
| **자재/창고 담당자** | RMA 반품 입고 및 보관; 스페어파트 재고 관리; 교환품 출하 |

# 6. 인도 후 활동 범위

## 6.1 활동 유형별 상세

| 활동 유형 | 내용 | 대응 기한 |
|----------|------|----------|
| **보증 (Warranty)** | 보증 기간 내 제조 결함에 대한 무상 수리/교환/환불 | 계약에 따름 (일반: 접수 후 30일 이내 처리) |
| **기술지원** | 제품 설치, 사용, 정비에 관한 기술 문의 응대 | 접수 후 3영업일 이내 초기 응답 |
| **유지보수 정보 제공** | 정비 매뉴얼, 정비 주기, 교체 부품 정보 제공 | 고객 요청 시 10영업일 이내 |
| **스페어파트 공급** | 교체 부품, 수리용 부품 공급 | 계약에 따름 (일반: 발주 후 4주 이내) |
| **고객 불만 처리** | 품질/성능/납기 관련 고객 불만 접수 및 해결 | 7절 참조 |
| **RMA 반품 처리** | 불량/결함 제품 반환 접수 및 분석/처리 | 8절 참조 |
| **안전/감항성 통보** | 안전 관련 즉시 통보 의무 이행 | 9절 참조 (즉시) |

## 6.2 인도 후 활동 결정 시 고려 사항

AS9100D 8.5.5에 따라 인도 후 활동의 범위를 결정할 때 다음 사항을 고려한다:

| 고려 사항 | 세부 내용 |
|----------|----------|
| 법적/규제 요구사항 | 항공 당국(FAA, EASA, CAAV 등)의 감항성 요구사항 |
| 제품 관련 잠재적 원하지 않는 결과 | 안전 위험, 환경 영향, 재산 피해 가능성 |
| 제품의 성질, 용도 및 수명 | 항공우주 부품의 중요도, 사용 환경, 예상 수명 |
| 고객 요구사항 | 계약에 명시된 인도 후 지원 범위 |
| 고객 피드백 | 과거 불만/반품 이력, 고객 만족도 조사 결과 |

# 7. 고객 불만 접수 및 처리

## 7.1 고객 불만 접수

| 접수 채널 | 내용 |
|----------|------|
| 이메일 | 영업 담당자 또는 품질팀 이메일 |
| 전화 | 영업 담당자 또는 대표전화 |
| 공문 | 고객 공식 서한 |
| 고객 포털 | 고객사 품질 관리 시스템 (해당 시) |

모든 고객 불만은 접수 후 **24시간 이내**에 접수 확인을 고객에게 회신한다.

## 7.2 고객 불만 처리 절차

```
고객 불만 접수 (영업 담당자)
         ↓
접수 등록 — 고객 불만 대장 (QF-815-01) 기록
         ↓
접수 확인 회신 (24시간 이내)
         ↓
초기 분류 (품질팀장)
├─ 품질 불량 → 불량 분석 (기술부서)
├─ 납기 지연 → 영업/생산 확인
├─ 사양 불일치 → 기술부서 확인
└─ 안전 관련 → 즉시 QMR 보고 → 9절 참조
         ↓
원인 분석 (Root Cause Analysis)
- 반품 분석 (해당 시)
- 공정 데이터 확인
- 추적성 조사 (QP-812)
         ↓
시정조치 수립 (QP-1001 연계)
- 즉시 대응 (봉쇄 조치, 교환/수리 등)
- 재발 방지 대책
         ↓
고객 회신 — 원인 분석 결과 및 시정조치 보고
(접수 후 15영업일 이내, 고객 요구 기한 우선)
         ↓
시정조치 실행 및 효과성 검증
         ↓
고객 불만 종결 (품질팀장 승인)
         ↓
데이터 분석 — 경영검토 입력 (QP-904)
```

## 7.3 고객 불만 분류 및 대응 등급

| 등급 | 기준 | 대응 기한 |
|------|------|----------|
| **Critical** | 안전 관련, 감항성 영향, 항공기 운항 중단(AOG) | **즉시** 대응, 24시간 이내 초기 보고 |
| **Major** | 기능 불량, 사양 미달, 대량 불량 | 3영업일 이내 초기 분석, 15영업일 이내 완료 |
| **Minor** | 외관 불량, 포장 불량, 경미한 사양 편차 | 5영업일 이내 초기 분석, 30영업일 이내 완료 |

## 7.4 고객 불만 데이터 분석

품질팀은 분기별로 고객 불만 데이터를 분석하여 다음 사항을 경영검토에 보고한다:
- 고객 불만 건수 추이 (월별/분기별)
- 불만 유형별 분포 (파레토 분석)
- 주요 불량 유형 및 재발 현황
- 시정조치 효과성 검증 결과
- 고객별 불만 현황 (주요 고객 중심)

# 8. 현장 반품 처리 (RMA)

## 8.1 RMA 절차

```
고객 반품 요청
      ↓
영업 담당자: 반품 사유 확인, 보증 조건 확인
      ↓
RMA 번호 발행 (QF-815-02)
- RMA 번호 체계: RMA-YYMMDD-NNN
      ↓
고객에게 RMA 번호 및 반품 지침 통보
- 반품 포장 요건
- 반품 운송 방법
- 반품 주소
      ↓
반품 입고 (자재/창고 담당자)
- RMA 번호 확인
- 수량 및 외관 확인
- RMA 전용 보관 구역에 격리 보관
      ↓
반품 분석 (기술부서 + 품질팀)
- 외관 검사
- 기능 시험
- 불량 모드 분류
- 근본 원인 분석
      ↓
분석 보고서 작성 (QF-815-03)
      ↓
처분 결정 (품질팀장)
├─ 수리 후 반환 → 수리 → 재검사 → 출하
├─ 교환 → 양품 출하, 불량품 보관/폐기
├─ 환불 → 영업팀 환불 처리
└─ 고객 책임 (오사용 등) → 분석 결과 통보
      ↓
고객 회신 — 분석 결과 및 처분 결과 보고
      ↓
RMA 종결 및 기록 완료
```

## 8.2 RMA 분석 내용

반품 분석 보고서(QF-815-03)에는 다음 사항을 기록한다:

| 분석 항목 | 내용 |
|----------|------|
| 제품 식별 | 품번, 시리얼번호, 로트번호, 제조일, 출하일 |
| 고객 불량 증상 | 고객이 보고한 불량 내용 |
| 외관 검사 결과 | 외관 손상, 오염, 변형, 부식 등 |
| 기능 시험 결과 | 전기적/기계적/환경적 시험 결과 |
| 불량 모드 | 불량 유형 분류 (오픈, 쇼트, 기능 불량, 외관 불량 등) |
| 근본 원인 | 설계/자재/공정/취급/오사용 등 원인 분류 |
| 영향 범위 | 동일 로트/공정 제품에 대한 영향 평가 |
| 시정조치 | 재발 방지를 위한 시정조치 내용 |
| 처분 결과 | 수리/교환/환불/고객 책임 |

## 8.3 RMA 데이터 활용

- RMA 데이터는 MTBF 계산, 신뢰성 분석에 활용한다.
- 동일 불량 모드의 반복 발생 시 QP-1001에 따라 CAR(시정조치 요구서)를 발행한다.
- 분기별 RMA 데이터 분석 결과를 경영검토에 보고한다.

# 9. 항공우주 특수 요구사항

## 9.1 감항성 정보 제공

항공우주 제품의 감항성(Airworthiness)에 영향을 미치는 정보를 인지한 경우, 다음 절차에 따라 조치한다:

| 항목 | 요구사항 |
|------|----------|
| 감항성 영향 판단 | 제품 결함, 설계 결함, 공정 편차 등이 감항성에 영향 여부를 기술부서장이 판단 |
| 즉시 통보 대상 | 고객, 해당 항공 당국 (FAA, EASA, CAAV 등) |
| 통보 시한 | 감항성 영향 인지 후 **즉시** (24시간 이내) |
| 통보 내용 | 영향받는 제품 식별 정보, 결함 내용, 잠재 영향, 권고 조치 |
| 통보 승인 | QMR 승인 후 통보 (긴급 시 사후 승인 가능) |

## 9.2 서비스 불리틴 (Service Bulletin) 발행

제품 사용 중 개선, 수정 또는 추가 검사가 필요한 경우 서비스 불리틴을 발행한다:

| 항목 | 내용 |
|------|------|
| 발행 사유 | 설계 개선, 공정 개선, 안전 관련 수정, 성능 향상 |
| 작성 | 기술부서장 |
| 검토/승인 | QMR 승인 |
| 배포 대상 | 영향받는 제품을 사용하는 모든 고객 |
| 분류 | Mandatory(의무) / Recommended(권고) / Optional(선택) |
| 추적 | 서비스 불리틴 이행 현황 추적 관리 |

서비스 불리틴에는 다음 정보를 포함한다:
- 서비스 불리틴 번호 (SB-YYMMDD-NNN)
- 영향받는 제품 식별 (품번, 시리얼번호 범위, 로트번호)
- 수정/개선 내용 상세 설명
- 조치 방법 (작업 지침)
- 분류 (Mandatory / Recommended / Optional)
- 조치 기한 (해당 시)
- 필요 부품/자재 목록

## 9.3 안전 관련 즉시 통보

제품 안전에 관련된 결함 또는 위험을 인지한 경우, 다음 절차를 즉시 수행한다:

```
안전 관련 결함/위험 인지
         ↓
기술부서장 → QMR 즉시 보고
         ↓
안전 영향도 평가 (4시간 이내)
├─ 즉각적 위험 → 모든 고객에게 즉시 사용 중지 권고
├─ 잠재적 위험 → 영향 범위 조사 후 24시간 이내 통보
└─ 위험 아님 → 일반 시정조치 절차 (QP-1001)
         ↓
고객 통보 (전화 + 이메일/공문)
         ↓
해당 항공 당국 통보 (해당 시)
         ↓
시정조치 수립 및 실행
         ↓
서비스 불리틴 발행 (해당 시)
         ↓
조치 완료 확인 및 기록
```

**안전 관련 즉시 통보 대상**:
- 항공기 또는 승객/승무원의 안전에 위험을 초래할 수 있는 결함
- 감항성에 영향을 미치는 제품 결함
- 인명 피해 또는 재산 피해를 초래할 수 있는 결함

## 9.4 수리/오버홀 지원

고객이 현장에서 수리 또는 오버홀을 수행하는 경우 다음 지원을 제공한다:

| 지원 항목 | 내용 |
|----------|------|
| 수리 매뉴얼 제공 | 구성요소 수리 매뉴얼(CMM) 제공 (해당 시) |
| 기술 문의 대응 | 수리/오버홀 관련 기술 문의에 3영업일 이내 응답 |
| 스페어파트 공급 | 수리/오버홀에 필요한 교체 부품 공급 |
| 기술 자문 | 현장 수리 가능 여부 판단, 수리 방법 자문 |
| 수리 후 시험 기준 | 수리/오버홀 후 검사/시험 기준 제공 |

## 9.5 제품 수명 주기 지원

| 활동 | 내용 | 빈도 |
|------|------|------|
| 설계 변경 통보 | 제품 설계 변경 시 영향받는 고객에게 사전 통보 | 변경 발생 시 |
| 단종 통보 (EOL Notice) | 제품 단종 결정 시 최소 12개월 전 고객 통보 | 단종 결정 시 |
| 최종 구매 기회 (Last Time Buy) | 단종 전 최종 구매 기회 제공 | 단종 전 6개월 |
| 장기 스페어파트 보유 | 단종 후 계약에 명시된 기간 스페어파트 보유 | 계약에 따름 |
