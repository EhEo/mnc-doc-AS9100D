# 1. 목적

본 절차서는 M&C Electronics VINA의 QMS 운영에 관련된 조직의 역할, 책임 및 권한을 규정하고, 이를 조직 내에서 부여하고 전달하는 절차를 규정한다.

- AS9100D 5.3에 따른 역할, 책임 및 권한의 부여와 전달
- QMS가 AS9100D 요구사항에 적합하도록 보장하는 인원 지정
- 항공우주 산업 특수 역할(제품안전 책임자, 형상관리 위원회 등) 지정
- 프로세스의 의도된 출력을 달성하도록 보장
- 고객 요구사항 및 법적/규제적 요구사항 충족에 대한 인식 촉진

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 최고경영자 (Top Management) | 최고 수준에서 조직을 지휘하고 관리하는 인원 또는 그룹. 본 조직에서는 대표이사 |
| QMR (Quality Management Representative) | 품질책임자. QMS의 수립, 실행, 유지 및 개선에 대한 권한과 책임을 위임받은 경영진 구성원 |
| 프로세스 책임자 (Process Owner) | 특정 프로세스의 효과적 운영과 개선에 대한 책임을 가진 인원 |
| CCB (Configuration Control Board) | 형상통제위원회. 형상 변경의 승인/거부를 결정하는 다기능팀 |
| 제품안전 책임자 | AS9100D 요구에 따라 제품 안전 관련 이슈를 관리하고 에스컬레이션하는 지정 인원 |
| 권한 위임 (Delegation of Authority) | 특정 업무에 대한 결정 권한을 타인에게 이전하는 행위 |

# 5. 책임과 권한 총괄

## 5.1 핵심 역할 매트릭스

| 역할 | 주요 책임 | 주요 권한 | AS9100D 조항 |
|------|----------|----------|-------------|
| **대표이사** | QMS 총괄 책임, 자원 배분, 품질방침 승인 | 최종 승인 권한, 조직 변경 권한 | 5.1, 5.2 |
| **품질책임자 (QMR)** | QMS 수립/유지/개선, 규격 적합성 보장, 경영진 보고 | QMS 의사결정, 부적합품 출하 정지 | 5.3 |
| **제품안전 책임자** | 제품 안전 요구사항 식별/관리, 안전 이슈 에스컬레이션 | 안전 관련 출하 정지 권한 | 8.1.4 |
| **기술팀장** | 설계/개발 관리, 형상관리(CM), 기술 문서 관리 | 설계 변경 승인, CCB 의장 | 8.3, 8.1.3 |
| **생산팀장** | 생산 공정 관리, FOD 방지 프로그램 실행 | 공정 변경 요청, 생산 정지 권한 | 8.5 |
| **품질팀장** | 검사/시험, 내부심사, 시정조치, 교정 관리 | 부적합품 판정, 심사 시행 권한 | 8.6, 9.2 |
| **구매팀장** | 공급자 관리, ASL 유지, 외주공정 관리 | 공급자 선정/제거 권한 | 8.4 |
| **영업팀장** | 고객 요구사항 접수, 계약검토, CSR 전달 | 계약 체결 권한 | 8.2 |
| **인사팀장** | 인력 확보, 교육훈련, 자격 관리 | 교육 기획/실행 권한 | 7.2, 7.3 |

## 5.2 부적합/안전 관련 특수 권한

**출하 정지 권한**: 다음 인원은 품질 또는 안전 문제 발견 시 독립적으로 출하를 정지할 수 있다:

| 권한자 | 정지 사유 | 재개 조건 |
|--------|----------|----------|
| 대표이사 | 모든 사유 | 본인 판단 |
| QMR | QMS 부적합, 규격 미충족 | QMR 승인 |
| 제품안전 책임자 | 제품 안전 관련 이슈 | 안전성 검증 완료 + QMR 승인 |
| 품질팀장 | 검사 부적합, 공정 이탈 | 시정조치 완료 + QMR 승인 |

> **원칙**: 출하 정지 권한은 상위 직급의 압력에 의해 무시될 수 없다. 안전 관련 출하 정지는 반드시 기술적 검증을 거친 후에만 해제한다.

# 7. 핵심 역할별 상세 책임 및 권한

## 7.1 대표이사

**AS9100D 5.1 리더십과 의지에 따른 책임:**

| 책임 | 실증 방법 |
|------|----------|
| QMS의 효과성에 대한 최종 책임 | 경영검토 주재, 품질방침 승인 |
| 품질방침/품질목표 수립 및 전략 일치 | QP-501에 따른 연간 검토 및 승인 |
| QMS 요구사항의 비즈니스 프로세스 통합 | 조직 전략에 QMS 요건 반영 |
| 프로세스 접근법 및 리스크 기반 사고 촉진 | 리스크 관리 자원 배분, 리스크 검토 참여 |
| 필요 자원의 가용성 보장 | 예산 승인, 인력 배치 |
| 효과적 품질경영의 중요성 전달 | 품질 회의 참여, 현장 방문 |
| QMS의 의도된 결과 달성 보장 | 품질목표 달성 현황 검토 |
| 인원의 참여 지휘 및 지원 | 교육 지원, 포상 제도 |
| 개선 촉진 | 개선 제안 제도, 자원 배분 |

## 7.2 품질책임자 (QMR)

| 구분 | 내용 |
|------|------|
| **지정 요건** | 경영진의 일원으로서 대표이사가 임명 (임명장 발행) |
| **독립성** | 품질에 관한 판단 시 생산/납기 압력으로부터 독립적 의사결정 보장 |
| **보고 라인** | 대표이사에게 직접 보고 |

**상세 책임:**

| 책임 | 활동 |
|------|------|
| QMS 적합성 보장 | AS9100D 전 조항 요구사항 충족 유지 |
| 프로세스 성과 보고 | 경영검토 시 QMS 성과 지표 보고 |
| 고객 요구사항 인식 촉진 | 고객 요구사항의 조직 내 전달/교육 |
| QMS 변경 시 무결성 유지 | QMS 변경의 기획 및 적합성 검토 |
| 외부 의사소통 | 인증기관, 규제당국과의 QMS 관련 창구 |
| 부적합 관리 | 시정조치 프로세스 총괄, CAR 추적 |
| 내부심사 프로그램 관리 | 심사 계획, 심사원 자격 관리 |

**주요 권한:**
- QMS 관련 의사결정 권한
- 부적합품 출하 정지 권한
- 시정조치 요구 및 추적 권한
- QMS 문서 최종 검토/승인 권한 (2계층)
- 경영검토 소집 요청 권한

# 8. 항공우주 특수 역할

## 8.1 제품안전 책임자

AS9100D 8.1.4에 따라 제품 안전에 관한 전담 책임자를 지정한다.

| 항목 | 내용 |
|------|------|
| **지정** | 대표이사가 임명 (임명장 발행) |
| **자격 요건** | 항공우주 품질/안전 관련 경력 5년 이상; 제품 안전 교육 이수 |
| **독립성** | 안전 관련 판단 시 생산/납기 일정과 무관하게 독립적 의사결정 |
| **보고 라인** | 대표이사에게 직접 보고 (품질 조직과 별도 보고 가능) |

**상세 책임:**

| 책임 | 활동 |
|------|------|
| 안전 요구사항 식별 | 고객 요구, 법규, 규격에서 안전 관련 요건 식별 |
| 안전 리스크 평가 | 제품/공정 안전 리스크 분석 및 완화 조치 |
| 안전 이슈 에스컬레이션 | 안전 문제 발생 시 경영진 및 고객에 즉시 보고 |
| 안전 관련 출하 정지 | 안전 위험 시 출하 즉시 정지 (독립 권한) |
| 안전 기록 관리 | 안전 관련 이슈, 조치, 결과 기록 유지 |
| 안전 교육 | 제품 안전 관련 교육/인식 활동 |

## 8.2 형상통제위원회 (CCB)

AS9100D 8.1.3에 따라 형상 변경의 승인/거부를 결정하는 다기능팀을 구성한다.

| 역할 | 구성원 | 책임 |
|------|--------|------|
| 의장 | 기술팀장 | CCB 회의 주재, 최종 결정 |
| 상임위원 | QMR | 품질 영향 검토 |
| 상임위원 | 생산팀장 | 제조성/공정 영향 검토 |
| 상임위원 | 품질팀장 | 검사/시험 영향 검토 |
| 비상임위원 | 구매팀장 | 공급자/자재 영향 검토 (해당 시) |
| 비상임위원 | 영업팀장 | 고객 요구/계약 영향 검토 (해당 시) |

**CCB 운영 원칙:**
- 형상 변경 요청(ECR)에 대해 영향 분석 후 승인/거부/보류 결정
- 의장 포함 상임위원 과반수 출석으로 회의 성립
- 결정은 출석 위원 과반수 합의로 확정
- 안전 관련 변경은 제품안전 책임자 동의 필수
- 모든 결정은 문서화하여 형상관리 기록으로 보존

## 8.3 내부심사 팀

AS9100D 9.2에 따라 내부심사를 수행하는 역할을 지정한다.

| 역할 | 자격 요건 | 독립성 요건 |
|------|----------|-----------|
| 수석심사원 | AS9100D 내부심사원 교육 이수 + 심사 경력 2회 이상 | 심사 대상 업무와 무관한 인원 |
| 심사원 | AS9100D 내부심사원 교육 이수 | 심사 대상 업무와 무관한 인원 |
| 심사 프로그램 관리자 | QMR 또는 QMR이 지정한 인원 | — |

# 9. 역할 부여 및 전달

## 9.1 역할 부여 절차

| 단계 | 활동 | 책임자 | 산출물 |
|------|------|--------|--------|
| 1 | 조직 구조 및 역할 설계/검토 | 대표이사/QMR | 조직도 (최신) |
| 2 | 직무기술서(JD) 작성/갱신 | 인사팀장/각 부서장 | F-502-01 직무기술서 |
| 3 | 항공우주 특수 역할 지정 | 대표이사 | 임명장 (F-502-02) |
| 4 | 역할/책임/권한 문서화 | QMR | 본 절차서 및 관련 절차서 |
| 5 | 해당 인원에 전달 | 인사팀장/각 부서장 | 전달 확인 서명 |
| 6 | 역량 확인 | 인사팀장 | 역량 평가 기록 |

## 9.2 전달 방법

| 전달 대상 | 전달 방법 | 시기 |
|----------|----------|------|
| 신규 임명자 | 임명장 교부 + 직무 오리엔테이션 | 임명 즉시 |
| 기존 인원 (역할 변경 시) | 변경 통보 + 인수인계 | 변경 시행 전 |
| 전 임직원 | 조직도 게시/배포 + DMS 등록 | 변경 발생 시 |
| 신규 입사자 | 입사 교육 시 조직 구조/역할 안내 | 입사 1주 이내 |

## 9.3 역할 변경 트리거

다음 상황 발생 시 조직 역할/책임/권한을 검토하고 필요 시 갱신한다:

- 조직 구조 변경 (부서 신설/통합/폐지)
- 인사 이동 (핵심 역할 담당자 변경)
- QMS 범위 변경
- 인증심사 부적합 지적 (역할/책임 관련)
- 경영검토 결과 (조직 효과성 개선 필요)
- 고객/규제 요구사항 변경

# 10. 권한 위임

## 10.1 위임 원칙

| 원칙 | 설명 |
|------|------|
| 문서화 | 모든 권한 위임은 서면으로 기록 (F-502-03) |
| 역량 확인 | 위임받는 자의 역량이 해당 업무에 적합한지 확인 |
| 범위 한정 | 위임 범위와 기간을 명확히 특정 |
| 책임 불가분 | 권한 위임 시에도 최종 책임은 원래 책임자에게 잔존 |
| 제한 사항 | 아래 권한은 위임 불가 |

## 10.2 위임 불가 권한

다음 권한은 지정된 책임자만 행사할 수 있으며, 위임할 수 없다:

| 권한 | 지정 책임자 | 사유 |
|------|-----------|------|
| 품질방침 승인 | 대표이사 | AS9100D 5.2 최고경영자 책임 |
| QMR 임명 | 대표이사 | AS9100D 5.3 |
| 제품안전 관련 출하 정지 해제 | 제품안전 책임자 + QMR | 안전 최우선 원칙 |
| CCB 형상 변경 최종 결정 | CCB 의장 | 형상관리 무결성 |
| 인증심사 대응 공식 답변 | QMR | 규격 적합성 보장 |

## 10.3 위임 절차

1. 위임자가 권한위임서(F-502-03)를 작성
2. 수임자의 역량 적합성 확인 (인사팀장/부서장)
3. 위임자의 직속 상급자 승인
4. 수임자에게 전달 및 서명
5. 위임 기간 중 위임자가 수시 모니터링
6. 위임 종료 시 권한위임서 폐기 및 기록 보관
