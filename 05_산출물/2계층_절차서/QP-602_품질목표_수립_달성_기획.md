# 1. 목적

본 절차서는 M&C Electronics VINA가 품질방침과 일관된 품질목표를 관련 기능, 계층 및 프로세스에서 수립하고, 달성을 위한 기획을 체계적으로 수행하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 6.2 요구사항에 따른 품질목표의 체계적 수립 및 달성 기획
- SMART 원칙에 기반한 측정 가능한 품질목표 설정
- 품질목표의 전사적 전개(Deployment) 및 성과 추적
- 품질목표와 프로세스 성과 지표(KPI)의 연계

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 품질목표 (Quality Objective) | 품질에 관련하여 추구하거나 목표로 하는 결과. 품질방침과 일관되어야 함 |
| 품질방침 (Quality Policy) | 최고경영자가 공식적으로 표명한 품질에 관한 조직의 의도와 방향 |
| SMART | Specific(구체적), Measurable(측정 가능), Achievable(달성 가능), Relevant(관련성), Time-bound(기한 설정) |
| 전개 (Deployment) | 전사 품질목표를 부서별/프로세스별 세부 목표로 분해하여 할당하는 활동 |
| KPI | 핵심 성과 지표 (Key Performance Indicator) |
| 선행지표 (Leading Indicator) | 미래 성과를 예측하는 지표 (예: 교육 이수율, 예방조치 건수) |
| 후행지표 (Lagging Indicator) | 결과를 측정하는 지표 (예: 불량률, 고객 불만 건수) |

# 6. 품질목표 수립

## 6.1 품질목표 수립 원칙

AS9100D 6.2.1에 따라 품질목표는 다음 요건을 충족해야 한다:

| 요건 | 내용 |
|------|------|
| **a) 품질방침과 일관** | 품질방침의 의도와 방향에 부합 |
| **b) 측정 가능** | 정량적 지표와 목표치로 표현 (SMART) |
| **c) 적용되는 요구사항 고려** | 고객, 규격, 법적 요구사항 반영 |
| **d) 적합성 관련** | 제품/서비스의 적합성 및 고객만족 향상에 기여 |
| **e) 모니터링** | 달성도를 주기적으로 모니터링 |
| **f) 의사소통** | 관련 인원에게 전달 |
| **g) 갱신** | 적절하게 갱신 |

## 6.2 품질목표 수립 프로세스

```
① 품질방침 검토
    │
    ▼
② 외부/내부 이슈 및 이해관계자 요구 검토 (QP-401)
    │
    ▼
③ 리스크/기회 분석 결과 검토 (QP-601)
    │
    ▼
④ 전년도 품질목표 달성도 검토
    │
    ▼
⑤ 전사 품질목표 수립 (QMR 초안 → 대표이사 승인)
    │
    ▼
⑥ 부서별 품질목표 전개 (각 부서장)
    │
    ▼
⑦ 달성 기획 수립 (7절)
    │
    ▼
⑧ 전 임직원 의사소통 (게시, 교육, 회의)
```

## 6.3 품질목표 수립 시기

| 활동 | 시기 |
|------|------|
| 전사 품질목표 수립 | 매년 1월 (또는 경영검토 직후) |
| 부서별 목표 전개 | 전사 목표 승인 후 2주 이내 |
| 중간 검토/갱신 | 반기 경영검토 시 (필요 시 목표 조정) |
| 최종 평가 | 연말 경영검토 시 |

## 6.4 품질목표 체계

### 기존 품질목표 (ISO 9001/IATF 16949 기반)

> 기존 품질목표(ISO.03.05.03, 2024-01-05 발행)는 다음 4개 항목으로 운영되고 있다:

| # | 기존 품질목표 (Mục tiêu chất lượng 2024) | 목표치 |
|---|--------------------------------------|-------|
| 1 | 효율(히에우 수앗) 달성률 | ≥ 95% |
| 2 | 고객처 품질 발생 건수 | < 48건/년 |
| 3 | 고객 클레임 비용 vs 매출 비율 | < 5% |
| 4 | 채용 및 교육 완료율 | ≥ 90% |

### AS9100D 확장 품질목표

기존 4개 항목을 유지하면서 **항공우주 필수 목표**를 추가한다:

| 목표 영역 | 품질목표 | 지표 | 목표치 | 측정 주기 | 비고 |
|----------|---------|------|--------|----------|------|
| 효율 | 생산 효율 달성 | 효율률 (Hiệu suất) | ≥ 95% | 월간 | *기존 항목 1 유지* |
| 고객 품질 | 고객처 품질이슈 감소 | 고객 품질 발생 건수 | < 48건/년 | 월간 | *기존 항목 2 유지* |
| 클레임 비용 | 클레임 비용 억제 | 클레임비용/매출 | < 5% | 월간 | *기존 항목 3 유지* |
| 인적자원 | 채용/교육 이행 | 채용·교육 완료율 | ≥ 90% | 분기 | *기존 항목 4 유지* |
| **납기** *(AS9100D 추가)* | 납기 준수율 향상 | OTD (On-Time Delivery) | ≥ 95% | 월간 | AS9100D 9.3 필수 |
| **제품 안전** *(AS9100D 추가)* | 안전 관련 부적합 | 안전 부적합 건수 | 0건 | 월간 | AS9100D 8.1.4 |
| **FAI** *(AS9100D 추가)* | FAI 적시 완료 | FAI 기한 내 완료율 | 100% | 분기 | AS9102 |

### 부서별 목표 전개 (예시)

| 전사 목표 | 부서 | 부서 목표 | 부서 지표 | 목표치 |
|----------|------|----------|----------|--------|
| OTD ≥ 95% | 생산팀 | 생산 계획 달성률 | 계획 대비 실적 | ≥ 98% |
| OTD ≥ 95% | 구매팀 | 자재 적시 입고율 | 입고 지연 건수 | ≤ 2건/월 |
| 불량률 ≤ 5,000 PPM | 품질팀 | 검출 불량률 | 검사 불량률 | 공정 불량률의 80% 이상 검출 |
| 불량률 ≤ 5,000 PPM | 생산팀 | 공정 내 불량률 | 공정별 불량률 | ≤ 3,000 PPM |

# 7. 달성 기획

## 7.1 달성 계획 수립

AS9100D 6.2.2에 따라 품질목표 달성을 위한 기획 시 다음을 결정한다:

| 기획 항목 | 내용 |
|----------|------|
| **a) 수행할 사항** | 구체적인 실행 활동/과제 |
| **b) 필요한 자원** | 인력, 장비, 예산, 시간 |
| **c) 책임자** | 각 활동의 실행 책임자 |
| **d) 완료 시기** | 각 활동의 목표 완료일 |
| **e) 결과 평가 방법** | 달성도 측정 방법 및 평가 기준 |

## 7.2 달성 계획서 양식

각 품질목표에 대해 F-602-01 (품질목표 달성 계획서)을 작성한다:

| 항목 | 기재 내용 |
|------|----------|
| 품질목표 | 목표 문구 및 목표치 |
| 연계 품질방침 | 관련 품질방침 항목 |
| 현재 수준 | 직전 연도 또는 현재 실적 |
| 목표 수준 | 금년도 달성 목표치 |
| 실행 과제 | 목표 달성을 위한 구체적 활동 목록 |
| 담당자/부서 | 각 과제별 책임자 |
| 일정 | 과제별 시작일/완료일 |
| 필요 자원 | 인력, 예산, 장비 등 |
| 측정 방법 | 데이터 수집 방법 및 측정 주기 |
| 보고 주기 | 달성도 보고 주기 (월간/분기) |

# 8. 모니터링 및 평가

## 8.1 모니터링 체계

| 활동 | 주기 | 책임자 | 산출물 |
|------|------|--------|--------|
| 데이터 수집 | 월간 | 각 부서 | 월간 KPI 데이터 |
| 달성 현황 분석 | 월간 | 품질팀장 | 품질목표 달성 현황 보고서 |
| 분기 검토 | 분기 | QMR | 분기 종합 보고 |
| 경영검토 보고 | 반기 | QMR | 경영검토 입력 (QP-904) |
| 연간 최종 평가 | 연말 | QMR | 연간 품질목표 평가 보고서 |

## 8.2 목표 달성도 평가 기준

| 달성도 | 기준 | 후속 조치 |
|--------|------|----------|
| **달성 (Achieved)** | 목표치 100% 이상 달성 | 다음 연도 목표 상향 검토 |
| **근접 (Near Target)** | 목표치의 90~99% 달성 | 잔여 기간 집중 관리 |
| **미달 (Below Target)** | 목표치의 70~89% 달성 | 원인 분석 + 개선 조치 수립 |
| **심각 미달 (Critical)** | 목표치의 70% 미만 | 긴급 개선 조치 + 경영검토 보고 |

## 8.3 목표 미달 시 조치

| 단계 | 활동 | 책임자 |
|------|------|--------|
| 1 | 미달 원인 분석 (데이터 분석, 근본 원인 파악) | 해당 부서장 |
| 2 | 개선 조치 수립 | 해당 부서장 |
| 3 | QMR 검토/승인 | QMR |
| 4 | 개선 조치 실행 | 해당 부서 |
| 5 | 효과 확인 | 품질팀장 |
| 6 | 필요 시 목표 재설정 (합리적 근거 필요, 대표이사 승인) | QMR |

# 9. 항공우주 특수 요구사항

## 9.1 항공우주 필수 품질목표

AS9100D에서 특별히 강조하는 항공우주 관련 품질목표 영역:

| 영역 | 필수 목표 | 비고 |
|------|----------|------|
| **납기 성과 (OTD)** | 납기 준수율 목표 설정 필수 | AS9100D 9.3 경영검토 입력 |
| **제품 안전** | 안전 관련 부적합 0건 목표 | AS9100D 8.1.4 |
| **초도품 검사 (FAI)** | FAI 적시 완료율 | AS9102 |
| **고객 불만** | 고객 불만 건수 감소 또는 대응 시간 | AS9100D 9.1.2 |
| **공급자 품질** | 공급자 품질 성과 (입고 불량률) | AS9100D 8.4 |

## 9.2 고객 요구 품질목표

항공우주 고객이 특정 품질목표를 요구하는 경우:
- 고객 요구 목표를 전사/부서 품질목표에 포함
- 고객 요구 목표치가 자사 목표보다 높은 경우 고객 요구 우선 적용
- 고객별 품질목표 이행 현황을 별도 추적

## 9.3 인증심사 대비

인증심사 시 심사원은 다음을 확인한다:
- 품질목표가 품질방침과 일관되는지
- 목표가 SMART 원칙에 부합하는지
- 달성 기획이 구체적으로 수립되었는지
- 모니터링이 실제로 이루어지고 있는지 (데이터 증거)
- 미달 시 적절한 조치가 취해지는지
