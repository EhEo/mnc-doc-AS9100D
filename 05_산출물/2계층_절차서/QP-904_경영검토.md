# 1. 목적

본 절차서는 M&C Electronics VINA의 최고경영자가 품질경영시스템(QMS)의 지속적인 적합성, 충족성, 효과성 및 전략적 방향과의 정렬을 보장하기 위해 실시하는 경영검토의 절차를 규정한다.

본 절차서의 목적:
- AS9100D 9.3 요구사항에 따른 체계적 경영검토 실시
- QMS의 전략적 방향과의 정렬 확인
- 개선 기회 식별 및 자원 배분 결정
- 항공우주 산업 특수 요구사항의 경영 차원 검토

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 경영검토 (Management Review) | 최고경영자가 QMS의 적합성, 충족성, 효과성 및 전략적 방향과의 정렬을 평가하는 체계적 활동 |
| 경영검토 입력 (Input) | 검토를 위해 수집·분석된 데이터 및 정보 |
| 경영검토 출력 (Output) | 검토 결과에 따른 결정 사항 및 조치 지시 |
| 적합성 (Suitability) | QMS가 조직의 상황과 전략적 방향에 적합한 정도 |
| 충족성 (Adequacy) | QMS가 요구사항을 충족하기에 충분한 정도 |
| 효과성 (Effectiveness) | 계획된 활동이 실행되고 계획된 결과가 달성되는 정도 |
| 납기 준수율 (OTD) | On-Time Delivery — 약속된 납기에 맞춰 인도한 비율 |

| **품질팀장** | 경영검토 보고서 초안 작성; 데이터 수집/분석; 후속 조치 사항 추적 관리; 회의록 작성 |

# 7. 경영검토 입력 (Input)

## 7.1 AS9100D 필수 입력 사항

AS9100D 9.3.2에 따라 다음 사항을 경영검토 입력으로 포함한다:

| 입력 항목 | 데이터 출처 | 보고 부서 |
|----------|-----------|----------|
| **a) 이전 경영검토 조치 사항 이행 현황** | 이전 경영검토 회의록 | QMR |
| **b) 외부/내부 이슈의 변화** | 조직 상황 분석 (QP-401) | QMR |
| **c) QMS 성과 및 효과성 정보**: | | |
| - 고객 만족 및 이해관계자 피드백 | 고객 만족도 조사, 불만 현황 | 영업팀 |
| - 품질목표 달성도 | 품질목표 모니터링 결과 (QP-501) | 품질팀 |
| - 프로세스 성과 | 프로세스 KPI 실적 | 각 부서 |
| - 부적합 및 시정조치 현황 | CAR 현황, 추세 분석 (QP-1001) | 품질팀 |
| - 모니터링/측정 결과 | 검사 실적, 불량률 추세 | 품질팀 |
| - 내부심사 결과 | 심사 보고서, NCR 현황 (QP-903) | 품질팀 |
| - 외부공급자 성과 | 공급자 평가 결과 (QP-808) | 구매팀 |
| **d) 자원의 충족성** | 인력, 장비, 시설 현황 | 각 부서 |
| **e) 리스크/기회 관리 효과성** | 리스크 등록부 (QP-601/802) | 품질팀 |
| **f) 개선 기회** | 제안, 벤치마킹, 기술 동향 | 전 부서 |

## 7.2 항공우주 추가 입력 사항

AS9100D는 ISO 9001 대비 다음 항목을 추가로 경영검토 입력에 포함하도록 요구한다:

| 추가 입력 항목 | 데이터 출처 |
|--------------|-----------|
| **납기 성과 (OTD)** | 납기 준수율 데이터 |
| **제품 안전 관련 이슈** | 안전 사고/이슈 보고 (QP-804) |
| **위조 부품 방지 활동** | 위조 부품 관련 보고 |
| **고객 특수 요구사항 이행** | 고객별 특수 요구사항 이행 현황 |
| **규제 변경 사항** | 항공우주 법규/규격 변경 영향 |

# 8. 경영검토 실시

## 8.1 검토 진행

| 순서 | 의제 | 소요 시간(권장) |
|------|------|---------------|
| 1 | 개회 및 이전 조치 사항 이행 확인 | 15분 |
| 2 | QMS 성과 종합 보고 (품질목표, KPI) | 20분 |
| 3 | 고객 만족/불만 및 납기 성과 | 15분 |
| 4 | 내부심사 및 부적합/시정조치 현황 | 15분 |
| 5 | 공급자 성과 및 외부 이슈 | 10분 |
| 6 | 항공우주 특수 사항 (안전, 위조부품, 규제) | 10분 |
| 7 | 리스크/기회 관리 효과성 | 10분 |
| 8 | 자원 적절성 및 개선 기회 | 10분 |
| 9 | 결정 사항 및 조치 지시 (경영검토 출력) | 15분 |
| | **합계** | **약 2시간** |

## 8.2 검토 원칙

- **데이터 기반 판단**: 정량적 데이터와 추세를 근거로 판단
- **전략적 시각**: 단기 성과뿐 아니라 중장기 전략과의 정렬 검토
- **균형적 관점**: 긍정적 성과와 개선 필요 사항 모두 검토
- **의사결정 중심**: 보고에 치중하지 않고 결정과 조치에 집중

# 9. 경영검토 출력 (Output)

## 9.1 AS9100D 필수 출력 사항

| 출력 항목 | 내용 |
|----------|------|

# 10. 항공우주 특수 요구사항

## 10.1 납기 성과 검토

AS9100D는 경영검토 시 **납기 성과(On-Time Delivery)**를 필수 입력으로 요구한다:

| 검토 항목 | 내용 |
|----------|------|
| OTD 실적 | 월별/분기별 납기 준수율 추세 |
| 지연 원인 분석 | 납기 미달 건의 원인 분류 |
| 개선 조치 | 납기 개선을 위한 조치 사항 |
| 목표 대비 실적 | 품질목표 대비 OTD 실적 평가 |

## 10.2 제품 안전 검토

| 검토 항목 | 내용 |
|----------|------|
| 안전 관련 사고/이슈 | 보고 기간 중 발생한 안전 관련 사항 |
| 안전 요구사항 이행 | 제품 안전 관리 절차(QP-804)의 이행 현황 |
| 안전 관련 시정조치 | 안전 관련 CAR의 처리 현황 |
| 규제 변경 | 안전 관련 규제/규격 변경 사항 |

## 10.3 리스크 기반 검토

| 검토 항목 | 내용 |
|----------|------|
| 리스크 등록부 현황 | 식별된 리스크의 현재 상태 및 조치 현황 |
| 신규 리스크 | 새로 식별된 리스크 및 대응 계획 |
| 리스크 완화 효과 | 기존 리스크 완화 조치의 효과성 |
| 기회 활용 | 식별된 기회의 활용 현황 |

# 11. 후속 조치 관리

## 11.1 조치 사항 추적

QMR은 경영검토 출력(조치 사항)의 이행을 체계적으로 추적한다.

| 추적 활동 | 주기 | 책임자 |
|----------|------|--------|
| 조치 사항 이행 현황 점검 | 월 1회 | QMR |
| 기한 초과 사항 에스컬레이션 | 즉시 | QMR → 대표이사 |
| 이행 완료 확인 | 완료 보고 시 | QMR |
| 차기 경영검토 보고 | 차기 검토 시 | 품질팀장 |

## 11.2 미완료 사항 관리

| 상태 | 조치 |
|------|------|
| 기한 내 완료 | 완료 보고 및 기록 |
| 기한 연장 필요 | 사유 기술 + QMR 승인으로 기한 재설정 |
| 2회 이상 기한 초과 | 대표이사 에스컬레이션, 자원 재배분 검토 |
