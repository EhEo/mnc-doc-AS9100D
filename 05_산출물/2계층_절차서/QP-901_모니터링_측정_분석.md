# 1. 목적

본 절차서는 M&C Electronics VINA의 품질경영시스템 및 제품/서비스의 성과를 모니터링, 측정, 분석 및 평가하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 9.1.1 / 9.1.3 요구사항에 따른 체계적 모니터링 및 측정 수행
- 프로세스 성과와 제품 적합성을 객관적 데이터 기반으로 평가
- 통계적 기법(SPC, 파레토 분석, 추세 분석)을 활용한 과학적 데이터 분석 체계 구축
- QMS 개선 기회의 식별 및 의사결정 근거 제공

> **CAR-014 대응**: 본 절차서는 갭 분석 결과 식별된 CAR-014 (모니터링/측정/분석 절차 부재)에 대한 시정조치로서 제정되었다. 기존 SPC 활동(Folder 30)을 AS9100D 9.1 요구사항에 맞게 체계화하고 확장한다.

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| KPI | Key Performance Indicator, 핵심성과지표. 프로세스/제품 성과를 정량적으로 측정하는 지표 |
| SPC | Statistical Process Control, 통계적 공정 관리. 관리도 등 통계적 기법을 활용하여 공정 변동을 감시/관리하는 기법 |
| Cpk | Process Capability Index, 공정능력지수. 공정이 규격 한계 내에서 제품을 생산하는 능력의 척도 |
| Cp | 공정능력(잠재). 공정 중심과 무관하게 산포만 고려한 공정능력 |
| 파레토 분석 | Pareto Analysis. 불량 유형, 원인 등을 빈도순으로 정렬하여 핵심 소수(Vital Few)를 식별하는 분석 기법 |
| 추세 분석 | Trend Analysis. 시계열 데이터를 분석하여 증가/감소/안정 등의 경향을 파악하는 기법 |
| 관리도 | Control Chart. 공정의 시간적 변동을 관리 한계(UCL/LCL) 내에서 감시하는 차트 |
| UCL/LCL | Upper/Lower Control Limit, 관리 상한/하한 |
| OTD | On-Time Delivery, 정시 납기율 |
| DPPM | Defective Parts Per Million, 백만 개당 불량 수 |
| RFT | Right First Time, 초회 합격률 |

# 6. 모니터링/측정 대상 결정

## 6.1 결정 원칙

AS9100D 9.1.1에 따라 다음 사항을 결정하여 운영한다:

| 결정 사항 | 내용 |
|----------|------|
| **무엇을** 모니터링/측정할 것인가 | KPI 항목 선정 (7장 참조) |
| **어떤 방법**으로 수행할 것인가 | 데이터 수집 방법, 분석 기법 (8장 참조) |
| **언제** 수행할 것인가 | 모니터링/측정 주기 (6.2절 참조) |
| **언제** 결과를 분석/평가할 것인가 | 분석 주기, 보고 주기 (6.2절 참조) |
| **누가** 수행할 것인가 | 담당자/부서 (5장 참조) |

## 6.2 모니터링/측정 주기

| 구분 | 수집 주기 | 분석 주기 | 보고 주기 |
|------|----------|----------|----------|
| 공정 SPC 데이터 | 실시간/일별 | 주간 | 월간 |
| 검사 합격률 | 일별 | 주간 | 월간 |
| 고객 불만/클레임 | 발생 즉시 | 월간 | 분기별 |
| 공급자 성과 | 입고 시 | 분기별 | 반기별 |
| 품질목표 달성도 | 월간 | 분기별 | 경영검토 시 |
| OTD(정시 납기율) | 출하 건별 | 월간 | 월간/경영검토 |
| 내부심사 결과 | 심사 시 | 심사 후 즉시 | 경영검토 시 |

# 7. 프로세스 성과 모니터링 (KPI 체계)

## 7.1 KPI 체계 구조

```
경영 수준 KPI (전사 품질목표)
    ├── 프로세스 수준 KPI (부서별 성과지표)
    │       ├── 품질 KPI
    │       ├── 납기 KPI
    │       ├── 생산 KPI
    │       └── 공급자 KPI
    └── 제품 수준 KPI (제품/공정 적합성)
            ├── 검사 합격률
            ├── 공정능력(Cpk)
            └── 불량률(DPPM)
```

## 7.2 주요 KPI 목록

### 7.2.1 품질 KPI

| KPI 항목 | 산출 공식 | 목표 | 측정 주기 | 담당 |
|----------|---------|------|----------|------|
| 수입검사 합격률 | (합격 LOT 수 / 총 검사 LOT 수) × 100 | ≥ 98% | 월간 | 품질팀 |
| 공정 불량률 | (공정 불량 수 / 총 생산 수) × 1,000,000 (DPPM) | ≤ 500 DPPM | 월간 | 품질팀 |
| 최종검사 합격률 | (합격 LOT 수 / 총 검사 LOT 수) × 100 | ≥ 99% | 월간 | 품질팀 |
| 고객 클레임 건수 | 월간 접수 건수 | ≤ 2건/월 | 월간 | 품질팀 |
| 고객 반품률 | (반품 수량 / 출하 수량) × 1,000,000 (DPPM) | ≤ 100 DPPM | 월간 | 품질팀 |
| 시정조치 완료율 | (기한 내 완료 건수 / 총 발행 건수) × 100 | ≥ 95% | 월간 | 품질팀 |
| 초회 합격률(RFT) | (초회 합격 수 / 총 검사 수) × 100 | ≥ 95% | 월간 | 품질팀 |

### 7.2.2 납기 KPI

| KPI 항목 | 산출 공식 | 목표 | 측정 주기 | 담당 |
|----------|---------|------|----------|------|
| OTD(정시 납기율) | (정시 출하 건수 / 총 출하 건수) × 100 | ≥ 95% | 월간 | 생산/영업 |
| 납기 지연 건수 | 월간 지연 건수 | ≤ 3건/월 | 월간 | 생산팀 |

### 7.2.3 공급자 KPI

| KPI 항목 | 산출 공식 | 목표 | 측정 주기 | 담당 |
|----------|---------|------|----------|------|
| 공급자 품질 합격률 | (합격 LOT / 총 입고 LOT) × 100 | ≥ 98% | 분기별 | 구매/품질 |
| 공급자 납기 준수율 | (정시 입고 건 / 총 발주 건) × 100 | ≥ 95% | 분기별 | 구매팀 |

### 7.2.4 QMS 성과 KPI

| KPI 항목 | 산출 공식 | 목표 | 측정 주기 | 담당 |
|----------|---------|------|----------|------|
| 품질목표 달성률 | (달성 목표 수 / 총 목표 수) × 100 | ≥ 90% | 반기별 | QMR |
| 내부심사 부적합 개선율 | (개선 완료 건 / 부적합 발견 건) × 100 | 100% | 심사 후 | 품질팀 |

# 8. 데이터 분석 방법

## 8.1 통계적 공정 관리 (SPC)

> 기존 SPC 절차(Folder 30)를 AS9100D 요구사항에 맞게 체계화한다.

### 8.1.1 SPC 적용 대상

| 적용 대상 | SPC 유형 | 관리도 종류 |
|----------|---------|------------|
| 치수 특성 (계량형) | 계량형 SPC | X̄-R 관리도, X̄-S 관리도 |
| 외관/기능 특성 (계수형) | 계수형 SPC | p 관리도, np 관리도, c 관리도 |
| 핵심 특성(KC) | 계량형 SPC (필수) | X̄-R 관리도 |
| 안전핵심 특성 | 계량형 SPC (필수) | X̄-R 관리도 + Cpk 관리 |

### 8.1.2 SPC 운영 절차

1. **관리 특성 선정**: 핵심 특성(KC), 안전핵심 특성, 고객 지정 항목 우선
2. **데이터 수집**: 서브그룹 크기(n=3~5), 수집 주기 설정
3. **관리도 작성**: UCL, CL, LCL 계산 및 관리도 작성
4. **공정 관리**: 관리 한계 이탈 시 즉시 조사 및 조치
5. **공정능력 분석**: Cp, Cpk 주기적 산출 및 평가
6. **개선 활동**: 공정능력 미달 시 개선 활동 수행

### 8.1.3 관리도 이상 판정 기준 (Nelson 규칙)

| 규칙 | 이상 패턴 | 조치 |
|------|----------|------|
| Rule 1 | 1점이 관리 한계 밖 | 즉시 조사 및 시정조치 |
| Rule 2 | 연속 9점이 중심선 한쪽 | 공정 이동 조사 |
| Rule 3 | 연속 6점이 증가 또는 감소 추세 | 추세 원인 조사 |
| Rule 4 | 연속 14점이 교대로 상하 반복 | 혼합 원인 조사 |

### 8.1.4 관리도 이탈 시 대응 절차

```
관리도 이상 감지
    ↓
생산 일시 중단 (필요 시)
    ↓
원인 조사 (5 Why, 특성요인도)
    ↓
시정조치 실행
    ↓
효과 확인 (관리도 안정 확인)
    ↓
기록 (이상 원인, 조치 내용, 결과)
```

## 8.2 파레토 분석 (Pareto Analysis)

### 8.2.1 적용 영역

| 분석 대상 | 분류 기준 | 분석 주기 |
|----------|----------|----------|
| 공정 불량 | 불량 유형별, 공정별, 제품별 | 월간 |
| 고객 클레임 | 클레임 유형별, 고객별, 제품별 | 분기별 |
| 공급자 부적합 | 부적합 유형별, 공급자별 | 분기별 |
| 내부심사 부적합 | 조항별, 부서별 | 심사 후 |

### 8.2.2 파레토 분석 절차

1. **데이터 수집**: 분석 대상 기간의 부적합/불량 데이터 수집
2. **분류 정리**: 유형별로 건수/비율 집계
3. **내림차순 정렬**: 빈도 높은 순서로 정렬
4. **누적 비율 산출**: 누적 백분율 계산
5. **파레토 차트 작성**: 막대그래프 + 누적 곡선
6. **핵심 소수(Vital Few) 식별**: 상위 20% 원인이 80% 영향을 차지하는 항목 식별
7. **개선 우선순위 결정**: 핵심 소수에 대한 집중 개선

## 8.3 추세 분석 (Trend Analysis)

### 8.3.1 적용 영역

| 분석 대상 | 분석 방법 | 분석 주기 |
|----------|----------|----------|
| KPI 추이 | 시계열 그래프, 이동 평균 | 월간/분기별 |
| 불량률 추이 | 시계열 그래프 | 월간 |
| 고객 만족도 추이 | 시계열 그래프, 회귀 분석 | 반기별 |
| 공정능력 추이 | Cpk 시계열 그래프 | 월간 |

### 8.3.2 추세 판단 기준

| 추세 유형 | 판단 기준 | 조치 |
|----------|----------|------|
| 개선 추세 | 3개월 연속 목표 대비 개선 | 현 관리 방법 유지, 우수 사례 전파 |
| 악화 추세 | 2개월 연속 목표 미달 또는 전월 대비 악화 | 원인 분석 및 시정조치 개시 |
| 급격한 변동 | 전월 대비 ±20% 이상 변동 | 즉시 원인 조사 |
| 안정 추세 | 6개월 이상 관리 한계 내 유지 | 목표 상향 검토 |

## 8.4 기타 분석 기법

# 10. 항공우주 특수 요구사항

## 10.1 정시 납기율 (OTD: On-Time Delivery)

AS9100D는 정시 납기 성과를 모니터링하고 개선할 것을 명시적으로 요구한다.

| 항목 | 내용 |
|------|------|
| 산출 공식 | OTD(%) = (약속 납기 이내 출하 건수 / 총 출하 건수) × 100 |
| 목표 | ≥ 95% (항공우주 고객 기준) |
| 측정 주기 | 월간 |
| 보고 | 월간 품질 보고서, 경영검토 입력 |
| 미달 시 조치 | 원인 분석 → 시정조치(QP-1001) → 효과 확인 |

### OTD 분석 세부 사항

- 지연 원인을 분류하여 파레토 분석을 수행한다 (자재 지연, 공정 불량, 설비 고장, 물류 등).
- OTD 미달 추세 발생 시, 근본 원인을 분석하고 생산/영업/구매 부서와 합동 개선 활동을 수행한다.

## 10.2 공정능력 관리 (Cpk)

항공우주 산업에서는 공정능력에 대한 엄격한 관리가 요구된다.

| 구분 | Cpk 기준 | 적용 대상 | 미달 시 조치 |
|------|---------|----------|-------------|
| 일반 특성 | Cpk ≥ 1.33 | 일반 치수, 기능 특성 | 공정 개선, 전수검사 전환 |
| **안전핵심 특성** | **Cpk ≥ 1.67** | 안전 관련 치수, 기능 항목 | **즉시 공정 중단 및 원인 분석, 전수검사 필수** |
| 핵심 특성(KC) | Cpk ≥ 1.33 | 고객 지정 핵심 특성 | 고객 통보, 개선 계획 수립 |
| 신규 공정 | Cpk ≥ 1.67 (초기) | 신규 설비/공정 도입 시 | 안정화 확인 후 1.33 전환 가능 |

### 공정능력 관리 절차

```
Cpk 주기적 산출 (월간 이상)
    ↓
기준 대비 평가
    ├── Cpk ≥ 기준 → 현 관리 유지, 추세 모니터링
    └── Cpk < 기준 → 아래 절차 수행
            ↓
        원인 분석 (SPC 관리도, 특성요인도)
            ↓
        개선 활동 수행
            ↓
        개선 후 Cpk 재산출
            ↓
        효과 확인 및 표준화
```

## 10.3 항공우주 데이터 분석 요구사항
