# 1. 목적

본 절차서는 Công ty TNHH M&C ELECTRONICS VIỆT NAM (이하 M&C Electronics VINA)의 고객이 조직의 제품 및 서비스에 대해 인식하는 만족 수준을 체계적으로 모니터링, 측정 및 분석하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 9.1.2 요구사항에 따른 고객만족 모니터링 체계 수립
- 고객의 요구 및 기대 충족 정도에 대한 객관적 데이터 수집 및 분석
- 제품 품질, 납기 성과, 서비스 수준에 대한 정량적 측정 지표 운영
- 고객 피드백을 기반으로 한 지속적 개선 기회 발굴
- 항공우주 산업 고객의 스코어카드(Scorecard) 요구사항 체계적 대응

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 고객만족 (Customer Satisfaction) | 고객의 요구사항이 충족된 정도에 대한 고객의 인식 (ISO 9000:2015, 3.9.2) |
| CSI (Customer Satisfaction Index) | 고객만족지수 — 복수의 측정 지표를 가중 평균하여 산출한 종합 만족도 수치 |
| DPPM (Defective Parts Per Million) | 백만 개당 불량 부품 수 — 제품 품질 수준을 나타내는 정량 지표 |
| OTD (On-Time Delivery) | 정시 납기율 — 약속된 납기일 이내 납품 완료한 비율 (%) |
| 고객 스코어카드 (Customer Scorecard) | 고객이 공급업체 성과를 평가하기 위해 운영하는 정량 평가표 |
| 고객 클레임 (Customer Claim/Complaint) | 고객이 제기하는 품질, 납기, 서비스 관련 공식 불만 사항 |
| VOC (Voice of Customer) | 고객의 소리 — 고객의 요구, 기대, 불만을 포괄하는 피드백 정보 |
| NPS (Net Promoter Score) | 순추천 고객지수 — 고객 충성도를 측정하는 지표 (해당 시 활용) |

# 6. 고객만족 측정 지표

## 6.1 핵심 측정 지표 (Key Metrics)

### 6.1.1 품질 지표

| 지표 | 산출 방법 | 목표 | 측정 주기 |
|------|----------|------|----------|
| DPPM | (고객 반품 불량 수 ÷ 총 납품 수) × 1,000,000 | ≤ 500 DPPM | 월간 |
| 고객 클레임 건수 | 월간 고객 공식 클레임 접수 건수 | ≤ 2건/월 | 월간 |
| 클레임 해결 기간 | 클레임 접수일 ~ 시정조치 완료일 | ≤ 30일 (8D 완료) | 건별 |
| 고객 반품률 | (반품 수량 ÷ 납품 수량) × 100% | ≤ 0.1% | 월간 |

### 6.1.2 납기 지표

| 지표 | 산출 방법 | 목표 | 측정 주기 |
|------|----------|------|----------|
| OTD (정시 납기율) | (정시 납품 건수 ÷ 총 납품 건수) × 100% | ≥ 95% | 월간 |
| 납기 지연 평균일수 | 지연 건의 평균 지연일수 | ≤ 3일 | 월간 |
| 긴급 주문 대응률 | (긴급 주문 정시 납품 ÷ 긴급 주문 총 건수) × 100% | ≥ 90% | 분기 |

### 6.1.3 서비스 지표

| 지표 | 산출 방법 | 목표 | 측정 주기 |
|------|----------|------|----------|
| 고객 문의 응답 시간 | 고객 문의 접수 ~ 초기 응답 소요 시간 | ≤ 24시간 | 건별 |
| 기술 지원 만족도 | 기술 지원 후 고객 평가 (5점 척도) | ≥ 4.0점 | 건별 |
| 견적 응답 시간 | 견적 요청 접수 ~ 견적서 제출 소요 시간 | ≤ 3영업일 | 건별 |

## 6.2 고객만족지수 (CSI) 산출

### 6.2.1 CSI 산출 공식

```
CSI = Σ (각 지표 점수 × 가중치) / Σ 가중치 × 100
```

### 6.2.2 지표별 가중치

| 영역 | 지표 | 가중치 | 비고 |
|------|------|--------|------|
| 품질 | DPPM | 30% | 항공우주 핵심 |
| 품질 | 고객 클레임 건수 | 10% | |
| 납기 | OTD | 25% | 항공우주 핵심 |
| 납기 | 납기 지연 일수 | 10% | |
| 서비스 | 응답 시간 | 10% | |
| 서비스 | 기술 지원 | 5% | |
| 종합 | 고객 설문 평가 | 10% | 연 1회 |

### 6.2.3 CSI 등급 기준

| 등급 | CSI 점수 | 조치 |
|------|---------|------|
| A (우수) | ≥ 90점 | 현행 유지, 모범 사례 공유 |
| B (양호) | 80~89점 | 약점 영역 개선 계획 수립 |
| C (보통) | 70~79점 | 개선 프로젝트 실행, 경영검토 보고 |
| D (미흡) | < 70점 | 긴급 개선 조치, 고객 대응 회의, CAR 발행 |

# 7. 고객만족도 조사

## 7.1 정기 조사

| 항목 | 내용 |
|------|------|
| 주기 | 연 1회 이상 (상반기 또는 하반기) |
| 대상 | 전체 활성 고객 (최근 12개월 내 거래 실적 있는 고객) |
| 방법 | 구조화된 설문지 배포 (이메일/온라인/우편) |
| 회수 목표 | 배포 대상의 60% 이상 회수 |
| 분석 기한 | 설문 마감 후 30일 이내 분석 보고서 작성 |
| 보고 | QMR에게 보고, 경영검토 입력 자료로 활용 |

## 7.2 설문 조사 항목

| 평가 영역 | 세부 항목 | 평가 척도 |
|----------|----------|----------|
| 제품 품질 | 규격 적합성, 외관 품질, 기능 신뢰성, 품질 일관성 | 5점 척도 |
| 납기 성과 | 약속 납기 준수, 납기 변경 시 소통, 긴급 대응 능력 | 5점 척도 |
| 기술 지원 | 기술 문의 대응, 설계 지원 역량, 문제 해결 속도 | 5점 척도 |
| 의사소통 | 응답 신속성, 정보 정확성, 소통 태도 | 5점 척도 |
| 가격 경쟁력 | 가격 대비 품질, 원가 절감 노력 | 5점 척도 |
| 종합 만족도 | 전반적 만족 수준, 재구매/추천 의향 | 5점 척도 |
| 자유 의견 | 개선 제안, 불만 사항, 기대 사항 | 서술형 |

## 7.3 수시 조사

다음 상황에서 수시 고객만족 조사를 실시한다:
- 신규 고객 거래 개시 후 3개월 시점
- 주요 클레임 시정조치 완료 후
- 고객 요청에 의한 수시 평가
- 신제품/신규 서비스 공급 개시 후

# 8. 고객 피드백 수집 및 분석

## 8.1 피드백 수집 채널

| 채널 | 수집 방법 | 담당 |
|------|----------|------|
| 고객 클레임 | 공식 클레임 접수 (이메일, 팩스, 포탈) | 영업팀 → 품질팀 |
| 고객 방문/미팅 | 방문 보고서에 고객 의견 기록 | 영업팀 |
| 고객 심사 | 고객 심사(2nd Party Audit) 결과 | 품질팀 |
| 고객 스코어카드 | 고객이 발행하는 공급업체 성과 평가표 | 영업팀 → 품질팀 |
| 전화/이메일 | 일상 업무 중 접수되는 의견/요구 | 해당 부서 → 영업팀 |
| 전시회/세미나 | 업계 행사에서 수집되는 시장 피드백 | 영업팀 |
| 반품/RMA | 반품 사유 및 고객 요구사항 | 품질팀 |

## 8.2 피드백 분류 및 등록

수집된 고객 피드백은 다음과 같이 분류하여 고객 피드백 대장에 등록한다:

| 분류 | 내용 | 대응 수준 |
|------|------|----------|
| 클레임 (Complaint) | 품질/납기/서비스에 대한 공식 불만 | 즉시 대응 (QP-1001 연계) |
| 불만 (Dissatisfaction) | 공식 클레임 이전의 불만 표현 | 48시간 이내 초기 대응 |
| 요구 (Request) | 개선, 변경, 추가 서비스 요구 | 검토 후 대응 계획 수립 |
| 칭찬 (Compliment) | 긍정적 피드백, 만족 표현 | 기록 및 해당 부서 공유 |
| 제안 (Suggestion) | 개선 아이디어, 협력 제안 | 검토 후 반영 여부 결정 |

## 8.3 데이터 분석

### 8.3.1 분석 주기 및 방법

| 분석 유형 | 주기 | 방법 | 출력 |
|----------|------|------|------|
| 월간 성과 분석 | 매월 | DPPM, OTD 추이 분석, 클레임 파레토 분석 | 월간 품질 보고서 |
| 분기 추세 분석 | 분기 | CSI 추세, 고객별/제품별 분석 | 분기 고객만족 보고서 |
| 연간 종합 분석 | 연 1회 | 설문 결과 + 정량 지표 종합 평가 | 연간 고객만족 보고서 |
| 비교 분석 | 연 1회 | 전년 대비, 고객별 비교, 목표 대비 | 경영검토 입력 자료 |

### 8.3.2 분석 도구

- 추세도 (Trend Chart): DPPM, OTD 월별/분기별 추이
- 파레토 분석 (Pareto): 클레임 유형별, 제품별, 고객별 불량 분석
- 레이더 차트 (Radar): 평가 영역별 성과 시각화
- 비교 분석표: 고객별, 기간별, 제품군별 CSI 비교

# 9. 항공우주 특수 요구사항

## 9.1 제품 적합성 모니터링

AS9100D 9.1.2는 단순한 고객 만족도 설문을 넘어 **제품/서비스 적합성**에 대한 고객 인식을 모니터링할 것을 요구한다.

| 모니터링 항목 | 측정 방법 | 기준 |
|-------------|----------|------|
| 제품 규격 적합률 | 1 - (부적합 건수 ÷ 납품 총 건수) | ≥ 99.5% |
| 현장 불량률 (Field Failure Rate) | 고객 현장 불량 보고 건수/납품 수 | ≤ 100 DPPM |
| 규격 이탈 통보 건수 | Deviation/Concession 발행 건수 | 감소 추세 |
| 인증서/성적서 적합률 | 서류 부적합(문서 오류) 건수 | Zero Defect |

## 9.2 납기 성과 보고

항공우주 고객에 대한 납기 성과는 별도로 관리하며 고객 요구 시 보고한다:

- 고객별 OTD 성과를 월별로 산출하여 관리
- 고객의 납기 평가 기준(고객 도착일 기준/출하일 기준)에 맞춰 산출
- 납기 지연 발생 시 원인 분석 및 재발 방지 대책 수립
- 납기 성과가 고객 목표 미달 시 개선 계획(Recovery Plan) 제출

## 9.3 고객 스코어카드 대응

### 9.3.1 스코어카드 관리 프로세스

1. **접수**: 영업팀이 고객 스코어카드를 접수하여 품질팀에 전달
2. **분석**: 품질팀이 스코어카드 평가 항목별 성과 분석
3. **검증**: 평가 결과와 내부 데이터 비교 검증 (불일치 시 고객과 협의)
4. **개선 계획**: 미달 항목에 대한 개선 계획 수립
5. **보고**: 경영진 보고 및 경영검토 입력
6. **고객 회신**: 개선 계획을 고객에게 회신 (고객 요구 시)

### 9.3.2 스코어카드 평가 등급 대응

| 고객 평가 등급 | 해당 조건 (일반적 기준) | 대응 |
|-------------|---------------------|------|
| Preferred / Gold | 최우수 성과 | 현행 유지, 모범 사례 내부 공유 |
| Approved / Acceptable | 양호한 성과 | 약점 영역 지속 개선 |
| Conditional / Warning | 성과 미달 발생 | 개선 계획(Improvement Plan) 제출, 정기 진척 보고 |
| Disqualified / Critical | 심각한 성과 미달 | 긴급 시정조치, 경영진 직접 대응, 특별 관리 |

## 9.4 고객 소통 연계

- 고객만족 데이터는 QP-805(고객 의사소통/요구사항 관리)와 연계하여 고객 소통 시 활용
- 주요 고객과의 정기 품질 회의(Quarterly Business Review 등)에서 만족도 데이터 공유
- 고객 불만이 반복되는 영역에 대한 선제적 소통 및 개선 보고

# 10. 개선 조치 연계

## 10.1 시정조치 연계 (QP-1001)

| 업계 벤치마킹 대비 열위 | 격차 분석 후 개선 계획 수립 |

# 11. 경영검토 입력

## 11.1 보고 내용 (QP-904 연계)

경영검토 시 다음 고객만족 데이터를 입력 자료로 제출한다:

| 보고 항목 | 내용 | 주기 |
|----------|------|------|
| CSI 종합 점수 및 추세 | 고객별/기간별 CSI 점수, 전년 대비 | 반기/연간 |
| DPPM 추이 | 고객별 DPPM 월별 추세, 목표 대비 | 월간 집계 |
| OTD 추이 | 고객별 OTD 월별 추세, 목표 대비 | 월간 집계 |
| 고객 클레임 현황 | 유형별 건수, 처리 현황, Top 불량 | 월간 집계 |
| 고객 스코어카드 결과 | 고객별 평가 등급, 미달 항목, 개선 현황 | 수시/반기 |
| 고객 설문 결과 | 영역별 평균 점수, 주요 의견 요약 | 연간 |
| 개선 조치 추적 | 개선 과제 진척률, 완료 건수, 효과 | 반기/연간 |

## 11.2 경영검토 출력 반영

경영검토에서 결정된 고객만족 관련 조치 사항:
- 고객만족 목표 재설정 (CSI, DPPM, OTD 목표)
- 자원 배분 (인력, 설비, 예산) 의사결정
- 신규 개선 프로젝트 승인
- 고객 대응 전략 방향 설정
