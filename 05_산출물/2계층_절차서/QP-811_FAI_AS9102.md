# 1. 목적

본 절차서는 M&C Electronics VINA의 제조공정에서 생산된 초도품(First Article)이 설계 도면, 규격 및 고객 요구사항에 적합함을 객관적으로 검증하기 위한 절차를 규정한다.

본 절차서의 목적:

- **AS9100D 8.5.1.2** 요구사항 및 **AS9102B** 규격에 적합한 FAI 체계 수립
- FAI 적용 대상 기준, 수행 방법, 기록 요건의 명확한 정의
- FAI 기록 패키지(FAIR: First Article Inspection Report) 작성 및 관리 체계 확립
- 설계 특성(Design Characteristics)의 100% 검증 보장
- 원자재 인증, 특수공정 인증, 기능 시험 결과의 체계적 검증

> **시정조치 연계:**
> - **CAR-006** (Major): AS9102 기준 FAI 절차 없음 → 본 절차서 제정으로 시정
> - **CAR-007** (Major): 기존 양산 품목 FAI 기록 패키지 누락 → 11절 소급 적용 계획으로 시정

# 4. 용어 및 정의

| 용어 | 영문 | 정의 |
|------|------|------|
| 초도품 검사 | First Article Inspection (FAI) | 대표 품목에 대해 설계 기록(Design Record)에 명시된 모든 설계 특성을 독립적으로 검증하는 계획된 완전 검사 |
| 초도품 검사 보고서 | First Article Inspection Report (FAIR) | FAI 결과를 기록한 공식 문서 패키지. AS9102 Form 1, 2, 3으로 구성 |
| 부분 FAI | Partial FAI | 이전 FAI가 유효한 상태에서 변경된 특성만 검증하는 축소된 FAI |
| 특성 책임 | Characteristic Accountability | 설계 기록에 정의된 모든 특성(치수, 규격, 참조사항 포함)에 대한 검증 추적성 |
| 설계 기록 | Design Record | 품목을 정의하는 도면, 규격, CAD 모델 등 기술 문서 일체 |
| 설계 특성 | Design Characteristic | 설계 기록에 명시된 치수, 공차, 재질, 성능 등 모든 검증 대상 특성 |
| 주요 특성 | Key Characteristic (KC) | 변동이 제품의 형상/기능/안전에 중대한 영향을 미치는 특성 |
| 특수 공정 | Special Process | 후속 모니터링/검사만으로 결과를 완전히 검증할 수 없는 공정 (열처리, 도금, 용접, NDT 등) |
| 풍선도면 | Ballooned Drawing | 설계 도면에 각 특성을 일련번호(Balloon Number)로 식별한 도면 |
| 설계 기준선 | Design Baseline | 형상관리에 의해 공식 승인된 설계 상태 |
| 초도품 | First Article | FAI를 위해 생산된 대표 품목. 양산과 동일한 공정/설비/재료로 제조 |
| PPAP | Production Part Approval Process | 생산 부품 승인 프로세스 (고객 요구 시 FAI와 병행) |

# 6. FAI 적용 기준

## 6.1 Full FAI 적용 사유 (AS9102B §4.2)

다음 중 하나 이상에 해당하는 경우 **Full FAI**를 수행한다:

| No. | 적용 사유 | 상세 기준 | 예시 |
|-----|----------|----------|------|
| 1 | **신규 제품** | 최초 양산 전환되는 신규 품번(Part Number) | 신규 수주 품목 |
| 2 | **설계 변경** | 형상/기능/재질에 영향을 미치는 설계 변경(ECN/ECO) | 도면 개정, 재질 변경 |
| 3 | **공정 변경** | 제조 방법, 공정 순서, 검사 방법의 중대 변경 | 가공 방법 변경, 금형 신규 제작 |
| 4 | **2년 이상 미생산** | 최종 생산일로부터 24개월 이상 경과한 품목의 재생산 | 장기 미생산 후 재가동 |
| 5 | **제조 장소 변경** | 생산 설비 이전 또는 제조 사업장 변경 | 라인 이전, 공장 이전 |
| 6 | **공급자 변경** | 원자재 또는 외주 가공 공급자의 변경 | 새 원자재 공급사, 외주처 변경 |
| 7 | **금형/치구 변경** | 주요 금형, 다이, 치구의 교체 또는 대수리 | 금형 신규 제작, 치구 재설계 |
| 8 | **이전 FAI 부적합** | 이전 FAI에서 부적합이 발생하여 시정 후 재검증 필요 | 재작업 후 재FAI |
| 9 | **고객 요구** | 고객이 서면으로 FAI 수행을 요구한 경우 | PO/Contract에 FAI 명시 |

## 6.2 FAI 면제 기준

다음 조건을 **모두** 충족하는 경우에 한해 FAI를 면제할 수 있다:

- 고객이 서면으로 FAI 면제를 승인
- 품질책임자(QMR)가 면제 타당성을 검토/승인
- 면제 사유 및 근거를 기록으로 유지 (QF-811-08 FAI 면제 기록서)

> **주의:** FAI 면제 여부와 관계없이, 초기 양산 제품의 치수/기능 검증은 최종검사(QP-817)에 따라 별도 수행한다.

# 7. FAI 계획 수립

## 7.1 FAI 계획서 작성

FAI 대상 품목이 확인되면, 품질팀장은 **FAI 계획서(QF-811-01)**를 작성한다.

| 계획 항목 | 내용 |
|----------|------|
| 대상 품번 (Part Number) | 품번, 품명, 도면번호, 도면 개정 수준 |
| FAI 유형 | Full FAI / Partial FAI |
| FAI 사유 | 6절의 적용 기준 중 해당 사유 |
| 풍선도면 준비 | 기술팀 작성, 모든 설계 특성에 번호 부여 |
| 검사 계획 | 측정 항목, 측정 방법, 사용 장비, 검사 기준 |
| 원자재 확인 | 재질 인증서(Mill Cert.) 수집 계획 |
| 특수공정 확인 | 특수공정 인증서/결과 수집 계획 |
| 기능 시험 | 기능/성능 시험 항목 및 방법 |
| 일정 | FAI 생산일, 검사일, 보고서 완료 목표일 |
| 담당자 | FAI 수행 검사원, 검토자, 승인자 |
| 고객 제출 요건 | 고객 FAIR 제출 필요 여부 및 양식 |

## 7.2 풍선도면(Ballooned Drawing) 작성

기술팀장은 FAI 대상 품목의 풍선도면을 작성한다:

1. 설계 도면의 **모든 치수, 공차, 주기(Note), 규격 참조**에 일련번호(Balloon Number)를 부여
2. 번호 부여 원칙:
   - 좌→우, 상→하 순으로 일련번호 부여
   - 복수 도면 시 시트별 구분 (예: Sheet 1 → 1.1, 1.2 ... / Sheet 2 → 2.1, 2.2 ...)
   - 참조 규격 내 세부 요구사항도 별도 번호 부여
3. **100% Characteristic Accountability** 보장: 누락 특성 없이 전수 번호 부여
4. 풍선도면은 기술팀장 검토, 품질팀장 확인 후 확정

## 7.3 검사 방법 결정

| 특성 유형 | 검사 방법 | 장비 예시 |
|----------|----------|----------|
| 일반 치수 | 직접 측정 | 버니어, 마이크로미터, 하이트게이지 |
| 정밀 치수 | 3차원 측정 | CMM (3차원 측정기) |
| 형상 공차 (GD&T) | 3차원 측정 / 전용 게이지 | CMM, 프로파일 프로젝터 |
| 표면 거칠기 | 표면 조도 측정 | 표면 조도계 |
| 재질 | Mill Certificate 검증 | 성적서 대조 |
| 경도 | 경도 시험 | 경도계 (Rockwell/Vickers) |
| 기능/성능 | 기능 시험 | 전용 시험 장비 |
| 외관 | 육안 검사 | 확대경, 조명 설비 |

> **참고:** 모든 측정 장비는 유효한 교정 상태여야 하며, QP-702(측정장비/계측기 관리)에 따라 관리한다.

## 8.2 초도품 생산

생산팀장은 FAI용 초도품을 다음 조건하에 생산한다:

| 조건 | 요구사항 |
|------|---------|
| 양산 동일 조건 | 양산 시 사용할 설비, 금형, 치구, 재료와 **동일한 조건**으로 생산 |
| 공정 순서 | 양산 공정 라우팅(Routing)과 동일 |
| 작업자 | 양산 시 배치 예정인 작업자 또는 동등 자격 보유자 |
| 원자재 | 양산용 원자재와 동일 로트 또는 동일 규격 인증 자재 |
| 수량 | 최소 1개 (고객 요구 시 복수 개) |
| 기록 | 생산 일시, 설비 번호, 로트 번호, 작업자 등 추적성 기록 유지 |

## 8.3 AS9102 Form 1: Part Number Accountability

### 8.3.1 목적
Form 1은 FAI 대상 품번의 전체적인 파트 정보와 하위 부품(Sub-Assembly) 구성을 기록하여, 각 하위 품목의 FAI 상태를 추적한다.

### 8.3.2 작성 내용

| 항목 | 내용 | 담당 |
|------|------|------|
| Part Number | FAI 대상 품번 | 기술팀 |
| Part Name | 품명 | 기술팀 |
| Part Revision Level | 도면/규격 개정 수준 | 기술팀 |
| Drawing Number | 도면 번호 | 기술팀 |
| Additional Changes | 도면 외 추가 변경사항 (ECN 등) | 기술팀 |
| Manufacturing Process Reference | 공정 흐름도 / 라우팅 참조 | 생산팀 |
| Organization Name | (주)에이로텍 / M&C Electronics VINA | - |
| Supplier Code | 당사 공급자 코드 (고객 부여) | 영업팀 |
| P.O. Number | 관련 Purchase Order 번호 | 영업팀 |
| Detail Part Accountability | 하위 부품 목록 및 각각의 FAI 상태 | 품질팀 |
| FAI Type | Full / Partial | 품질팀 |
| Reason for Partial FAI | Partial 사유 (해당 시) | 품질팀 |

### 8.3.3 작성 절차

1. 기술팀이 BOM(Bill of Material) 기반으로 하위 부품 목록 제공
2. 품질팀이 각 하위 부품의 FAI 상태 확인 (완료/미완료/해당없음)
3. 모든 하위 부품의 FAI가 완료되어야 상위 어셈블리 FAI 완료 가능
4. 기술팀장 검토, 품질팀장 확인

## 8.4 AS9102 Form 2: Product Accountability — Raw Material, Special Process, Functional Test

### 8.4.1 목적
Form 2는 원자재 인증, 특수공정 인증, 기능 시험 결과를 종합적으로 기록한다.

### 8.4.2 원자재 검증 (Raw Material)

| 확인 항목 | 내용 | 증빙 문서 |
|----------|------|----------|
| 재질 규격 | 도면 지정 재질과 실제 사용 재질의 일치 여부 | Mill Certificate (재질 인증서) |
| 화학 성분 | 규격 요구 화학 성분 범위 충족 여부 | Mill Cert. 화학 분석 성적서 |
| 기계적 성질 | 인장강도, 항복강도, 연신율 등 | Mill Cert. 기계적 시험 성적서 |
| 열 번호 (Heat No.) | 원자재 추적성 확보 | Mill Cert. 열 번호 |
| 적합성 선언 | 공급자의 규격 적합 선언 | CoC (Certificate of Conformance) |

### 8.4.3 특수공정 검증 (Special Process)

| 확인 항목 | 내용 | 증빙 문서 |
|----------|------|----------|
| 공정 종류 | 열처리, 도금, 용접, NDT, 납땜 등 | 공정 결과 보고서 |
| 공정 규격 | 적용 공정 규격 번호 및 개정 수준 | 공정 규격서 |
| 공정 수행자 | 자격 인증된 수행자/업체 여부 | 자격 인증서, NADCAP 인증서 |
| 공정 결과 | 공정 파라미터 기록, 검사 결과 | 시험 성적서, 검사 기록 |
| 공급자 승인 | ASL(승인 공급업체 목록) 등재 여부 | ASL 확인 |

### 8.4.4 기능 시험 검증 (Functional Test)

| 확인 항목 | 내용 | 증빙 문서 |
|----------|------|----------|
| 시험 항목 | 도면/규격 요구 기능/성능 시험 목록 | 시험 절차서 |
| 시험 방법 | 적용 시험 규격 및 방법 | 시험 규격서 |
| 시험 조건 | 온도, 습도, 하중 등 시험 환경 | 시험 기록 |
| 시험 결과 | 합격 기준 대비 실측값 | 시험 성적서 |
| 시험 장비 | 사용 장비 및 교정 상태 | 교정 성적서 |

### 8.4.5 Form 2 작성 절차

1. 구매팀이 원자재 Mill Certificate 및 CoC 수집
2. 품질팀이 Mill Cert.의 재질 규격/성분/기계적 성질 적합성 검증
3. 품질팀이 특수공정 인증서/결과 보고서 수집 및 검증
4. 검사팀이 기능 시험 수행 및 결과 기록
5. 검사팀장 검토, 품질팀장 확인
6. 모든 항목에 대해 적합(Accept) / 부적합(Reject) 판정

## 8.5 AS9102 Form 3: Characteristic Accountability

### 8.5.1 목적
Form 3은 풍선도면의 **모든 설계 특성(100%)**에 대한 실측 데이터를 기록하고, 규격 적합 여부를 개별 판정한다.

### 8.5.2 작성 내용

| 항목 | 내용 |
|------|------|
| Characteristic Number | 풍선도면 번호 (일련번호) |
| Reference Location | 도면상 특성 위치 (시트, 존, 주기 번호 등) |
| Characteristic Designator | 특성 구분 (치수 / 참조 / 주기 / 규격 등) |
| Requirement | 규격값 (명목값 ± 공차) |
| Results | 실측값 |
| Designed Tooling (Y/N) | 전용 치공구 사용 여부 |
| Non-Conformance Number | 부적합 발생 시 부적합 보고서 번호 |

### 8.5.3 작성 절차

1. 검사원이 풍선도면의 모든 번호에 대해 순서대로 측정 수행
2. 각 특성별 실측값을 Form 3에 기록
3. 규격값 대비 합격/부적합 판정
4. 부적합 특성 발생 시:
   - 부적합 보고서(NCR) 번호를 Form 3에 기록
   - QP-818(부적합 출력 관리) 절차에 따라 처리
5. **참조 치수(Reference Dimension)**는 "참조"로 표기하되, 실측값은 기록
6. **주기/규격 참조(Note/Spec Reference)**는 적합성 확인 결과를 기록
7. 검사팀장 검토, 품질팀장 확인

### 8.5.4 측정 데이터 요구사항

- 모든 측정값은 **실측 수치**로 기록 (합격/불합격만 표기 불가)
- 측정 단위는 도면 단위와 동일 (mm, inch 등)
- CMM 측정 시 CMM 출력 보고서를 첨부
- GD&T 항목은 각 공차 요소별 실측값 개별 기록
- 측정 불확도(Measurement Uncertainty)가 공차의 25%를 초과하는 경우 별도 기록

# 9. FAI 결과 판정 및 승인

## 9.1 판정 기준

| 판정 | 기준 | 후속 조치 |
|------|------|----------|
| **합격 (Accept)** | Form 1, 2, 3의 모든 항목이 규격에 적합 | FAIR 패키지 완성 → 승인 → 양산 전환 가능 |
| **조건부 승인 (Conditional)** | 일부 부적합 항목에 대해 고객 특채(Concession) 또는 MRB(Material Review Board) 승인 획득 | 특채 문서 첨부, FAIR에 조건 명기 |
| **부적합 (Reject)** | 규격 불일치 항목 존재, 특채 미획득 | 시정조치 → 재FAI 수행 |

## 9.2 승인 절차

| 단계 | 수행자 | 활동 |
|------|--------|------|
| 1. 검사 완료 | 검사원 | Form 1/2/3 작성 완료, 모든 데이터 입력 |
| 2. 1차 검토 | 검사팀장 | 측정 데이터 정확성, 풍선도면 대조, 누락 항목 확인 |
| 3. 2차 검토 | 품질팀장 | FAIR 전체 완전성 검토, Form 간 정합성 확인 |
| 4. 기술 확인 | 기술팀장 | 설계 의도 반영 적합성, 특수 특성 검증 적절성 확인 |
| 5. 최종 승인 | 품질책임자(QMR) | FAIR 최종 승인, 양산 전환 승인 |
| 6. 고객 제출 | 영업팀 | 고객 제출 요건 시 FAIR 패키지 전송 |

## 9.3 부적합 발생 시 처리

1. 부적합 항목에 대해 **부적합 보고서(NCR)** 발행 (QP-818 절차)
2. 근본 원인 분석 수행 (5-Why, Fishbone 등)
3. 시정조치 이행
4. 시정조치 완료 후 **해당 부적합 특성에 대해 재검증** 수행
5. 재검증 결과를 FAIR에 반영 (재FAI 또는 Partial FAI)
6. 모든 부적합이 해소된 후 최종 승인

## 9.4 특채(Concession) 처리

고객 특채가 필요한 경우:

1. 부적합 내용 및 사용 적합성 분석 자료 준비
2. 고객에게 특채 요청서 제출 (영업팀 경유)
3. 고객 서면 승인 획득
4. 특채 승인 문서를 FAIR 패키지에 첨부
5. Form 3 해당 항목에 특채 번호 기록

# 10. Partial FAI (부분 FAI) 수행 기준

## 10.1 Partial FAI 적용 조건 (AS9102B §4.6)

이전 Full FAI가 **합격 상태로 유효**하고, 다음 조건에 해당하는 경우 Partial FAI를 수행할 수 있다:

| No. | 적용 조건 | Partial FAI 범위 |
|-----|----------|-----------------|
| 1 | 설계 변경이 일부 특성에만 영향 | 변경된 특성 + 변경에 의해 영향받는 관련 특성 |
| 2 | 공정 변경이 일부 특성에만 영향 | 변경된 공정으로 생산되는 특성 |
| 3 | 금형/치구 일부 수정 | 수정된 금형/치구로 가공되는 특성 |
| 4 | 원자재 규격 변경 없이 공급자만 변경 | 원자재 인증 검증 (Form 2) + 관련 특성 |
| 5 | 고객이 Partial FAI를 서면 승인 | 고객 지정 범위 |

## 10.2 Partial FAI 불가 조건

다음의 경우 Partial FAI는 **불가**하며, Full FAI를 수행해야 한다:

- 이전 FAI 기록이 존재하지 않거나 무효한 경우
- 2년 이상 미생산 후 재생산
- 제조 장소(사업장) 변경
- 제조 공정의 근본적 변경 (가공 방법 자체가 변경)
- 고객이 Full FAI를 요구하는 경우

## 10.3 Partial FAI 수행 절차

1. 기술팀장이 변경 영향 분석(Impact Analysis) 수행
2. 영향받는 특성 목록 작성 (풍선도면 번호 기준)
3. 품질팀장이 Partial FAI 범위 결정 및 승인
4. Form 1에 Partial FAI 사유 및 이전 Full FAI 참조 번호 기록
5. Form 2는 변경된 원자재/특수공정만 기록 (미변경 항목은 이전 FAI 참조)
6. Form 3은 영향받는 특성만 재측정 및 기록 (미변경 특성은 이전 FAI 참조)
7. 이전 FAI 참조 시 "이전 FAIR 번호 및 일자"를 명기

# 11. 기존 양산품 소급 적용 계획

> **CAR-007 시정조치:** 기존 양산 품목에 대한 FAI 기록 패키지가 누락된 상태이므로, 단계적 소급 적용을 통해 기존 품목의 FAI 기록을 확보한다.

## 11.1 소급 적용 원칙

1. **리스크 기반 우선순위화**: 제품 안전성, 고객 중요도, 불량 이력을 기준으로 우선순위 결정
2. **고객 합의**: 소급 적용 계획을 고객에게 사전 통보하고 합의
3. **단계적 수행**: 한꺼번에 전수 수행이 아닌 우선순위별 단계적 수행
4. **현행 양산 활용**: 별도 초도품 생산이 아닌 현행 양산 제품을 활용한 FAI 가능

## 11.2 우선순위 분류

| 우선순위 | 기준 | 목표 완료일 |
|---------|------|------------|
| **1순위 (긴급)** | 안전 관련 부품, 고객 요구 품목, 최근 불량 발생 품목 | 2026-06-30 |
| **2순위 (중요)** | 주요 고객 납품 품목, 고부가가치 품목 | 2026-07-31 |
| **3순위 (일반)** | 기타 양산 품목 | 2026-08-31 (Stage 2 전) |

## 11.3 소급 적용 절차

1. 품질팀이 **기존 양산 품목 전수 목록** 작성
2. 우선순위 분류 기준에 따라 소급 대상 우선순위 배정
3. 품질책임자(QMR) 승인 후 소급 적용 일정표 확정
4. 현행 양산 로트에서 대표 샘플 선정하여 FAI 수행
5. 풍선도면 작성, Form 1/2/3 작성 (Full FAI 절차 동일)
6. 원자재 인증서는 현행 양산 로트의 Mill Cert. 활용
7. 완료된 FAIR 패키지를 품번별 기록으로 보관
8. 진행 현황을 월간 품질 회의 및 경영검토에 보고

## 11.4 소급 적용 특례

- 설계 기록이 불완전한 경우: 기술팀이 도면 정비 후 FAI 수행
- 원자재 Mill Cert.가 없는 경우: 현행 로트에서 재질 시험 실시
- 특수공정 기록이 없는 경우: 현행 양산 기준으로 공정 인증 재확인

# 12. FAI 기록 패키지 관리

## 12.1 FAIR 패키지 구성

완성된 FAIR 패키지는 다음 문서로 구성된다:

| No. | 문서 | 내용 |
|-----|------|------|
| 1 | **Form 1** (QF-811-02) | Part Number Accountability |
| 2 | **Form 2** (QF-811-03) | Product Accountability (Raw Material / Special Process / Functional Test) |
| 3 | **Form 3** (QF-811-04) | Characteristic Accountability (전체 실측 데이터) |
| 4 | **풍선도면** | 100% 특성 번호가 부여된 도면 |
| 5 | **원자재 인증서** | Mill Certificate, CoC |
| 6 | **특수공정 인증서** | 공정 결과 보고서, NADCAP 인증서 (해당 시) |
| 7 | **기능 시험 성적서** | 기능/성능 시험 결과 |
| 8 | **CMM 보고서** | 3차원 측정 결과 (해당 시) |
| 9 | **특채 문서** | 고객 특채 승인서 (해당 시) |
| 10 | **부적합 보고서** | NCR 및 시정조치 기록 (해당 시) |
| 11 | **FAI 계획서** (QF-811-01) | FAI 계획 및 일정 |

## 12.2 문서 번호 체계

FAIR 패키지 번호 체계:

```
FAIR-[품번]-[개정]-[일련번호]
예: FAIR-MNC-A1001-Rev.A-001
     │     │       │     │
     │     │       │     └─ 일련번호 (동일 품번 FAI 횟수)
     │     │       └─────── 도면 개정 수준
     │     └─────────────── 품번 (Part Number)
     └───────────────────── FAIR 접두사
```

## 12.3 보관 및 관리

| 항목 | 요구사항 |
|------|---------|
| 보관 형태 | 전자 문서 (PDF) + 원본 (해당 시 종이) |
| 보관 위치 | DMS(문서관리시스템) 내 FAI 전용 폴더 |
| 보관 기간 | 제품 수명 + 3년 또는 고객 요구 기간 중 긴 기간 |
| 접근 권한 | 품질팀, 기술팀, 검사팀 (읽기); 품질팀장 이상 (수정) |
| 백업 | 서버 자동 백업 (일일) + 외부 백업 (월간) |
| 고객 제출 | 고객 요구 시 FAIR 패키지 전체 또는 요약본 제출 |

# 13. 항공우주 특수 요구사항

## 13.1 고객 승인 요구사항

| 요구사항 | 세부 내용 |
|---------|----------|
| 고객 FAIR 제출 | 고객 PO/Contract에 FAI 제출이 명시된 경우, 승인 전 고객에게 FAIR 제출 |
| 고객 양식 | 고객이 별도 양식을 요구하는 경우, 고객 양식 + AS9102 양식 병행 |
| 고객 입회 검사 | 고객이 FAI 입회를 요구하는 경우, 일정 조율 후 입회 지원 |
| 고객 승인 기록 | 고객 승인/반려 이력을 FAIR 패키지에 포함 보관 |
| PPAP 연계 | 고객이 PPAP를 요구하는 경우, FAI를 PPAP Level 3~5에 포함하여 제출 |

## 13.2 특수 특성 (Key Characteristics) 검증

| 요구사항 | 세부 내용 |
|---------|----------|
| 식별 | 도면상 특수 특성(KC, SC, CC 등)을 Form 3에 별도 표기 |
| 측정 강화 | 특수 특성은 측정 불확도를 고려한 정밀 측정 수행 |
| 공정능력 | 고객 요구 시 특수 특성에 대한 공정능력(Cpk) 데이터 추가 제공 |
| 관리 계획 | 양산 시 특수 특성에 대한 지속적 관리 방안을 관리 계획서에 반영 |

## 13.3 하위 공급자 FAI 관리

| 요구사항 | 세부 내용 |
|---------|----------|
| 공급자 FAI 요구 | 외주 가공 품목에 대해 공급자에게 FAI 수행 및 FAIR 제출 요구 |
| 공급자 FAIR 검토 | 접수된 공급자 FAIR의 완전성 및 적합성 검토 |
| 공급자 FAIR 보관 | 공급자 FAIR을 당사 FAIR 패키지에 포함 보관 |
| Flow-down | FAI 요구사항을 PO/계약서에 명시하여 공급자에게 전달 (Flow-down) |
| NADCAP 확인 | 특수공정 공급자의 NADCAP 인증 유효성 확인 |

## 13.4 형상관리 연계

- FAI 시점의 도면 개정 수준을 FAIR에 명확히 기록
- FAI 완료 후 설계 변경(ECN) 발생 시 Partial FAI 또는 Full FAI 재수행 여부 판단 (QP-803 절차)
- 형상 기준선(Configuration Baseline)과 FAI 기록의 정합성 유지

## 13.5 FOD 관리 연계

- FAI 검사 중 FOD(이물질) 발견 시 즉시 기록 및 보고 (QP-819 절차)
- FAI용 초도품의 청정도 확인 후 검사 수행
