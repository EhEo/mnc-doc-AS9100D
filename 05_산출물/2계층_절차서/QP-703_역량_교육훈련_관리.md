# 1. 목적

본 절차서는 M&C Electronics VINA의 품질경영시스템(QMS) 운용에 필요한 인원의 역량을 결정하고, 필요한 교육훈련을 체계적으로 관리하기 위한 절차를 규정한다.

본 절차서의 목적:
- AS9100D 7.2 역량 요구사항에 따른 인원의 적격성 보장
- AS9100D 7.3 인식 요구사항에 따른 품질 인식 제고
- 항공우주 산업 특수공정 인원의 자격 인증 및 유지 관리
- CAR-013 시정조치: 특수공정 자격 인증 기록 체계 수립

---

본 절차서는 M&C Electronics VINA의 QMS 활동에 영향을 미치는 모든 인원에 적용한다.

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 역량 (Competence) | 의도된 결과를 달성하기 위해 지식과 기술을 적용하는 능력 |
| 적격성 (Qualification) | 규정된 요구사항을 충족함을 입증한 상태 |
| 특수공정 (Special Process) | 결과를 후속 모니터링이나 측정으로 검증할 수 없는 공정 (용접, 열처리, 도금, NDT 등) |
| 자격 인증 (Certification) | 인원이 특정 업무를 수행할 자격이 있음을 공식 확인하는 문서/절차 |
| OJT (On-the-Job Training) | 실제 작업 현장에서 수행하는 교육훈련 |
| 역량 매트릭스 (Competence Matrix) | 직무별 필요 역량과 인원별 보유 역량을 대조한 표 |
| 교육훈련 니즈 분석 (TNA) | Training Needs Analysis — 교육 필요성을 체계적으로 파악하는 과정 |

# 6. 역량 관리

## 6.1 역량 결정 프로세스

```
① 직무 분석
   │  각 직무별 필요 역량 파악
   ▼
② 역량 요건 정의
   │  직무기술서(JD)에 학력/경력/자격/기술 명시
   ▼
③ 현 역량 평가
   │  역량 매트릭스를 통한 GAP 분석
   ▼
④ GAP 해소 계획
   │  교육훈련/OJT/외부교육/자격취득 계획 수립
   ▼
⑤ 실행 및 평가
   │  교육 실시 → 효과성 평가
   ▼
⑥ 기록 유지
      교육이력카드/자격인증서 관리
```

## 6.2 역량 매트릭스

각 부서별로 다음을 포함하는 역량 매트릭스를 수립하고 유지한다:

| 항목 | 내용 |
|------|------|
| 직무명 | 해당 직무의 공식 명칭 |
| 필요 역량 | 학력, 경력, 자격증, 기술 능력, 교육 이수 |
| 역량 수준 | Level 1(초급) ~ Level 4(전문가) |
| 현재 보유 역량 | 인원별 평가 결과 |
| GAP | 필요 역량 대비 미충족 영역 |
| 해소 계획 | 교육/OJT/자격취득 등 |

### 역량 수준 정의

| Level | 정의 | 업무 수행 조건 |
|-------|------|-------------|
| Level 1 | 교육 수료, 기본 지식 보유 | 숙련자 감독 하 업무 수행 |
| Level 2 | 독립 작업 가능 | 단독 업무 수행 가능 |
| Level 3 | 타인 지도 가능 | 업무 수행 + OJT 지도 |
| Level 4 | 전문가 수준 | 절차 개발, 문제 해결, 심사 수행 |

## 6.3 신규 채용 시 역량 확인

| 단계 | 활동 | 담당 |
|------|------|------|
| 채용 전 | 직무기술서 기반 자격 요건 확인 | 인사팀 |
| 채용 시 | 학력/경력/자격증 검증, 면접 평가 | 인사팀 + 해당 부서장 |
| 채용 후 | 입사 교육 (QMS 인식, 안전, 품질방침) | 인사팀 + 품질팀 |
| 수습 기간 | OJT 실시, 역량 평가 | 해당 부서장 |
| 수습 종료 | 역량 매트릭스 갱신, 독립 작업 승인 | 해당 부서장 + QMR |

# 7. 교육훈련 관리

## 7.1 교육훈련 니즈 분석 (TNA)

매년 1월(또는 회계연도 시작 시) 다음을 기반으로 교육 니즈를 분석한다:

| 입력 | 분석 방법 |
|------|----------|
| 역량 매트릭스 GAP | 미충족 역량 항목 식별 |
| 경영검토 결과 | 개선 필요 영역의 교육 니즈 |
| 내부심사 부적합 | 반복 부적합의 역량 원인 분석 |
| 고객 불만/클레임 | 품질 이슈의 역량 관련 원인 |
| 법규/규격 변경 | 신규 요건에 대한 교육 필요성 |
| 신규 장비/공정 도입 | 작업자 교육 필요성 |
| 조직 변경 | 직무 변경/이동에 따른 교육 필요성 |

## 7.2 연간 교육훈련 계획

인사팀장은 TNA 결과를 기반으로 연간 교육훈련 계획을 수립한다.

| 포함 항목 | 내용 |
|----------|------|
| 교육 과목 | 교육명, 내용 요약 |
| 대상자 | 부서/직무/인원 |
| 일정 | 예정 시기 |
| 교육 방법 | 내부강의/외부교육/OJT/e-Learning |
| 강사 | 내부 강사 또는 외부 교육기관 |
| 예산 | 교육비, 출장비 등 |
| 평가 방법 | 시험/실기평가/과제/설문 |

## 7.3 교육훈련 실시

### 교육 유형별 운영

| 유형 | 내용 | 기록 |
|------|------|------|
| 신입 교육 | QMS 인식, 품질방침/목표, 안전, FOD 인식 | 교육이력카드 |
| 직무 교육 | 담당 업무 관련 절차서/지침서 교육 | 교육이력카드 + 실기평가 |
| OJT | 현장 실습 교육 (숙련자 지도) | OJT 체크리스트 + 서명 |
| 품질 인식 교육 | AS9100D 요구사항, 품질목표, 부적합 사례 | 교육이력카드 |
| 특수공정 교육 | 용접/NDT/열처리 등 자격 취득/갱신 교육 | 자격인증서 + 교육이력카드 |
| 내부심사원 교육 | AS9100D 내부심사 과정 (외부 공인 교육) | 수료증 + 교육이력카드 |
| 법규/안전 교육 | 산업안전, 환경, 수출 통제 등 | 교육이력카드 |

### 필수 교육 (연 1회 이상)

| 교육 과목 | 대상 | 주기 | 비고 |
|----------|------|------|------|
| AS9100D 품질 인식 | 전 직원 | 연 1회 | 품질방침/목표 포함 |
| FOD 방지 교육 | 생산/검사/물류 인원 | 연 1회 | QP-819 연계 |
| 제품 안전 교육 | 설계/생산/품질 인원 | 연 1회 | QP-804 연계 |
| 이물질/오염 방지 | 클린룸/정밀 조립 인원 | 연 1회 | 해당 공정 한정 |
| 인간 요인 (Human Factors) | 생산/검사 인원 | 연 1회 | 항공우주 요건 |

## 7.4 교육 효과성 평가

모든 교육에 대해 효과성을 평가한다:

| 평가 방법 | 적용 기준 | 합격 기준 |
|----------|----------|----------|
| 필기 시험 | 이론 교육 | 70점 이상 |
| 실기 평가 | OJT/기능 교육 | 평가 항목 전 항목 "적합" |
| 과제 제출 | 절차서/지침서 교육 | 담당 부서장 "적합" 판정 |
| 업무 성과 관찰 | OJT 후 3개월 | 부서장 역량 확인 서명 |
| 교육 설문 | 전 교육 | 참조용 (합격/불합격 기준 없음) |

**불합격 시 조치**: 재교육 실시 → 재평가 → 2회 연속 불합격 시 직무 재배치 검토

# 8. 특수공정 인원 자격 관리

> **CAR-013 시정조치**: 항공우주 특수공정 자격 인증 기록 불완전 → 본 절의 체계에 따라 관리

## 8.1 자격 관리 대상 공정

| 특수공정 | 적용 규격 | 자격 수준 | 갱신 주기 |
|---------|----------|----------|----------|
| 용접 (Welding) | AWS D17.1 / D1.1 | Level I~III | 2년 (연속 6개월 미수행 시 재자격) |
| 비파괴검사 (NDT) | NAS 410 / EN 4179 | Level I~III | 5년 (연간 수행실적 유지) |
| 열처리 (Heat Treatment) | AMS 2750 | 작업자/감독자 | 2년 |
| 도금/표면처리 | 사내 기준 (고객 요건 반영) | 작업자/감독자 | 2년 |
| 납땜 (Soldering/Brazing) | IPC J-STD-001 / NASA-STD-8739 | 작업자/검사원 | 2년 |
| 특수 검사 (CMM 등) | 사내 기준 | 작업자 | 2년 |

## 8.2 자격 인증 절차

```
① 자격 요건 확인
   │  해당 규격/고객 요구사항 확인
   ▼
② 교육 이수
   │  이론 + 실기 교육 (내부 또는 외부 공인기관)
   ▼
③ 자격 시험
   │  필기시험 + 실기시험 (규격별 합격 기준 충족)
   ▼
④ 자격 인증서 발행
   │  QMR 또는 공인기관 서명
   ▼
⑤ 자격 관리 대장 등록
   │  유효기간, 갱신일, 수행실적 기록
   ▼
⑥ 정기 갱신/재자격
      유효기간 만료 전 90일 이내 갱신 절차 개시
```

## 8.3 자격 만료 관리

| 항목 | 기준 |
|------|------|
| 만료 예고 | 유효기간 만료 **90일 전** 인사팀 → 해당 부서장 통보 |
| 갱신 기한 | 만료일 **30일 전**까지 갱신 시험/교육 완료 |
| 미갱신 시 | 해당 공정 작업 즉시 중지, 자격 보유자에게 업무 재배정 |
| 장기 미수행 | 6개월 이상 해당 공정 미수행 시 재자격 시험 필요 |
| 자격 정지 | 중대 품질 사고 발생 시 자격 일시 정지 → 재평가 |

## 8.4 자격 관리 대장

자격 관리 대장에는 다음을 기록한다:

| 기록 항목 | 내용 |
|----------|------|
| 성명/사번 | 해당 인원 식별 |
| 자격 종류 | 공정명, 적용 규격 |
| 자격 수준 | Level I/II/III 또는 작업자/감독자 |
| 인증일 | 최초 인증 또는 최근 갱신일 |
| 유효기간 | 만료일 |
| 인증 기관 | 내부(QMR) 또는 외부 공인기관 |
| 수행 실적 | 최근 수행일, 수행 건수 |
| 비고 | 정지/제한 사항 등 |

# 9. 항공우주 특수 요구사항

## 9.1 인간 요인 (Human Factors) 교육

항공우주 산업에서 요구하는 인간 요인(Human Factors) 교육을 실시한다:

| 교육 내용 | 대상 | 목적 |
|----------|------|------|
| 주의력/집중력 관리 | 검사/조립 인원 | 검사 누락 방지 |
| 의사소통 절차 | 전 직원 | 작업 인수인계 오류 방지 |
| 작업 환경 영향 | 생산/검사 인원 | 피로, 조명, 소음 등 영향 인식 |
| 오류 보고 문화 | 전 직원 | 은폐 없는 즉시 보고 문화 |
| 규정 준수 의식 | 전 직원 | 절차 이탈 방지 |

## 9.2 제품 안전 관련 역량

제품 안전에 영향을 미치는 직무 수행자는 추가 역량 요건을 충족해야 한다:

- 안전 관련 법규/규격에 대한 이해
- 안전 위해 보고(Safety Hazard Reporting) 절차 숙지
- 제품 안전 관련 결정 권한 및 한계 인식
- QP-804(제품/서비스 안전 관리) 관련 교육 이수

## 9.3 위조 부품 인식 교육

항공우주 공급망 위조부품(Counterfeit Parts) 방지를 위해:

| 대상 | 교육 내용 | 주기 |
|------|----------|------|
| 구매 인원 | 위조부품 식별 기법, ASL 관리 원칙 | 연 1회 |
| 수입검사 인원 | 위조부품 물리적 식별 방법, 의심 시 보고 절차 | 연 1회 |
| 입고/물류 인원 | 위조부품 유입 경로 인식, 격리 절차 | 연 1회 |
