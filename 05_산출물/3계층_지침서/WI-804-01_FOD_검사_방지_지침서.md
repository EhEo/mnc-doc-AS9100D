## 1. 목적

본 지침서는 M&C의 항공우주 제품 제조 과정에서 이물질(Foreign Object)에 의한 손상(FOD)을 예방하고, FOD 통제구역 내 작업자가 준수해야 할 구체적 행동 기준과 점검 방법을 규정한다.

## 2. 적용범위

- 항공우주 제품의 가공, 조립, 검사, 포장이 이루어지는 모든 작업 구역
- FOD 통제구역으로 지정된 모든 장소
- 해당 구역에 출입하는 모든 인원 (직원, 방문자, 외주 인력 포함)
- 제품 보관, 운반, 출하 과정

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| **FO (Foreign Object)** | 제품 또는 시스템에 속하지 않는 이물질. 금속 칩, 와이어 조각, 볼트/너트, 테이프, 개인 소지품, 먼지 등 |
| **FOD (Foreign Object Debris/Damage)** | 이물질 자체(Debris) 또는 이물질로 인한 손상(Damage) |
| **FOD 통제구역 (FOD Control Area)** | FOD 방지를 위해 특별한 관리 절차가 적용되는 지정 구역 |
| **FOD 워크 (FOD Walk)** | 구역 전체를 걸어 다니며 이물질을 수거하는 정기 점검 활동 |
| **Tool Accountability** | 작업에 사용되는 모든 공구를 반입/반출 시 수량 확인하는 관리 체계 |

## 5. FOD 통제구역 설정

### 5.1 구역 표시

1. FOD 통제구역의 경계를 **노란색 바닥 라인**으로 명확히 표시한다.
2. 출입구에 **"FOD CONTROL AREA / FOD 통제구역"** 표지판을 게시한다.
3. 표지판에는 아래 내용을 포함한다:
   - FOD 통제구역 명칭 및 번호
   - 출입 시 준수사항 요약
   - FOD 관리 책임자 성명/연락처
4. 구역 내부에 FOD 방지 수칙 포스터를 부착한다.

### 5.2 출입 규칙

| 항목 | 규칙 |
|------|------|
| 출입 자격 | FOD 교육 이수자만 출입 가능 |
| 방문자 | 반드시 교육 이수 직원이 동행 |
| 반입 금지 물품 | 개인 음료, 음식, 담배, 신문/잡지, 불필요한 개인 소지품 |
| 필수 착용 | 지정 작업복, 안전화, 필요 시 모자/헤어네트 |
| 휴대폰 | 지정 보관함에 보관 (구역 내 반입 금지, 단 업무용 단말 허용 시 스트랩 착용) |
| 출입 기록 | FOD 통제구역 출입 대장에 서명 |

## 7. 작업 중 FOD 관리

### 7.1 공구 관리

1. **작업 중 공구 배치**: 사용 중인 공구는 지정된 공구 거치대/매트 위에 놓는다. 제품 위나 작업 구역 바닥에 직접 놓지 않는다.
2. **그림자판(Shadow Board)**: 가능한 경우 공구 그림자판을 사용하여 공구 위치를 즉시 파악한다.
3. **소형 공구**: 드릴 비트, 인서트, 핀 등 소형 부품은 자석 트레이 또는 투명 용기에 보관한다.
4. **공구 사용 후**: 사용 완료 공구는 즉시 원위치에 반납한다. "나중에" 정리하지 않는다.
5. **공구 파손 시**: 파손된 부분(조각)을 **모두 수거**한다. 모든 조각이 확인될 때까지 작업을 중단한다.

### 7.2 개구부 보호

1. 제품의 구멍, 튜브, 포트, 커넥터 등 **개구부(Opening)**는 작업하지 않는 동안 반드시 커버/캡/플러그로 보호한다.
2. 보호 덮개는 **밝은 색(빨강, 주황)의 전용 캡**을 사용하여 제거 전 식별이 용이하게 한다.
3. 보호 덮개를 제거할 때는 제거 사실을 기록하고, 작업 완료 후 재장착한다.
4. 보호 덮개 사용 현황을 "개구부 보호 체크리스트"에 기록한다.

### 7.3 파손물/부스러기 즉시 처리

1. 가공 중 발생하는 금속 칩, 버(Burr), 분진은 **즉시** 제거한다.
2. 압축 공기(에어건) 사용 시:
   - 칩이 제품 내부로 유입되지 않도록 방향에 주의한다.
   - 인접 작업대에 영향이 없도록 차폐판을 설치한다.
3. 드릴링/보링 작업 시 관통 전에 반대편 개구부를 보호한다.
4. 작업 중 발생한 폐기물은 즉시 지정된 수거함에 투입한다.

### 7.4 자재/부품 보호

1. 사용 대기 중인 자재/부품은 **밀폐 용기** 또는 **보호 포장** 상태로 보관한다.
2. 부품 이동 시 전용 트레이/운반함을 사용한다.
3. 부품과 공구를 동일 용기에 혼합 보관하지 않는다.
4. 화학 물질(세척제, 윤활유 등) 근처에 부품을 노출 방치하지 않는다.

## 8. 작업 후 FOD 점검

### 8.1 구역 최종 청결도 확인

| 점검 항목 | 점검 내용 | 확인 |
|-----------|-----------|------|
| 작업대 청소 | 모든 칩, 분진, 잔여물 제거 완료 | □ |
| 바닥 청소 | 작업 구역 바닥 청소 완료 | □ |
| 개구부 확인 | 제품의 모든 개구부에 보호 캡/커버 재장착 완료 | □ |
| 제품 표면 | 제품 표면에 이물질, 스크래치, 오염 없음 확인 | □ |
| 폐기물 처리 | 모든 폐기물(칩, 소모품 잔여분)을 수거함에 투입 완료 | □ |

### 8.2 공구 반납 확인

| 점검 항목 | 점검 내용 | 확인 |
|-----------|-----------|------|
| 공구 수량 대조 | 작업 전 기록한 목록과 현재 수량을 **1:1 대조**한다 | □ |
| 소형 부품 확인 | 드릴 비트, 인서트 등 소형 공구 수량 확인한다 | □ |
| 소모품 잔량 | 와이어 타이, 테이프 등 소모품 잔량을 확인한다 | □ |
| 파손 공구 | 파손된 공구가 있으면 모든 조각이 회수되었는지 확인한다 | □ |
| 불일치 시 조치 | 수량 불일치 시 **작업을 중단**하고 FOD 조사를 시작한다 | □ |

> **핵심 규칙**: 공구 수량이 맞지 않으면 해당 구역의 제품 출하를 중지한다. 분실 공구를 찾을 때까지 조사한다.

## 9. FOD 발견 시 조치 절차

### 9.1 즉시 보고

1. FOD를 발견한 작업자는 **즉시 작업을 중단**한다.
2. 발견된 이물질을 **그 자리에서 보전**한다 (증거 훼손 방지).
3. 직속 상사(팀장/반장)에게 구두 보고한다.
4. 팀장은 품질팀에 통보한다.

### 9.2 격리

1. FOD가 발견된 제품을 **부적합품 구역**으로 격리한다.
2. "FOD 의심 — 검사 대기" 라벨을 부착한다.
3. 동일 로트 또는 동일 구역에서 생산된 인접 제품도 격리 대상으로 검토한다.

### 9.3 조사

1. 품질팀이 FOD 발견 보고서(QF-819-02)를 발행한다.
2. 조사 내용:
   - FOD 유형 (금속, 비금속, 개인 소지품, 공구 파편 등)
   - FOD 발견 위치 (제품 내부/외부, 작업대, 바닥 등)
   - 추정 원인 (공정 중 발생, 외부 유입, 공구 파손 등)
   - 제품 손상 여부 평가
3. 제품 손상이 확인된 경우 NCR을 발행한다.

### 9.4 시정조치

1. 근본 원인 분석을 수행한다 (5 Why 또는 Fishbone 기법).
2. 시정조치를 수립하고 실행한다:
   - 작업자 재교육
   - 공구/소모품 관리 강화
   - 작업 환경 개선 (조명, 차폐, 청소 빈도 등)
3. 시정조치 효과를 확인하고 기록을 마감한다.
4. 유사 공정/구역에 수평 전개한다.

## 11. 교육 요건

|------|------|
