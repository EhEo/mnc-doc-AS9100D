## 1. 목적

본 지침서는 M&C의 항공우주 제품 제조에 수반되는 특수공정(Special Process)을 효과적으로 관리하기 위한 구체적 작업 방법을 규정한다. 특수공정은 결과를 후속 검사만으로 완전히 검증할 수 없는 공정이므로, 공정 파라미터 관리, 인원 자격인증, 장비 검증을 통한 사전 유효성 확인이 필수이다.

## 2. 적용범위

- 당사 내부에서 수행하는 모든 특수공정
- 외주처에 위탁하는 모든 특수공정
- 특수공정이 적용되는 모든 항공우주 제품 및 부품
- 특수공정 관련 장비, 인원, 자재, 기록

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| **Special Process (특수공정)** | 결과 출력물을 후속 모니터링 또는 측정에 의해 완전히 검증할 수 없는 공정. 제품 사용 후 또는 서비스 인도 후에만 결함이 드러나는 공정 |
| **NADCAP (National Aerospace and Defense Contractors Accreditation Program)** | 항공우주/방산 특수공정에 대한 국제 인정 프로그램 |
| **CQI (Continuous Quality Improvement)** | AIAG에서 발행한 특수공정 평가 규격 시리즈 |
| **공정 유효성 확인 (Process Validation)** | 특수공정이 의도한 결과를 일관되게 생산할 수 있음을 증명하는 활동 |
| **공정 파라미터** | 특수공정의 결과에 영향을 미치는 온도, 시간, 전류, 농도 등 관리 변수 |
| **Pyrometry** | 열처리 온도 측정 및 기록 시스템 (AMS 2750 준거) |

## 5. 특수공정 목록 및 분류

### 5.1 당사 관련 특수공정 분류

| 분류 | 특수공정 | NADCAP 코드 | 내부/외주 | 적용 규격 예 |
|------|---------|-------------|-----------|-------------|
| 열처리 | 용체화 처리, 시효 처리, 소둔, 응력제거 | AC7004 | 외주 | AMS 2750, AMS 2770 |
| 도금 | 전기 도금(Ni, Cr, Cd, Zn), 무전해 도금 | AC7108 | 외주 | AMS 2404, QQ-N-290 |
| 비파괴검사(NDT) | 침투탐상(PT), 자분탐상(MT), 방사선투과(RT), 초음파(UT) | AC7114 | 내부/외주 | ASTM E1417, ASTM E1444 |
| 용접 | 아르곤 용접(TIG), 스폿 용접, 저항 용접 | AC7102 | 내부 | AWS D17.1 |
| 화학처리 | 양극산화, 화학 에칭, 화성 피막 | AC7108 | 외주 | MIL-A-8625, AMS 2473 |
| 코팅 | 페인트, 프라이머, DFL(건식 윤활 피막) | AC7109 | 외주 | MIL-PRF-23377 |

### 5.2 특수공정 해당 여부 판정

새로운 공정이 특수공정에 해당하는지 아래 기준으로 판정한다:

1. 공정 완료 후 **비파괴적 방법으로 결과를 100% 검증**할 수 있는가?
2. 검증이 불가능하거나 비경제적인 경우 → **특수공정으로 분류**
3. 고객 도면/사양에서 특수공정으로 지정한 경우 → **특수공정으로 분류**
4. 판정 결과를 기술팀장이 승인하고, 특수공정 목록에 등록한다.

## 6. 내부 특수공정 관리

### 6.1 공정 유효성 확인 (Process Validation)

특수공정을 최초 운용하거나 주요 변경 시 아래 유효성 확인을 수행한다:

**1단계: IQ (Installation Qualification) — 장비 적격성 확인**

| 확인 항목 | 내용 |
|-----------|------|
| 장비 설치 | 제조사 사양에 따라 올바르게 설치되었는가 |
| 유틸리티 | 전원, 가스, 냉각수 등이 요구 사양을 충족하는가 |
| 안전 장치 | 비상 정지, 과열 방지 등 안전 장치가 정상 작동하는가 |
| 계측 장비 | 온도계, 전류계 등 계측 장비가 교정되었는가 |

**2단계: OQ (Operational Qualification) — 운전 적격성 확인**

| 확인 항목 | 내용 |
|-----------|------|
| 공정 파라미터 | 규격 요구 범위 내에서 안정적으로 운전되는가 |
| 균일성 | 작업 영역 전체에서 균일한 결과가 나오는가 (예: 노내 온도 균일성) |
| 반복성 | 동일 조건 반복 시 결과가 일관되는가 |
| 극한 조건 | 규격 허용 한계에서도 합격 결과가 나오는가 |

**3단계: PQ (Performance Qualification) — 성능 적격성 확인**

| 확인 항목 | 내용 |
|-----------|------|
| 실제 제품 | 양산 제품을 사용하여 공정을 수행한다 |
| 결과 검증 | 시험/검사를 통해 제품이 모든 사양을 충족하는지 확인한다 |
| 통계적 검증 | 충분한 샘플 수로 공정 능력을 확인한다 |
| 문서화 | Validation Report를 작성하고 승인한다 |

### 6.2 인원 자격인증

| 항목 | 요건 |
|------|------|
| 자격 기준 | 해당 특수공정에 대한 교육 이수 + 실기 평가 합격 |
| NDT 인원 | NAS 410 / EN 4179 / ISO 9712에 따른 Level I, II, III 자격 필수 |
| 용접 인원 | AWS D17.1 또는 해당 규격에 따른 용접사 자격인증 |
| 재인증 주기 | NDT: 5년(Level III), 연간 시력검사 / 용접: 6개월~1년 |
| 자격 기록 | 자격인증서 사본, 교육 이수 기록, 시력검사 결과를 인사 파일에 보관 |
| 자격 정지 | 6개월 이상 해당 공정 미수행 시 재인증 필요 |

### 6.3 장비 검증

| 관리 항목 | 주기 | 기준 |
|-----------|------|------|
| 장비 교정 | 교정 주기에 따라 (연 1회 이상) | 장비 제조사 사양 + 적용 규격 |
| 온도 측정 시스템 (Pyrometry) | AMS 2750 요구 주기 | SAT(System Accuracy Test) 분기별 |
| 용접기 파라미터 | 사용 전 점검 | 전류, 전압, 가스 유량 확인 |
| NDT 장비 | 사용 전 점검 + 정기 교정 | ASTM/해당 규격 요구사항 |
| 도금 조(Bath) | 정기 분석 | 농도, pH, 온도 분석 주기 준수 |

### 6.4 공정 파라미터 관리

1. 각 특수공정의 **필수 관리 파라미터**를 공정 사양서에 명시한다.
2. 파라미터별 **허용 범위(상한/하한)**를 설정한다.
3. 작업자는 공정 수행 중 파라미터를 **실시간으로 모니터링**한다.
4. 자동 기록 장치(데이터 로거, 차트 레코더)가 있는 경우 기록을 저장한다.
5. 파라미터 이탈 발생 시:
   - 즉시 공정을 중단하고 팀장에게 보고한다.
   - 해당 로트를 격리한다.
   - 이탈 원인을 조사하고, 제품 영향을 평가한다.
   - NCR 발행 여부를 품질팀과 협의한다.

#### 열처리 파라미터 관리 예시

| 파라미터 | 관리 항목 | 기록 방법 |
|---------|-----------|-----------|
| 온도 | 설정 온도 ± 허용 편차 | 차트 레코더 / 데이터 로거 |
| 시간 | 유지 시간 (Soak Time) | 타이머 기록 |
| 분위기 | 보호 가스 종류, 유량, 이슬점 | 유량계, 이슬점 측정기 |
| 냉각 | 냉각 매체, 냉각 속도 | 온도-시간 곡선 기록 |
| 장입 | 장입 방법, 간격, 지그 사용 | 장입 기록지 |

## 7. 외주 특수공정 관리

### 7.1 NADCAP 인증 공급자 요건

1. 항공우주 고객이 NADCAP을 요구하는 경우, **NADCAP 인증 외주처**만 사용한다.
2. NADCAP 인증이 요구되지 않는 경우에도, 당사 자체 감사를 통해 공정 능력을 확인한다.
3. 승인된 외주처 목록은 **ASL(Approved Supplier List)**에 등록 관리한다.

### 7.2 외주처 감사

| 감사 유형 | 주기 | 내용 |
|-----------|------|------|
| 초기 감사 | 신규 등록 시 | 공정 능력, 품질 시스템, 자격인증, 장비 상태 전반 평가 |
| 정기 감사 | 연 1회 | 공정 파라미터 관리, 기록 유지, 자격인증 유효성 확인 |
| 특별 감사 | 품질 문제 발생 시 | 부적합 원인 조사, 시정조치 확인 |

**외주처 감사 체크리스트 주요 항목:**

| 항목 | 확인 내용 |
|------|-----------|
| 인증 유효성 | NADCAP 인증서 유효기간, 범위(Scope) 확인 |
| 공정 사양 | 고객/산업 규격에 따른 공정 사양서 보유 여부 |
| 장비 교정 | 계측 장비, 공정 장비의 교정 상태 |
| 인원 자격 | NDT Level, 용접사 자격 등 유효 자격 보유 |
| 기록 관리 | 공정 기록의 완전성, 추적성, 보존 기간 |
| 부적합 관리 | 부적합 발생 시 처리 절차 및 이력 |

## 8. 특수공정 기록 관리

### 8.1 로트별 공정 파라미터 기록

| 기록 항목 | 내용 |
|-----------|------|
| 로트 번호 | 해당 제품/자재의 로트 번호 |
| 작업 지시서 번호 | 해당 작업 지시서(W/O) 번호 |
| 공정 사양 | 적용된 규격 및 사양 번호 |
| 파라미터 기록 | 온도, 시간, 전류, 농도 등 실제 관리값 |
| 차트/데이터 | 온도 차트, 데이터 로거 출력물 |
| 수행자 | 공정 수행자 성명, 자격 번호 |
| 수행 일시 | 공정 시작/종료 일시 |
| 장비 ID | 사용 장비 식별 번호, 교정 유효일 |
| 검사 결과 | 공정 후 검사 결과 (경도, 두께, NDT 결과 등) |
| 합격 여부 | PASS / FAIL |
| 검사자 | 검사 수행자 성명, 자격 번호 (NDT의 경우 Level) |

### 8.2 자격인증 기록

| 기록 유형 | 보관 내용 | 보존 기간 |
|-----------|-----------|-----------|
| NDT 자격인증서 | 자격 등급, 시험 결과, 발급일, 만료일 | 자격 유효기간 + 5년 |
| 용접사 자격인증서 | 자격 범위(재질/두께/자세), 시험편 결과 | 자격 유효기간 + 5년 |
| 교육 기록 | 교육 내용, 이수일, 평가 결과 | 재직기간 + 3년 |
| 시력검사 기록 | 검사일, 결과, 다음 검사 예정일 | 연간 갱신 유지 |

### 8.3 기록 보존

- 특수공정 기록은 최소 **10년** 또는 고객 요구 기간 중 긴 기간 동안 보존한다.
- 전자 기록의 경우 백업을 유지하고, 원본성을 보장한다.
- 기록에 수정이 필요한 경우 취소선 + 수정 내용 + 서명/일자를 기입한다 (백지 수정 금지).

## 9. 특수공정 변경 시 조치

### 9.1 변경 유형 분류

| 변경 유형 | 예시 | FAI 재수행 |
|-----------|------|-----------|
| **Major 변경** | 공정 추가/삭제, 규격 변경, 장비 교체 | Full FAI 필수 |
| **Minor 변경** | 파라미터 범위 내 조정, 동일 유형 장비 교체 | Partial FAI 또는 불요 (품질팀 판단) |
| **외주처 변경** | 특수공정 외주처 변경 | Full FAI 필수 |

### 9.2 변경 처리 절차

1. **변경 요청**: ECR을 발행하여 변경 내용과 사유를 기술한다 (WI-803-01 참조).
2. **영향 분석**: 아래 사항을 평가한다:
   - 제품 성능/안전에 미치는 영향
   - 인증/규격 적합성 영향
   - FAI 재수행 범위
3. **고객 승인**: 고객 사양에서 특수공정을 지정한 경우, 변경 전 **고객 사전 승인**을 득한다.
4. **CCB 심의**: 변경 내용을 CCB에서 심의하고 승인한다.
5. **유효성 재확인**: 변경된 공정의 IQ/OQ/PQ를 필요 범위 내에서 재수행한다.
6. **FAI 재수행**: 품질팀 판정에 따라 Full FAI 또는 Partial FAI를 수행한다 (WI-802-01 참조).
7. **기록 갱신**: 특수공정 목록, 공정 사양서, 외주처 목록 등을 업데이트한다.

### 9.3 고객 승인 필요 사항

아래 경우 반드시 고객 승인을 받은 후 변경을 시행한다:

- 고객 도면/사양에 지정된 특수공정의 규격 변경
- 특수공정 외주처 변경
- NADCAP 인증 범위 외 공정 추가
- 재료와 특수공정의 조합 변경 (예: 다른 합금에 대한 열처리 조건)

## 10. 주의사항

1. **특수공정은 "검사로 품질을 보증할 수 없다"**: 공정 중의 관리가 유일한 품질 보증 수단이다. 공정 파라미터 관리와 기록에 만전을 기한다.
2. **무자격자 작업 금지**: NDT, 용접 등 자격이 요구되는 특수공정은 유효한 자격 보유자만 수행한다. 자격 만료자의 작업은 무효이다.
3. **NADCAP 인증 범위 확인**: 외주처의 NADCAP 인증이 당사 요구 공정/재질/규격을 포함하는지 범위(Scope)를 반드시 확인한다. 인증이 있더라도 범위 밖이면 해당 공정을 위탁할 수 없다.
4. **재작업 관리**: 특수공정의 재작업(예: 재열처리, 재도금)은 재질/제품에 미치는 영향을 기술적으로 검토한 후 수행한다. 무한정 재작업은 허용되지 않는다.
5. **이중 공정 금지**: 도면/사양에 지정되지 않은 추가 특수공정을 임의로 수행하지 않는다.
6. **로트 추적성**: 특수공정 기록과 제품 로트 번호가 반드시 연결되어야 한다. 추적이 불가능한 제품은 출하할 수 없다.
7. **교정 만료 주의**: 공정 중 사용되는 계측/제어 장비의 교정 만료 시, 만료 기간 동안 생산된 제품의 적합성을 재평가해야 한다.
8. **Pyrometry 규정 준수**: 열처리 공정의 온도 측정 시스템은 AMS 2750 요구사항을 충족해야 한다 (TUS, SAT, 열전대 유형/위치 등).
