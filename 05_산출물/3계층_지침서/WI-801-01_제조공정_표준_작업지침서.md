## 1. 목적

본 지침서는 (주)에이로텍의 항공우주 부품 제조 시 각 공정별 표준 작업 방법을 규정하여, 제품 품질의 일관성을 확보하고 AS9100D 8.5.1 요구사항에 적합한 관리된 조건 하에서 생산이 이루어지도록 함을 목적으로 한다.

## 2. 적용범위

- (주)에이로텍 베트남 사업장의 모든 생산 공정에 적용한다.
- 적용 공정: CNC 가공, 프레스 가공, 조립, 용접, 표면처리, 검사
- 특수공정(용접, 표면처리)은 해당 공정 인증 요건을 추가로 준수한다.

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| 초품(First Article) | 생산 시작 시 또는 조건 변경 후 최초 생산된 부품 |
| 4M | Man(작업자), Machine(설비), Material(자재), Method(방법) |
| FOD | Foreign Object Debris, 이물질 손상 |

| 로트(Lot) | 동일 조건에서 생산된 부품 그룹 단위 |

## 5. 공정별 표준 작업 체계

### 5.1 공정 흐름도

```
자재 입고 → 자재 검사 → CNC 가공 / 프레스 가공
    → 용접 (해당 시) → 표면처리 (해당 시)
    → 조립 → 최종 검사 → 포장/출하
```

### 5.2 공정별 관리 기준

| 공정 | 관리 항목 | 기록 양식 |
|------|----------|----------|
| CNC 가공 | 프로그램, 공구, 치수 | QF-810-01 |
| 프레스 가공 | 금형, 압력, 치수 | QF-810-01 |
| 조립 | 토크, 순서, 체결 상태 | QF-810-01 |
| 용접 | 전류, 전압, 속도, 용접사 자격 | QF-810-03 |
| 표면처리 | 온도, 시간, 농도 | QF-810-04 |
| 검사 | 치수, 외관, 기능 | QF-820-01 |

## 6. 작업 시작 전 확인사항

모든 공정의 작업자는 작업 시작 전 아래 사항을 반드시 확인한다.

### 6.1 작업지시서 확인
- 작업지시서(WO) 번호, 부품번호, 수량, 납기일 확인
- 최신 개정판 도면 여부 확인 (도면 번호 및 개정번호 대조)
- 특별 지시사항(고객 요구사항, 특기사항) 확인

### 6.2 도면 확인
- 도면의 개정번호가 작업지시서와 일치하는지 확인
- 주요 치수, 공차, 표면 거칠기 요구사항 확인
- 재질 사양 확인

### 6.3 자재 및 지그 확인
- 자재 식별 라벨과 작업지시서의 자재 사양 일치 여부 확인
- 자재 상태(외관 손상, 유효기간) 확인
- 지그/치구의 상태 및 교정 유효기간 확인

### 6.4 설비 점검
- 일상 점검표에 따른 설비 상태 확인
- 안전장치 작동 확인
- 점검 결과 이상 시 설비보전 담당에게 즉시 보고

## 7. CNC 가공 표준 작업

### 7.1 프로그램 확인
- NC 프로그램 번호가 작업지시서에 지정된 것과 일치하는지 확인
- 프로그램 개정 이력 확인 (최신판 사용)
- 시뮬레이션 실행으로 공구 경로 이상 여부 사전 확인

### 7.2 원점 설정
- 공작물 원점(Work Offset)을 도면 기준점에 맞추어 설정
- 터치 센서 또는 엣지 파인더를 사용하여 정밀 원점 설정
- 원점 설정값을 기록지에 기재

### 7.3 초품 확인
- 초품 1개를 가공하여 주요 치수 전수 측정
- 측정 결과를 초품 검사 기록서(QF-810-02)에 기록
- 품질 담당자 승인 후 양산 진행
- 부적합 시 프로그램/설정 수정 후 재가공 및 재확인

### 7.4 가공 실행
- 절삭 조건(회전수, 이송속도, 절삭깊이)을 프로그램 지정값 준수
- 가공 중 이상 소음, 진동, 공구 마모 발생 시 즉시 정지
- 공구 수명 관리표에 따른 공구 교체 시기 준수

### 7.5 치수 확인
- 로트 내 정기 치수 검사(검사 빈도: 작업지시서 지정)
- 핵심 치수(Key Characteristic)는 전수 또는 SPC 관리
- 치수 이탈 발견 시 즉시 작업 중단, 품질 부서 보고

## 8. 프레스 가공 표준 작업

### 8.1 금형 셋업
- 금형 번호가 작업지시서와 일치하는지 확인
- 금형 상태(손상, 마모, 청결) 점검
- 프레스 스트로크, 압력 설정값을 작업지시서에 따라 설정
- 안전 센서(광전 센서, 양손 조작 장치) 작동 확인

### 8.2 시험타 및 초품 검사
- 시험타 3~5개 실시
- 초품 치수, 외관, 버(Burr) 상태 검사
- 초품 검사 기록서(QF-810-02) 작성 후 품질 승인

### 8.3 양산 작업
- 설정 조건 변경 금지 (변경 필요 시 4M 변경 절차 적용)
- 정기 치수 검사 실시 (매 100개 또는 작업지시서 지정 주기)
- 금형 윤활 주기 준수

## 9. 조립 공정 표준 작업

### 9.1 조립 순서
- 작업지시서에 명시된 조립 순서를 반드시 준수
- 부품 누락 방지를 위한 키트(Kit) 확인
- 부품 식별 라벨 확인 후 조립 진행

### 9.2 토크 관리
- 토크 렌치 사용, 지정 토크값 준수
- 토크 렌치 교정 유효기간 확인
- 토크 적용 후 마킹(토크 실링) 실시
- 토크값을 작업일보에 기록

### 9.3 검사 포인트
- 조립 단계별 자주 검사 실시
- 핵심 체결부위 더블 체크(작업자 + 검사자)
- 조립 완료 후 외관 검사 및 기능 검사 실시

## 10. 용접 공정 표준 작업

### 10.1 용접 자격 확인
- 용접사 자격 인증서 유효기간 확인
- 해당 용접 방법(TIG, MIG 등) 자격 보유 여부 확인

### 10.2 용접 조건 설정
- 전류, 전압, 용접 속도를 WPS(Welding Procedure Specification)에 따라 설정
- 보호가스 유량 확인
- 모재 및 용가재 사양 확인

### 10.3 용접 실행 및 검사
- 용접 전 모재 청결 상태 확인 (기름, 이물질 제거)
- 용접 후 외관 검사(크랙, 기공, 언더컷 등)
- 비파괴검사(NDT) 대상 시 검사 의뢰 및 기록

## 11. 표면처리 공정 표준 작업

### 11.1 처리 조건 확인
- 도금/아노다이징/도장 등 처리 종류 확인
- 처리액 온도, 농도, pH를 관리 기준 내 확인
- 처리 시간 준수

### 11.2 전처리
- 탈지, 산세, 수세 등 전처리 공정 순서 준수
- 전처리 상태 확인 (물 끊김 시험 등)

### 11.3 후처리 및 검사
- 처리 후 외관(색상, 균일성, 결함) 검사
- 두께 측정(필요 시)
- 밀착성 시험(필요 시)

## 12. 작업 완료 후 처리

### 12.1 작업일보 기록
- 작업일보(QF-810-01)에 작업 내용 기록
  - 작업일시, 작업자, 부품번호, 로트번호, 수량(양품/불량)
  - 설비번호, 주요 공정 조건
  - 이상 발생 사항

### 12.2 도구 및 측정기 반납
- 사용한 공구, 지그, 측정기를 지정 장소에 반납
- 파손/이상 공구는 별도 표시 후 관리자에게 보고

### 12.3 청소 및 FOD 관리
- 작업 완료 후 작업대, 설비 주변 청소 실시
- FOD 방지 지침서(WI-FOD-01)에 따른 이물질 제거 확인
- FOD 체크리스트 기록
- 불필요한 자재, 폐자재는 지정 장소로 이동

## 13. 4M 변경 시 조치

### 13.1 4M 변경 정의

| 구분 | 변경 사례 |
|------|----------|
| Man | 작업자 변경, 신규 투입, 교대 |
| Machine | 설비 변경, 수리 후 재가동, 금형 교체 |
| Material | 자재 로트 변경, 대체재 사용 |
| Method | 공정 조건 변경, 프로그램 수정 |

### 13.2 변경 시 처리 절차
1. 변경 사항 발생 시 변경 관리 절차서(QP-816) 에 따라 보고
2. 초품 검사 재실시
3. 초품 합격 후 양산 재개
4. 변경 내용 및 초품 검사 결과를 작업일보에 기록

## 14. 주의사항

1. 도면 및 작업지시서의 최신 개정판만 사용한다. 구판 발견 시 즉시 회수한다.
2. 승인되지 않은 공정 조건 변경은 절대 금지한다.
3. 측정기는 교정 유효기간 내의 것만 사용한다.
4. 안전보호구(안전안경, 장갑, 귀마개 등)를 반드시 착용한다.
5. 부적합품 발견 시 즉시 격리하고, 부적합품 라벨을 부착한 후 품질 부서에 보고한다.
6. FOD 발생 위험이 있는 작업 시 FOD 방지 절차를 준수한다.
7. 특수공정(용접, 표면처리)은 인증된 작업자만 수행한다.
8. 모든 공정 기록은 추적성 확보를 위해 정확하고 즉시 기록한다.
9. 고객 재산(Customer Property)에 해당하는 자재/지그는 별도 식별 관리한다.
10. 본 지침서의 내용과 현장 상황이 상이할 경우, 작업을 중단하고 관리자에게 보고한다.
