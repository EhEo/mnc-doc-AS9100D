# 1. 목적

본 지침서는 QP-818 (부적합 출력 관리 절차서)의 하위 문서로서, M&C Electronics VINA에서 부적합으로 판정된 제품의 식별, 격리, 등급 분류, 처분 결정 및 재검사에 이르는 구체적인 실무 수행 방법을 규정한다.

본 지침서의 주요 목적:
- 부적합품의 신속하고 정확한 식별로 의도하지 않은 사용/인도 방지
- 물리적 격리를 통한 적합품과의 혼입 방지
- 체계적인 등급 분류에 따른 적정 처분 결정
- 항공우주 규격에 부합하는 부적합품 관리 체계 확보

# 4. 용어 및 정의

| 용어 | 정의 |
|------|------|
| 부적합(Nonconformity) | 규정된 요구사항을 충족하지 못하는 상태 |
| NCR (Nonconformance Report) | 부적합 보고서, 부적합 사항의 내용과 처분을 기록하는 문서 |
| 격리(Segregation) | 부적합품을 적합품과 물리적으로 분리하여 보관하는 행위 |
| 특채(Concession) | 규정 요구사항을 충족하지 못한 제품의 사용/인도를 고객이 승인하는 것 |
| 재작업(Rework) | 부적합 제품을 원래 요구사항에 적합하도록 수정하는 작업 |
| 수리(Repair) | 부적합 제품을 의도된 용도에 사용 가능하도록 수정하되, 원래 요구사항을 완전히 충족하지 않을 수 있는 작업 |
| Use-As-Is | 부적합 상태 그대로 사용/인도를 승인하는 것 |
| MRB | 자재심의위원회 (Material Review Board) |
| 의심품(Suspect Part) | 적합 여부가 확인되지 않은 품목 |
| 위조부품(Counterfeit Part) | 출처, 소재, 규격 등이 고의로 허위 표시된 부품 |

# 5. 부적합품 식별 방법

## 5.1 식별 수단 개요

부적합품은 발견 즉시 다음 3가지 수단 중 하나 이상을 사용하여 식별한다.

| 식별 수단 | 적용 대상 | 규격 |
|----------|----------|------|
| 적색 태그 (Red Tag) | 개별 부품, 소형 제품 | 80mm × 120mm, 적색 바탕, 백색 문자 |
| 적색 라벨 (Red Label) | 포장 단위, 파레트 | 50mm × 100mm, 적색 접착 라벨 |
| 적색 마킹 (Red Marking) | 대형 부품, 마킹 가능 소재 | 적색 페인트 또는 적색 유성 마커 |

## 5.2 적색 태그 (Red Tag) 부착 방법

1. 부적합 판정 즉시 적색 태그를 해당 품목에 부착한다.
2. 적색 태그에 다음 정보를 기재한다:
   - NCR 번호 (부여 전이면 "NCR 미부여" 기재)
   - 품명/품번
   - 로트 번호 또는 일련번호
   - 부적합 내용 (간략 기술)
   - 발견 일자
   - 발견자 성명
   - 수량
3. 태그는 케이블 타이 또는 끈으로 제품에 직접 결속한다.
4. 태그가 탈락되지 않도록 견고하게 결속하되, 제품 손상이 없도록 주의한다.

## 5.3 적색 라벨 (Red Label) 부착 방법

1. 포장 단위 또는 파레트 단위 부적합품에 적색 라벨을 부착한다.
2. 라벨에 NCR 번호, 품명, 수량, 발견 일자를 기재한다.
3. 라벨은 포장 외면의 잘 보이는 위치에 부착한다.
4. 개봉 시 훼손되지 않는 위치를 선택한다.

## 5.4 적색 마킹 (Red Marking) 방법

1. 태그/라벨 부착이 불가능한 대형 부품에 적용한다.
2. 고객 도면에서 마킹 허용 구역을 확인한 후 해당 구역에만 마킹한다.
3. 마킹 내용: "NC" 문자 + NCR 번호
4. 마킹은 제품 기능에 영향을 주지 않는 위치에 한다.
5. 처분 후 재작업 시 마킹 제거 가능 여부를 사전 확인한다.

# 6. 격리 구역 설정 및 관리

## 6.1 격리 구역 지정

| 구역 구분 | 위치 | 관리 책임 |
|----------|------|----------|
| 주 격리 구역 (Main Hold Area) | 품질검사실 인접, 지정 보관대 | 품질팀장 |
| 공정 임시 격리 구역 | 각 생산라인 끝단 적색 구역 | 해당 라인 관리자 |
| 수입검사 격리 구역 | 수입검사장 내 지정 구역 | 수입검사 담당자 |
| 고객 반품 격리 구역 | 자재창고 내 별도 구역 | 자재관리 담당자 |

## 6.2 격리 구역 설치 기준

1. **바닥 표시**: 적색 테이프 또는 적색 페인트로 구역 경계를 명확히 표시한다.
2. **표지판**: "부적합품 격리 구역 / NONCONFORMING PRODUCT HOLD AREA" 표지판을 구역 상단에 부착한다.
3. **잠금장치**: 주 격리 구역에는 잠금장치를 설치하고, 열쇠는 품질팀장이 관리한다.
4. **접근 제한**: 격리 구역 출입은 품질팀 승인 인원만 가능하다.
5. **CCTV**: 주 격리 구역에 CCTV를 설치하여 부적합품 무단 이동을 방지한다.

## 6.3 격리 구역 관리 규정

1. 격리 구역 내 부적합품은 NCR 번호순으로 정리 보관한다.
2. 격리 구역 입출고 대장을 비치하고, 입출고 시 기록한다.
3. 격리 구역 관리 담당자는 매주 1회 재고 현황을 점검한다.
4. 30일 이상 체류하는 부적합품은 품질팀장에게 보고하고, 처분 촉진 조치를 취한다.
5. 격리 구역의 환경 조건(온도, 습도)은 제품 보존 요건을 충족해야 한다.

# 7. 부적합 등급 분류

## 7.1 등급 분류 기준

| 등급 | 정의 | 판정 기준 | 처분 권한 |
|------|------|----------|----------|
| **Critical** | 안전성 또는 적합성에 직접 영향을 미치는 부적합 | 안전 관련 규격 불충족, 기능 상실, 구조적 결함 | MRB + 고객 승인 필수 |
| **Major** | 제품 요구사항의 주요 항목 불충족 | 치수 규격 초과, 재료 물성 미달, 외관 주요 결함 | MRB 또는 품질책임자 |
| **Minor** | 제품 기능에 경미한 영향 | 외관 경미 결함, 보조 치수 경미 이탈 | 품질팀장 |

## 7.2 등급 판정 절차

1. 검사자가 부적합을 발견하면 초기 등급을 판정한다.
2. 품질팀 담당자가 초기 등급의 적절성을 검토한다.
3. Critical 등급은 반드시 품질책임자(QMR)가 최종 확인한다.
4. 등급 판정이 어려운 경우 상위 등급으로 분류한다 (보수적 접근 원칙).
5. 등급은 NCR에 기록하고, 등급 변경 시 변경 사유를 명기한다.

# 8. 처분 방법 결정 흐름

## 8.1 처분 유형

| 처분 유형 | 설명 | 고객 승인 | 비고 |
|----------|------|----------|------|
| **재작업 (Rework)** | 원래 요구사항에 적합하도록 수정 | 불요 (원 규격 복원) | 재작업 지시서 필요 |
| **Use-As-Is** | 부적합 상태로 사용 승인 | **필수** | 특채 절차 적용 |
| **수리 (Repair)** | 사용 가능하나 원 규격 미충족 | **필수** | 수리 방법 승인 필요 |
| **특채 (Concession)** | 규격 이탈 상태로 인도 승인 | **필수** | 고객 서면 승인 필수 |
| **폐기 (Scrap)** | 사용 불가 판정, 파쇄/폐기 | 불요 | 폐기 기록 보존 |
| **반품 (Return to Supplier)** | 공급업체로 반환 | 불요 | 공급업체 통보 |

## 8.2 처분 결정 흐름도

```
부적합 발견
    │
    ▼
식별 및 격리 (적색 태그/라벨/마킹)
    │
    ▼
NCR 작성 (F-807-01)
    │
    ▼
등급 분류 (Critical / Major / Minor)
    │
    ├─ Critical ──→ MRB 소집 ──→ 처분 결정 ──→ 고객 승인 요청
    │
    ├─ Major ──→ MRB 또는 품질책임자 ──→ 처분 결정
    │                                      │
    │                          ┌────────────┼────────────┐
    │                          ▼            ▼            ▼
    │                       재작업        수리/특채      폐기
    │                          │         (고객 승인)      │
    │                          ▼            │            ▼
    │                       재검사          ▼         폐기 기록
    │                          │        고객 승인
    │                          ▼            │
    │                       합격/불합격      ▼
    │                                    재검사
    │
    └─ Minor ──→ 품질팀장 ──→ 처분 결정
```

## 8.3 MRB 소집 기준

다음 중 하나에 해당하면 MRB를 소집한다:
- Critical 등급 부적합
- Major 등급 중 안전/기능 영향 가능성 있는 건
- 동일 부적합이 3회 이상 반복 발생한 건
- 고객 특별 요구사항 관련 부적합

# 9. 고객 승인 필요 항목

## 9.1 고객 사전 승인 필수 항목

다음 처분은 반드시 실행 전 고객의 서면 승인을 받아야 한다:

| 항목 | 승인 요건 | 제출 서류 |
|------|----------|----------|
| 특채 (Concession) | 고객 서면 승인 | NCR, 부적합 상세, 영향 분석 |
| 수리 (Repair) | 고객 서면 승인 + 수리 방법 승인 | NCR, 수리 절차서, 재검사 계획 |
| Use-As-Is | 고객 서면 승인 | NCR, 기능 영향 분석, 안전성 검토 |
| 설계 변경 수반 처분 | 고객 + 설계 권한자 승인 | ECN, NCR, 영향 분석 |

## 9.2 고객 승인 요청 절차

1. NCR에 처분 의견과 기술적 근거를 기재한다.
2. 품질책임자(QMR)가 승인 요청서를 작성한다.
3. 고객 품질 담당자에게 서면(이메일/공문)으로 승인을 요청한다.
4. 고객 승인 회신을 NCR에 첨부하여 보관한다.
5. 고객 미승인 시 대안 처분을 결정한다 (재작업 또는 폐기).

# 10. 재작업 후 재검사 요건

## 10.1 재작업 실시 요건

1. 재작업 전 반드시 재작업 지시서를 작성하고 품질팀 승인을 받는다.
2. 재작업 지시서에는 다음을 포함한다:
   - NCR 번호
   - 재작업 대상 품목 및 수량
   - 부적합 내용
   - 재작업 방법 (구체적 작업 절차)
   - 재작업 후 검사 항목 및 기준
   - 담당 작업자 및 검사자

## 10.2 재검사 수행

1. 재작업 완료 후 원래 검사 항목 전체에 대해 재검사를 실시한다.
2. 재검사는 원래 검사를 수행한 검사자와 다른 인원이 수행하는 것을 원칙으로 한다.
3. 재검사 결과를 NCR의 재검사 란에 기록한다.
4. 재검사 합격 시 적색 태그/라벨/마킹을 제거하고, 격리 구역에서 반출한다.
5. 재검사 불합격 시 재차 NCR을 발행하거나 기존 NCR에 추가 기록하고, 처분을 재결정한다.

## 10.3 재작업 횟수 제한

- 동일 품목에 대한 재작업은 최대 2회까지 허용한다.
- 2회 재작업 후에도 부적합인 경우 폐기 또는 MRB 재검토를 실시한다.
- 고객이 재작업 횟수를 별도 규정한 경우 고객 요구를 우선 적용한다.

# 11. NCR 작성 방법

## 11.1 NCR(F-807-01) 작성 항목

| 항목 | 기재 내용 | 작성 책임 |
|------|----------|----------|
| NCR 번호 | 연도-일련번호 (예: NCR-2026-001) | 품질팀 |
| 발견 일자 | 부적합 발견 일시 | 발견자 |
| 발견 공정 | 수입검사/공정검사/최종검사/고객반품 | 발견자 |
| 품명/품번 | 해당 제품 정보 | 발견자 |
| 로트번호/일련번호 | 추적성 확보용 식별 번호 | 발견자 |
| 수량 | 부적합 수량 / 검사 수량 | 발견자 |
| 부적합 등급 | Critical / Major / Minor | 품질팀 |
| 부적합 내용 | 객관적 사실에 기반한 부적합 기술 | 발견자 |
| 적용 규격/기준 | 불충족한 규격 또는 도면 요구사항 | 품질팀 |
| 처분 결정 | 재작업/수리/특채/폐기/반품 | MRB/품질책임자 |
| 고객 승인 여부 | 해당 시 승인 문서번호 및 일자 | 품질팀 |
| 재검사 결과 | 합격/불합격, 검사 데이터 | 검사자 |
| 시정조치 연계 | CAR 번호 (해당 시) | 품질팀 |

## 11.2 NCR 작성 시 주의사항

1. 부적합 내용은 객관적 사실만 기술한다 (추측, 원인 분석은 별도 란에 기재).
2. 측정값이 있는 경우 실측값과 규격값을 함께 기재한다.
3. 가능한 경우 사진을 첨부한다.
4. NCR은 발견 후 24시간 이내에 발행한다.
5. Critical 등급 NCR은 발견 후 4시간 이내에 품질책임자에게 보고한다.

# 12. 항공우주 특수 요구사항

## 12.1 의심품 (Suspect Part) 처리

1. 적합 여부가 불확실한 품목은 "의심품"으로 분류하고 부적합품과 동일하게 격리한다.
2. 의심품 태그 (황색 바탕)를 부착하고, "SUSPECT" 표기를 추가한다.
3. 의심품의 출처, 이력, 인증서를 조사하여 적합/부적합을 판정한다.
4. 판정 완료 시까지 의심품은 사용하지 않는다.
5. 판정 결과에 따라 적합품 반환 또는 부적합 절차를 진행한다.

## 12.2 위조부품 (Counterfeit Part) 격리

1. 위조가 의심되는 부품은 즉시 격리하고 품질책임자(QMR)에게 보고한다.
2. 위조부품 전용 격리함 (잠금 보관함)에 별도 보관한다.
3. SAE AS6081/AS6174에 따라 해당 부품의 출처 및 유통 경로를 조사한다.
4. 조사 결과를 정부 당국 또는 인증 기관에 보고한다 (해당 국가 법규에 따름).
5. 위조부품은 폐기 처분하며, 재사용/수리/특채를 절대 허용하지 않는다.
6. 해당 공급업체에 대한 시정조치를 요구하고, ASL(승인 공급업체 목록) 재검토를 실시한다.

## 12.3 안전 관련 부적합 에스컬레이션

1. 제품 안전에 영향을 미치는 부적합 발견 시 다음 에스컬레이션 절차를 따른다:

| 단계 | 시간 기준 | 조치 내용 | 보고 대상 |
|------|----------|----------|----------|
| 1단계 | 발견 즉시 | 생산 중지, 해당 로트 전수 격리 | 라인 관리자 |
| 2단계 | 발견 후 1시간 | 초기 보고 및 영향 범위 파악 | 품질책임자(QMR) |
| 3단계 | 발견 후 4시간 | 상세 보고 및 초기 원인 분석 | 공장장/경영진 |
| 4단계 | 발견 후 24시간 | 고객 통보 및 시정조치 계획 | 고객 품질 담당 |
| 5단계 | 발견 후 48시간 | 규제 당국 보고 (해당 시) | 인증 기관/당국 |

2. 안전 관련 부적합은 모든 처분에 고객 승인을 필수로 한다.
3. 이미 출하된 제품 중 동일 로트가 있는 경우 고객에게 즉시 통보한다.
4. 안전 관련 부적합 기록은 영구 보존한다.
