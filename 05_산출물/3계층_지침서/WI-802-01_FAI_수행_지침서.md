## 1. 목적

본 지침서는 (주)에이로텍이 생산하는 항공우주 부품에 대해 AS9102B 규격에 따른 초도품 검사(First Article Inspection, FAI)를 정확하고 일관되게 수행하기 위한 구체적 작업 방법을 규정한다.

## 2. 적용범위

- 신규 부품(Part Number) 최초 양산 시
- 설계 변경(ECN)으로 형상/치수/재료/공정이 변경된 경우
- 제조 공정 변경(공정 이전, 장비 교체, 금형 변경 등)
- 2년 이상 생산 중단 후 재개 시
- 고객이 FAI를 별도 요청한 경우

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| **FAI (First Article Inspection)** | 최초 생산품이 도면/사양의 모든 요구사항을 충족하는지 검증하는 계획된 완전 검사 |
| **FAIR (First Article Inspection Report)** | FAI 결과를 기록한 보고서 패키지 (Form 1+2+3) |
| **발룬 도면 (Ballooned Drawing)** | 각 치수/특성에 고유 번호를 부여하여 Form 3과 연결한 도면 |
| **KC (Key Characteristic)** | 고객 또는 사양에서 지정한 특수특성 (안전/기능/성능에 중대 영향) |
| **Partial FAI** | 변경된 특성만 재검사하는 부분 FAI |
| **Full FAI** | 모든 특성을 100% 검사하는 완전 FAI |

## 5. FAI 수행 전 준비사항

### 5.1 발룬 도면 작성

1. 최신 개정판 도면을 확보한다 (개정 번호, 날짜 반드시 확인).
2. 도면의 모든 치수, 공차, 표면 조도, 기하공차(GD&T), 노트에 **고유 일련번호**(Balloon Number)를 부여한다.
3. 번호 부여 규칙:
   - 순번은 1부터 시작, 연속 번호 사용
   - 도면이 여러 장인 경우 시트별 연번 이어서 부여
   - 참고 치수(Reference Dimension)는 "(REF)" 표기하고 Form 3에서 제외 가능
4. 발룬 도면에 작성자, 작성일, 도면 번호/개정 번호를 기재한다.

> **Tip**: 발룬 번호 원은 빨간색으로 표시하면 식별이 용이하다.

### 5.2 측정 장비 준비

1. Form 3의 각 특성에 적합한 측정 장비를 사전에 선정한다.
2. 모든 측정 장비의 **교정 유효기간**을 확인한다 (만료 시 사용 불가).
3. 교정 스티커의 유효기간, 장비 ID를 기록용으로 메모한다.
4. 측정 장비 정밀도는 공차의 **1/10 이하** 원칙을 적용한다.

| 측정 대상 | 권장 장비 | 정밀도 기준 |
|-----------|-----------|------------|
| 외경/내경 치수 | 마이크로미터, 버니어캘리퍼스 | 공차의 1/10 |
| 기하공차 (진원도, 직각도 등) | CMM (3차원측정기) | 공차의 1/10 |
| 표면 조도 | 표면 조도계 | Ra 측정 가능 |
| 경도 | 경도계 (로크웰/비커스) | 해당 규격 준수 |
| 나사 | 나사 게이지 (Go/No-Go) | 해당 규격 준수 |

### 5.3 자재 증명서 확인

1. 원자재 밀 시트(Mill Certificate / CoC)를 확보한다.
2. 밀 시트 상의 재질, 로트 번호, 화학 성분, 기계적 물성이 도면/사양과 일치하는지 대조한다.
3. 특수공정 인증서(열처리 성적서, 도금 성적서 등)가 필요한 경우 함께 준비한다.
4. 모든 증빙 서류에 **추적 가능한 로트 번호**가 기재되어 있는지 확인한다.

## 6. AS9102 Form 1 작성 방법

**Form 1: Part Number Accountability**

Form 1은 FAI 대상 부품의 기본 정보를 기록하는 표지이다.

| 필드 | 기입 요령 |
|------|-----------|
| Part Number | 도면에 명시된 부품 번호 그대로 기입 |
| Part Name | 부품명 (도면/BOM 기준) |
| Part Revision Level | 현재 적용 도면 개정 번호 (예: Rev. C) |
| Drawing Number | 도면 번호 (Part Number와 다를 수 있음) |
| Organization Name | (주)에이로텍 / M&C Electronics VINA |
| Manufacturing Process Reference | 공정 흐름도 번호 또는 라우팅 문서 참조 |
| Serial Number | FAI 대상 제품의 시리얼 번호 |
| FAI Type | Full FAI 또는 Partial FAI 선택 |
| Reason for Partial FAI | Partial인 경우 사유 기입 (예: ECN-2026-045에 의한 치수 변경) |
| Baseline Part Number | Partial FAI 시 기존 Full FAI의 Part Number |
| Additional Changes | 부가 변경사항 기술 |
| Signature / Date | FAI 수행자 서명 및 날짜 |

### 기입 시 주의사항
- Part Number에 Dash Number가 있는 경우 완전한 번호를 기입한다 (예: MNC-1234-01).
- Full FAI와 Partial FAI를 명확히 구분한다.
- 서명란에는 FAI를 실제 수행한 검사원이 서명한다.

## 7. AS9102 Form 2 작성 방법

**Form 2: Raw Material, Special Process, Functional Test**

Form 2는 원자재, 특수공정, 기능시험의 적합성을 기록한다.

### 7.1 Raw Material (원자재) 섹션

| 필드 | 기입 요령 |
|------|-----------|
| Material or Process Name | 재질명 (예: Al 7075-T6, SUS 304) |
| Specification | 적용 규격 (예: AMS 4045, ASTM A240) |
| Code | D(Design), P(Process), S(Specification) |
| Certificate Number | 밀 시트/CoC 번호 |
| Report Result | PASS 또는 FAIL 기입 |

### 7.2 Special Process (특수공정) 섹션

| 필드 | 기입 요령 |
|------|-----------|
| Process Name | 공정명 (예: 열처리, 양극산화, NDT) |
| Specification | 적용 규격 (예: AMS 2750, AMS 2471) |
| Supplier | 수행업체명 (내부 또는 외주처) |
| Approval | NADCAP 인증번호 또는 고객 승인 번호 |
| Report Result | PASS 또는 FAIL 기입 |

### 7.3 Functional Test (기능시험) 섹션

| 필드 | 기입 요령 |
|------|-----------|
| Test Name | 시험명 (예: 내압시험, 기밀시험, 전기전도도) |
| Procedure Number | 시험 절차서 번호 |
| Report Number | 시험 성적서 번호 |
| Report Result | PASS 또는 FAIL 기입 |

### 증빙 연결 방법
- 각 항목의 Certificate/Report Number를 기입하고, 해당 원본 서류를 FAIR 패키지에 첨부한다.
- 증빙 서류에 **FAI Part Number와 로트 번호**가 명시되어 있는지 반드시 확인한다.

## 8. AS9102 Form 3 작성 방법

**Form 3: Characteristic Accountability — 핵심 양식**

Form 3은 발룬 도면의 모든 특성을 100% 측정하여 기록하는 양식이다.

### 8.1 필드별 기입 요령

| 필드 | 기입 요령 |
|------|-----------|
| Char. No. | 발룬 번호와 동일한 일련번호 |
| Reference Location | 해당 특성이 있는 도면 시트/존 (예: Sheet 1, Zone B3) |
| Characteristic Designator | KC(Key Characteristic), 또는 빈칸 |
| Requirement | 도면에 명시된 규격값 (예: 25.00 ±0.05) |
| Results | 실제 측정값 (예: 25.02) |
| Designed Tooling | Y(예) / N(아니오) — 전용 지그/툴링 사용 여부 |
| Non-Conformance Number | 부적합 발생 시 NCR 번호 기입 |
| Status | PASS 또는 FAIL |

### 8.2 100% 치수 측정 원칙

- FAI는 **도면의 모든 치수, 공차, 노트를 예외 없이** 측정/확인한다.
- 참고 치수(REF)는 측정 불요하나, Form 3에 "(REF)" 표기하여 누락이 아님을 명시한다.
- GD&T 특성(진원도, 위치도, 윤곽도 등)도 모두 개별 행으로 기록한다.
- 표면 조도, 경도 등 비치수 특성도 해당되는 경우 기록한다.

### 8.3 특수특성 KC 식별

1. 도면에서 KC(Key Characteristic) 표시를 확인한다 (삼각형 기호, 다이아몬드 기호, 또는 "KC" 표기).
2. Form 3의 "Characteristic Designator" 칸에 **"KC"**를 기입한다.
3. KC 항목은 공정능력(Cpk) 관리 대상이므로, SPC 데이터 연계 여부를 확인한다.

### 8.4 합격/불합격 판정

| 상황 | 판정 | 조치 |
|------|------|------|
| 측정값이 공차 이내 | **PASS** | 해당 없음 |
| 측정값이 공차 초과 | **FAIL** | NCR 발행 후 Non-Conformance Number 기입 |
| 측정 불가능한 특성 | 사유 기재 | 별도 설명 첨부 |

## 9. 측정 실행

### 9.1 측정 장비 선택 기준

측정 장비는 아래 기준에 따라 선택한다:

1. **정밀도 규칙**: 장비 분해능 ≤ 공차의 1/10
2. **반복성**: 동일 조건 반복 측정 시 편차 최소화
3. **측정 범위**: 대상 치수의 범위를 포함
4. **교정 상태**: 유효한 교정 인증서 보유

### 9.2 측정 방법

1. 측정 전 제품과 장비를 **동일 온도 환경**(20±2°C 권장)에 30분 이상 안정화한다.
2. 기준면(Datum)을 먼저 확인하고, 도면의 측정 기준에 따라 순서대로 측정한다.
3. 각 특성별로 측정값을 즉시 Form 3에 기록한다 (나중에 기억에 의존하지 않는다).
4. CMM 측정 시:
   - 측정 프로그램을 사전 작성/검증한다.
   - 프로브 교정을 매 측정 세션 시작 시 수행한다.
   - 측정 결과 리포트를 출력하여 FAIR에 첨부한다.

### 9.3 데이터 기록

- 측정값은 **도면 단위**(mm, inch)와 동일 단위로 기록한다.
- 소수점 자릿수는 공차 표기와 동일하게 맞춘다 (예: 공차 ±0.05 → 측정값 25.02).
- GD&T 특성은 측정 결과를 적절한 단위로 환산하여 기입한다.

## 10. 결과 판정

### 10.1 합격 (Accepted)

- Form 3의 **모든** 특성이 PASS이고, Form 2의 모든 항목이 PASS인 경우.
- FAI 수행자와 품질팀장이 서명한다.
- FAIR 패키지를 완성하여 품질기록으로 보관한다.

### 10.2 조건부 합격 (Conditional Acceptance)

- 일부 특성이 FAIL이나, 해당 부적합에 대해 **고객 승인(Concession/Waiver)**을 득한 경우.
- NCR 번호와 고객 승인 문서 번호를 Form 3에 기록한다.
- 고객 승인 서류를 FAIR에 첨부한다.
- 조건부 합격 사유를 Form 1 비고란에 명시한다.

### 10.3 불합격 (Rejected)

- 1건 이상의 특성이 FAIL이고 고객 승인이 없는 경우.
- NCR을 발행하고 시정조치를 수행한다.
- 시정조치 완료 후 **FAI를 재수행**한다 (해당 특성 또는 전체).
- 재수행 시 신규 FAIR 양식을 사용하며, 이전 FAIR 번호를 참조 기록한다.

## 11. FAIR 패키지 편철 순서

FAIR 패키지는 아래 순서로 편철하여 품질기록으로 보관한다:

| 순서 | 문서 | 비고 |
|------|------|------|
| 1 | FAIR 표지 (Form 1) | Part Number Accountability |
| 2 | Form 2 | Raw Material / Special Process / Functional Test |
| 3 | Form 3 | Characteristic Accountability (측정 결과) |
| 4 | 발룬 도면 | 모든 시트 포함 |
| 5 | 원자재 증명서 (Mill Certificate / CoC) | 로트 번호 추적 가능 |
| 6 | 특수공정 성적서 | 열처리, 도금, NDT 등 |
| 7 | 기능시험 성적서 | 해당 시 |
| 8 | CMM 측정 리포트 | 해당 시 |
| 9 | NCR / Concession 서류 | 부적합 발생 시 |
| 10 | 기타 고객 요구 서류 | 해당 시 |

- 전자 보관 시 PDF 변환하여 DMS에 등록한다.
- **보존 기간**: 고객 요구사항에 따르며, 최소 10년 또는 제품 수명 종료 후 3년 중 긴 기간.

## 12. 주의사항 및 FAQ

### 주의사항

1. **FAI는 양산 공정과 동일한 조건**에서 생산된 제품으로 수행해야 한다. 시작품(Prototype)으로는 FAI를 수행할 수 없다.
2. 발룬 번호가 Form 3의 Char. No.와 **1:1 대응**되어야 한다. 누락 또는 불일치 시 심사 부적합 사유가 된다.
3. Partial FAI 수행 시 **기존 Full FAI의 번호와 날짜**를 반드시 참조 기입한다.
4. FAI 결과가 FAIL인 경우 해당 로트는 **출하 불가**이다.
5. 측정 장비의 교정이 만료된 경우 해당 장비로 측정한 FAI는 **무효**이다.

### FAQ

**Q1. 동일 Part Number에 여러 Revision이 있을 때 FAI는?**
A1. 현재 유효한 최신 Revision에 대해 FAI를 수행한다. 이전 Rev의 FAI는 무효 처리한다.

**Q2. 다품종 소량 생산 시 매번 FAI를 수행해야 하나?**
A2. 최초 생산 시 Full FAI를 수행하고, 동일 조건 재생산 시에는 불요하다. 단, 2년 초과 시 재수행한다.

**Q3. Form 3에 특성이 100개 이상인 경우?**
A3. 여러 페이지에 걸쳐 연속 번호로 작성한다. 각 페이지에 Part Number와 페이지 번호를 기재한다.

**Q4. 고객이 AS9102가 아닌 자체 FAI 양식을 요구하면?**
A4. 고객 양식을 우선 적용하되, 내부적으로 AS9102 Form도 병행 작성하여 보관한다.

**Q5. 외주 가공품의 FAI 책임은?**
A5. 최종 조립/납품 책임이 있는 (주)에이로텍이 FAIR를 작성한다. 외주처 측정 데이터는 증빙으로 활용 가능하나, 당사 검증이 필요하다.
