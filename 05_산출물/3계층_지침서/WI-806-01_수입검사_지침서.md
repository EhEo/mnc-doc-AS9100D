## 1. 목적

본 지침서는 M&C(M&C Electronics VINA)에서 수행하는 수입검사(IQC)의 구체적인 방법과 기준을 규정하여, 외부 공급자로부터 입고되는 원자재, 부품 및 외주가공품의 품질을 확보하고 AS9100D 8.4.3 요구사항을 충족함을 목적으로 한다.

## 2. 적용범위

- 생산에 투입되는 모든 원자재, 부품, 외주가공품
- 항공우주 제품에 사용되는 모든 구매 품목
- 고객 지급품(Customer Furnished Material)
- 적용 제외: 사무용품, 간접자재 (품질에 영향을 미치지 않는 항목)

## 5. 수입검사 대상 분류

### 5.1 분류 기준

| 등급 | 대상 | 검사 수준 | 비고 |
|------|------|----------|------|
| A등급 (Critical) | 항공우주 핵심 원자재, 특수공정 외주품 | 전수검사 또는 강화 샘플링 | AQL 0.65 |
| B등급 (Major) | 일반 부품, 표준 외주가공품 | 보통 샘플링 | AQL 1.0 |
| C등급 (Minor) | 범용 자재, 볼트/너트 등 표준품 | 완화 샘플링 | AQL 2.5 |

### 5.2 대상별 분류

| 구분 | 예시 | 기본 검사 수준 |
|------|------|--------------|
| 원자재 | 알루미늄판, 스테인레스강, 동합금 | A등급 (Mill Cert 필수) |
| 부품 (구매) | 커넥터, 패스너, 씰, O-링 | B등급 |
| 외주가공품 | CNC 가공품, 열처리품, 도금품 | A~B등급 (공정에 따라) |
| 고객 지급품 | 고객 제공 자재/부품 | 수량 및 외관 확인 |

## 6. 검사 전 준비

### 6.1 발주서 확인

1. 구매발주서(PO)와 납품 수량 대조
2. 발주 사양(도면번호, 규격, 재질) 확인
3. 납기일 준수 여부 확인

### 6.2 납품서류 확인

입고 자재와 함께 다음 서류가 제출되었는지 확인한다:

| No. | 필수 서류 | 확인 항목 | A등급 | B등급 | C등급 |
|-----|----------|----------|-------|-------|-------|
| 1 | 거래명세서/납품서 | 품명, 수량, 로트번호 | 필수 | 필수 | 필수 |
| 2 | CoC (적합성 인증서) | 규격 충족 선언, 서명, 일자 | 필수 | 필수 | 선택 |
| 3 | Mill Certificate | 화학성분, 기계적 성질 | 필수 | 해당 시 | - |
| 4 | 시험성적서 | 경도, 인장, 내식성 등 | 필수 | 해당 시 | - |
| 5 | 열처리/표면처리 성적서 | 온도, 시간, 두께 등 | 해당 시 | 해당 시 | - |
| 6 | FAI 보고서 | 초도품 시 AS9102 양식 | 해당 시 | 해당 시 | - |
| 7 | 원산지 증명서 | 수입 자재 원산지 확인 | 해당 시 | 해당 시 | - |

### 6.3 서류 불비 시 처리

- 필수 서류가 미제출된 경우 **입고 보류** 처리
- 구매팀을 통해 공급자에게 서류 보완 요청
- 서류 보완 완료 전 생산 투입 금지

## 7. 검사 수행

### 7.1 외관검사

1. 포장 상태 확인 (파손, 오염, 습기 여부)
2. 제품 외관 확인:
   - 스크래치, 찍힘, 변색, 부식
   - 도금/도장 상태 (박리, 기포, 얼룩)
   - 이물질(FOD) 부착 여부
3. 라벨/마킹 상태 확인 (품번, 로트번호, 수량)

### 7.2 치수검사

1. 검사 도면 및 Control Plan 기준 측정 항목 확인
2. 교정된 측정장비를 사용하여 측정 수행:
   - 캘리퍼: 일반 치수 (±0.1mm 이상 공차)
   - 마이크로미터: 정밀 치수 (±0.05mm 이하 공차)
   - CMM: 형상 공차, 위치도 등 기하공차
3. 측정 결과를 검사 기록서에 기록

### 7.3 기능시험

- 해당되는 경우 기능/성능 시험을 실시한다:
  - 전기적 특성: 저항, 절연, 내전압
  - 기계적 특성: 조립 적합성, 체결력
  - 기밀 시험: 누설 여부

### 7.4 AQL 샘플링 기준 (ANSI/ASQ Z1.4)

| 로트 크기 | 검사 수준 II 샘플 수 | AQL 0.65 Ac/Re | AQL 1.0 Ac/Re | AQL 2.5 Ac/Re |
|----------|-------------------|---------------|---------------|---------------|
| 2~8 | 2 | 0/1 | 0/1 | 0/1 |
| 9~15 | 3 | 0/1 | 0/1 | 0/1 |
| 16~25 | 5 | 0/1 | 0/1 | 0/1 |
| 26~50 | 8 | 0/1 | 0/1 | 0/1 |
| 51~90 | 13 | 0/1 | 0/1 | 1/2 |
| 91~150 | 20 | 0/1 | 0/1 | 1/2 |
| 151~280 | 32 | 0/1 | 1/2 | 2/3 |
| 281~500 | 50 | 1/2 | 1/2 | 3/4 |
| 501~1200 | 80 | 1/2 | 2/3 | 5/6 |
| 1201~3200 | 125 | 2/3 | 3/4 | 7/8 |

> **Ac**: 합격판정 개수, **Re**: 불합격판정 개수

### 7.5 샘플링 전환 규칙

| 전환 | 조건 |
|------|------|
| 보통 → 완화 | 연속 10로트 합격 + 공급자 평가 A등급 |
| 보통 → 강화 | 연속 5로트 중 2로트 불합격 |
| 강화 → 보통 | 연속 5로트 합격 |
| 강화 → 검사 중지 | 연속 10로트 불합격 시 거래 중지 검토 |

## 8. 특수검사 항목

### 8.1 자재증명서(Mill Certificate) 대조

1. Mill Certificate의 화학성분이 **주문 규격(ASTM, AMS 등)**에 부합하는지 확인
2. 기계적 성질(인장강도, 항복강도, 연신율)이 규격 범위 이내인지 확인
3. Heat Number/Lot Number가 실물 마킹과 일치하는지 확인
4. 불일치 발견 시 **즉시 입고 보류** 및 NCR 발행

### 8.2 성분분석

- XRF(형광X선 분석기) 또는 외부 시험기관을 활용
- 다음 경우에 성분분석을 실시:
  - 신규 공급자 초도 납품
  - 자재증명서 미비 또는 의심 시
  - 고객 요구 시
  - 항공우주 안전 관련 핵심 자재

### 8.3 경도검사

- 해당 자재/부품에 경도 요구사항이 있는 경우 실시
- 로크웰(HRC/HRB), 브리넬(HB), 비커스(HV) 경도계 사용
- 측정 위치: 최소 3점 측정 후 평균값으로 판정
- 열처리품의 경우 경도 분포 확인 (표면/심부)

## 9. 합격/불합격 판정 기준

### 9.1 합격 기준

다음 조건을 **모두** 충족해야 합격으로 판정한다:

1. 납품 서류가 완비되고 내용이 적합함
2. 외관검사 결과 결함이 없음
3. 치수검사 결과 모든 측정값이 도면 공차 이내
4. 기능시험 결과 요구사항 충족 (해당 시)
5. 특수검사 결과 규격 이내 (해당 시)
6. AQL 샘플링 결과 합격판정 개수(Ac) 이내

### 9.2 불합격 기준

다음 중 **하나라도** 해당되면 불합격으로 판정한다:

1. 필수 서류 미비 또는 내용 부적합
2. 외관 결함 발견 (용인 불가 수준)
3. 치수 공차 초과
4. 기능/성능 요구사항 미달
5. 자재 성분/경도 규격 미달
6. AQL 샘플링 결과 불합격판정 개수(Re) 이상

## 10. 합격품 식별 및 입고 처리

### 10.1 식별 표시

1. 합격품에 **녹색 합격 스티커** 부착
2. 스티커 기재 항목:
   - 검사일
   - 검사원 서명/도장
   - 로트번호
   - 수량
3. 자재/부품의 **추적성 라벨** 부착 (로트번호, PO번호, 입고일)

### 10.2 입고 처리

1. 수입검사 기록서(F-804-01) 작성 완료
2. ERP 시스템에 입고 등록 (검사 합격 상태 기록)
3. 지정된 보관 장소(자재 창고)로 이동
4. 선입선출(FIFO) 원칙에 따라 보관

## 11. 불합격품 처리

### 11.1 격리

1. 불합격품에 **적색 불합격 태그** 부착
2. **불합격품 격리 구역**으로 즉시 이동
3. 불합격 사유를 태그에 명기

### 11.2 NCR 발행

1. 부적합 보고서(NCR)를 작성한다.
2. NCR에 다음 내용을 기록:
   - 불합격 내용 (결함 유형, 측정값, 판정 기준)
   - 불합격 수량
   - 영향 범위 (다른 로트, 기 투입 여부)

### 11.3 공급자 통보 및 SCAR

1. 불합격 내용을 공급자에게 **서면 통보**
2. 다음 경우 SCAR(공급자 시정조치 요구서)를 발행:
   - 동일 결함 반복 발생 (3개월 내 2회 이상)
   - 안전/기능에 영향을 미치는 중대 결함
   - 서류 위변조 의심
3. 공급자는 **15영업일** 이내에 시정조치 결과를 제출해야 한다.
4. SCAR 미회신 또는 부적절한 회신 시 **ASL 등급 강등** 또는 거래 중지 검토

### 11.4 불합격품 처분

| 처분 방법 | 조건 | 승인 권한 |
|----------|------|----------|
| 반품 (Return) | 공급자 귀책 | 품질팀장 |
| 특채 (Concession) | 사용 가능 판단 시 | 품질팀장 + 고객 승인 (항공우주) |
| 선별 사용 | 일부 합격품 활용 가능 시 | 품질팀장 |
| 폐기 (Scrap) | 사용 불가 | 품질팀장 |

> **항공우주 제품**: 특채(Concession) 시 반드시 **고객 사전 승인** 필요

## 12. 긴급 입고(Dock-to-Stock) 조건

### 12.1 적용 조건

긴급 입고는 다음 조건을 **모두** 충족하는 경우에만 허용한다:

1. 생산라인 정지 임박 등 **긴급 사유** 발생
2. 해당 공급자가 **ASL A등급** (최우수)
3. 최근 6개월간 **수입검사 불합격 이력 없음**
4. **CoC 및 필수 서류**가 완비된 경우
5. 품질팀장 및 **생산팀장 공동 승인**

### 12.2 긴급 입고 절차

1. 긴급 입고 요청서 작성 (요청 사유, 승인자 서명)
2. 서류 검토 (CoC, Mill Cert 등) 수행
3. 외관검사는 **반드시 실시**
4. 입고 후 **24시간 이내** 정규 수입검사 완료
5. 검사 불합격 시 즉시 생산라인 투입분 **소급 관리** 실시

### 12.3 제한사항

- 항공우주 **안전 관련 품목**은 긴급 입고 **적용 불가**
- 신규 공급자 납품분은 긴급 입고 **적용 불가**
- 월간 긴급 입고 건수가 **전체 입고의 5%** 초과 시 품질팀장 원인 분석 실시

## 13. 수입검사 기록서 작성

### 13.1 양식

**F-804-01 수입검사 기록서**를 사용한다.

### 13.2 기록 항목

| No. | 항목 | 내용 |
|-----|------|------|
| 1 | 검사일자 | 검사 수행 일자 |
| 2 | PO번호 | 구매발주서 번호 |
| 3 | 공급자명 | 납품 공급자 |
| 4 | 품번/품명 | 자재/부품 식별 정보 |
| 5 | 로트번호 | 공급자 로트번호 |
| 6 | 납품수량/검사수량 | 전체 수량 / 샘플 수량 |
| 7 | 검사 항목 | 외관/치수/기능/특수검사 항목 |
| 8 | 판정 기준 | 도면/규격 기준값 및 공차 |
| 9 | 측정 결과 | 실측값 기록 |
| 10 | 판정 | 합격/불합격 |
| 11 | 불합격 시 조치 | NCR 번호, 처분 방법 |
| 12 | 검사원 | 서명/도장 |
| 13 | 첨부서류 | CoC, Mill Cert 등 첨부 여부 |

### 13.3 기록 보관

- 수입검사 기록서: **최소 10년** 보관
- 항공우주 제품 관련: **제품 수명 + 5년** 보관
- 전자 파일 및 원본 동시 보관

## 14. 주의사항

1. **ASL에 등록되지 않은 공급자**로부터의 자재는 입고할 수 없다.
2. 수입검사를 생략하거나 검사 기준을 임의로 완화할 수 없다. 변경 시 품질팀장 승인 필수.
3. CoC 없이 입고된 항공우주 자재는 **생산 투입을 금지**한다.
4. 자재의 **유효기간(Shelf Life)**이 있는 경우, 만료일을 확인하고 만료 자재는 입고 거부한다.
5. 검사 중 **FOD(이물질)**가 발견된 경우 해당 로트 전체를 격리 조사한다.
6. 검사에 사용하는 측정장비는 반드시 **교정 유효기간 내** 장비만 사용한다.
7. 불합격 이력이 있는 공급자의 다음 납품분은 **강화 검사**를 적용한다.
8. 검사 결과에 대한 **위변조 및 조작은 엄격히 금지**하며, 적발 시 징계 조치한다.
9. 고객 지급품도 수량 확인 및 외관검사를 실시하고, 결함 발견 시 고객에게 즉시 통보한다.
10. 수입검사 데이터는 월별로 집계하여 **공급자 평가**에 반영한다.
