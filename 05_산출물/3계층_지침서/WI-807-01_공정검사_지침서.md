## 1. 목적

본 지침서는 M&C(M&C Electronics VINA)의 생산 공정 중 수행하는 공정검사(PQC)의 유형, 시점, 방법 및 부적합 처리 절차를 구체적으로 규정하여, 공정 품질을 확보하고 AS9100D 8.5.1 및 8.6 요구사항을 충족함을 목적으로 한다.

## 2. 적용범위

- 본 지침서는 모든 생산 공정에서의 품질검사 활동에 적용한다.
- 원자재 투입 후 최종검사 전까지의 모든 공정을 포함한다.
- 사내 제조 공정 및 사내 특수공정을 포함한다.

## 5. 공정검사 유형

### 5.1 초품검사 (First Piece Inspection)

| 항목 | 내용 |
|------|------|
| 목적 | 생산 시작 전 공정 세팅 상태 확인 |
| 시점 | 생산 개시 시, 로트 전환 시, 설비 세팅 변경 후 |
| 수량 | 최초 1~3개 (Control Plan 기준) |
| 수행자 | 작업자 실시 → 품질검사원 확인 |
| 검사 항목 | 전 치수 항목 + 외관 + 기능 |
| 합격 조건 | 모든 항목 도면/규격 이내 |
| 기록 | F-805-01 공정검사 기록서 |

**초품검사 절차:**
1. 작업자가 첫 번째 제품을 생산한다.
2. 작업자가 도면 기준 **전수 치수** 측정을 실시한다.
3. 품질검사원이 측정 결과를 **확인 검증**한다.
4. 합격 시 초품검사 기록서에 서명하고 양산을 개시한다.
5. 불합격 시 세팅을 조정하고 초품검사를 재실시한다.

### 5.2 순회검사 (Patrol Inspection)

| 항목 | 내용 |
|------|------|
| 목적 | 생산 중 공정 안정성 지속 확인 |
| 시점 | 생산 중 정해진 주기에 따라 |
| 빈도 | 2~4시간마다 (공정 위험도에 따라) |
| 수행자 | 품질검사원 |
| 검사 항목 | KC 항목 + 주요 치수 + 외관 |
| 샘플 수량 | 3~5개/회 (Control Plan 기준) |
| 기록 | F-805-01 공정검사 기록서 |

**순회검사 절차:**
1. 품질검사원이 정해진 일정에 따라 생산현장을 순회한다.
2. 각 공정에서 최근 생산된 제품을 **무작위 샘플링**한다.
3. KC 항목 및 주요 치수를 측정한다.
4. 작업 환경(온도, 습도, 청결 상태)을 확인한다.
5. 설비 상태 및 작업표준서 준수 여부를 확인한다.
6. 검사 결과를 기록하고, 이상 발견 시 즉시 조치한다.

### 5.3 자주검사 (Self Inspection)

| 항목 | 내용 |
|------|------|
| 목적 | 작업자가 자기 공정 품질을 직접 관리 |
| 시점 | 작업 중 연속적으로 |
| 빈도 | 매 N개마다 (Control Plan 기준, 통상 10~50개) |
| 수행자 | 해당 공정 작업자 (자주검사 인증자) |
| 검사 항목 | 지정 치수 항목 + 외관 |
| 기록 | 자주검사 체크시트 |

**자주검사 자격 요건:**
- 해당 공정 작업 경력 6개월 이상
- 자주검사 교육 이수 (측정기 사용법, 도면 판독)
- 품질팀장 승인에 의한 **자주검사원 인증** 보유

### 5.4 공정 이관검사 (Process Transfer Inspection)

| 항목 | 내용 |
|------|------|
| 목적 | 공정 간 이관 시 이전 공정 품질 확인 |
| 시점 | 한 공정 완료 후, 다음 공정 투입 전 |
| 수행자 | 이전 공정 작업자 또는 품질검사원 |
| 검사 항목 | 해당 공정 완료 기준 항목 |
| 합격 조건 | 이전 공정 전 항목 합격 |
| 기록 | Lot Traveler에 공정 완료 서명 |

## 6. 검사 시점 및 빈도

### 6.1 정기 검사 시점

| 시점 | 검사 유형 | 수행자 | 비고 |
|------|----------|--------|------|
| 생산 시작 시 | 초품검사 | 작업자 + 검사원 | 매 로트/시프트 |
| 생산 중 (2~4시간) | 순회검사 | 검사원 | KC 공정은 2시간 |
| 생산 중 (연속) | 자주검사 | 작업자 | Control Plan 기준 |
| 공정 완료 시 | 이관검사 | 작업자/검사원 | Lot Traveler 서명 |

### 6.2 비정기 검사 시점 (4M 변경 시)

다음의 4M 변경이 발생한 경우 **초품검사를 재실시**한다:

| 변경 유형 | 예시 | 조치 |
|----------|------|------|
| Man (작업자) | 작업자 교대/교체 | 초품검사 재실시 |
| Machine (설비) | 설비 교체, 금형 교환, 공구 교환 | 초품검사 재실시 |
| Material (자재) | 자재 로트 변경, 공급자 변경 | 초품검사 재실시 |
| Method (방법) | 공정조건 변경, 작업표준서 변경 | 초품검사 재실시 |

### 6.3 추가 검사 시점

- 장시간 생산 중단 후 재가동 시 (4시간 이상)
- 설비 고장 수리 후 재가동 시
- 공정 이상 발견 후 조치 완료 시
- 고객 클레임 관련 공정 재확인 시

## 7. 검사 항목 및 기준

### 7.1 Control Plan 기반 검사

1. 모든 공정검사는 해당 제품의 **Control Plan**에 명시된 항목과 기준에 따라 수행한다.
2. Control Plan에는 다음이 포함된다:
   - 공정 번호 및 공정명
   - 검사 항목 (치수, 외관, 기능)
   - 규격 (도면 기준값 ± 공차)
   - 측정 방법 및 장비
   - 샘플 크기 및 빈도
   - 반응 계획 (이상 발견 시 조치)

### 7.2 특수특성 KC (Key Characteristic) 관리

| KC 등급 | 정의 | 관리 방법 | 표시 |
|---------|------|----------|------|
| KC-S (Safety) | 안전에 영향을 미치는 특성 | SPC 필수 + 전수검사 | ◇ (다이아몬드) |
| KC-F (Fit/Function) | 기능/조립에 영향을 미치는 특성 | SPC 권장 + 강화 샘플링 | △ (삼각형) |
| 일반 특성 | 기본 품질 특성 | 표준 샘플링 | 표시 없음 |

**KC 항목 관리 원칙:**
1. KC-S 항목은 **100% 전수검사** 또는 **SPC 관리**를 실시한다.
2. KC 항목의 공정능력 목표: **Cpk ≥ 1.33**
3. KC 항목은 도면 및 Control Plan에 **특수특성 기호**로 표시한다.
4. KC 항목의 측정 데이터는 별도 관리하고 **추적성**을 확보한다.

### 7.3 검사 항목 세부

| 검사 유형 | 항목 | 기준 | 장비 |
|----------|------|------|------|
| 외관검사 | 스크래치, 찍힘, 버(Burr), 변색 | 한도 견본 기준 | 육안, 확대경 |
| 치수검사 | 길이, 폭, 두께, 직경, 위치도 | 도면 공차 | 캘리퍼, 마이크로미터, CMM |
| 형상검사 | 진원도, 평면도, 직각도 | 도면 기하공차 | CMM, 진원도 측정기 |
| 기능시험 | 조립 적합성, 체결력, 전기특성 | 사양서 기준 | 전용 지그, 테스터 |

## 8. SPC 적용 공정 관리

### 8.1 SPC 적용 대상

다음 조건에 해당하는 공정에 SPC를 적용한다:

1. KC-S (안전) 특성이 포함된 공정
2. KC-F (기능) 특성 중 공정능력 확보가 필요한 항목
3. 고객이 SPC 적용을 요구한 공정
4. 과거 부적합 이력이 빈번한 공정

### 8.2 관리도 운영

| 관리도 유형 | 적용 조건 | 데이터 |
|------------|----------|--------|
| X-bar/R 관리도 | 계량형 데이터, 소그룹(n=3~5) | 측정값 |
| X-bar/S 관리도 | 계량형 데이터, 대그룹(n≥10) | 측정값 |
| p 관리도 | 계수형 데이터, 불량률 | 불량 개수/검사 수 |

**관리도 작성 절차:**
1. 최소 **25개 서브그룹** 데이터를 수집한다.
2. 관리 한계선(UCL, LCL)을 계산한다.
3. 데이터를 관리도에 기입한다.
4. 이상 패턴을 확인한다.

### 8.3 관리도 이상 판정 규칙 (Western Electric Rules)

| 규칙 | 판정 기준 | 조치 |
|------|----------|------|
| Rule 1 | 1점이 관리 한계선(±3σ) 밖 | 즉시 공정 중지 및 조사 |
| Rule 2 | 연속 9점이 중심선 한쪽 | 공정 치우침 조사 |
| Rule 3 | 연속 6점이 증가 또는 감소 추세 | 공정 변동 원인 조사 |
| Rule 4 | 연속 14점이 교대로 증감 | 과잉 조정 여부 확인 |

### 8.4 Cpk 관리 기준

| Cpk 값 | 판정 | 조치 |
|--------|------|------|
| Cpk ≥ 1.67 | 우수 | 현 공정 유지 |
| 1.33 ≤ Cpk < 1.67 | 양호 | 현 공정 유지, 모니터링 |
| 1.00 ≤ Cpk < 1.33 | 주의 | 공정 개선 계획 수립 |
| Cpk < 1.00 | 부적합 | 전수검사 실시 + 즉시 공정 개선 |

> **항공우주 제품 목표**: KC 항목 **Cpk ≥ 1.33** 이상 유지

## 9. 공정 부적합 발견 시 조치

### 9.1 라인 정지 기준

다음의 경우 **즉시 해당 공정(또는 라인)을 정지**한다:

1. KC-S (안전특성) 항목 부적합 발견
2. 관리도 Rule 1 위반 (관리 한계선 이탈)
3. 동일 결함이 **연속 3개** 이상 발견
4. 설비 이상으로 제품 품질에 영향이 우려되는 경우
5. FOD(이물질) 발견 시

### 9.2 격리 조치

1. 부적합품에 **적색 부적합 태그** 부착
2. 부적합품을 **격리 구역**으로 즉시 이동
3. 마지막 합격 확인 시점 이후의 제품을 **의심품**으로 분류
4. 의심품 전수를 격리하여 재검사 실시

### 9.3 원인 조사

1. 4M 관점에서 부적합 원인을 분석한다:
   - Man: 작업자 숙련도, 작업 표준 준수 여부
   - Machine: 설비 상태, 공구 마모, 세팅 이상
   - Material: 자재 불량, 로트 변경
   - Method: 작업 방법, 공정 조건 이상
2. 원인 파악 후 시정조치를 실시한다.
3. 시정조치 후 **초품검사를 재실시**하여 공정 정상화를 확인한다.

### 9.4 재작업/폐기 판정

| 처분 방법 | 조건 | 승인 권한 |
|----------|------|----------|
| 재작업 (Rework) | 재가공으로 규격 충족 가능 | 품질팀장 |
| 수리 (Repair) | 원래 규격 미달이나 사용 가능 | 품질팀장 + 설계 확인 |
| 특채 (Concession) | 규격 미달이나 기능 영향 없음 | 품질팀장 + 고객 승인 (항공우주) |
| 폐기 (Scrap) | 재작업 불가 또는 비경제적 | 품질팀장 |

> **항공우주 제품 주의**: 재작업/수리 시 반드시 **승인된 절차**에 따라 수행하고, 재작업 후 **재검사**를 실시한다.

### 9.5 부적합 보고

1. 부적합 보고서(NCR)를 작성한다.
2. NCR에 다음을 기록한다:
   - 부적합 내용 및 발견 공정
   - 부적합 수량 및 로트번호
   - 원인 분석 결과
   - 처분 방법 (재작업/폐기/특채)
   - 시정조치 내역
   - 유효성 확인 결과
3. NCR은 품질팀장에게 보고하고, 월간 품질 회의에서 검토한다.

## 10. 검사 기록서 작성 방법

### 10.1 양식

**F-805-01 공정검사 기록서**를 사용한다.

### 10.2 기록 항목

| No. | 항목 | 내용 |
|-----|------|------|
| 1 | 검사일자/시간 | 검사 수행 일시 |
| 2 | 제품명/품번 | 제품 식별 정보 |
| 3 | 로트번호 | 생산 로트 번호 |
| 4 | 공정명/공정번호 | 검사 대상 공정 |
| 5 | 검사 유형 | 초품/순회/자주/이관 |
| 6 | 검사 항목 | 치수/외관/기능 항목 |
| 7 | 규격 (기준값±공차) | 도면/Control Plan 기준 |
| 8 | 측정값 | 실측 데이터 |
| 9 | 판정 | 합격/불합격 |
| 10 | 부적합 시 조치 | NCR 번호, 처분 내역 |
| 11 | 검사원 서명 | 수행자 확인 |
| 12 | 비고 | 4M 변경, SPC 데이터 등 |

### 10.3 기록 관리

- 공정검사 기록서는 생산 완료 후 **Lot Traveler**에 첨부 보관한다.
- 전자 기록은 서버에 백업하고, 원본은 품질팀에서 보관한다.
- 보관 기간: **최소 10년** (항공우주 제품: **제품 수명 + 5년**)
- SPC 데이터는 별도 데이터베이스에 축적하여 추이 분석에 활용한다.

## 11. 주의사항

1. **초품검사 합격 전에는 양산을 개시할 수 없다.** 초품 불합격 시 원인 조치 후 재검사를 실시한다.
2. 공정검사에 사용하는 측정장비는 반드시 **교정 유효기간 내** 장비만 사용한다. (WI-805-01 참조)
3. 자주검사를 수행하는 작업자는 반드시 **자주검사원 인증**을 보유해야 한다.
4. 4M 변경 발생 시 **반드시 초품검사를 재실시**한다. 미실시 시 해당 로트는 부적합 처리한다.
5. KC 항목은 일반 항목보다 **강화된 관리**(SPC, 전수검사)를 적용한다.
6. 공정 중 **FOD(이물질)**가 발견되면 즉시 라인을 정지하고 원인을 제거한다.
7. 검사 결과의 **위변조 및 조작은 엄격히 금지**하며, 적발 시 징계 조치한다.
8. 부적합 발생 시 **"마지막 합격 확인 시점"**까지 소급하여 의심품을 관리한다.
9. SPC 관리도에서 이상 패턴이 발견되면 즉시 원인을 조사하고 조치한다. 방치하지 않는다.
10. 모든 공정검사 기록은 **추적성(Traceability)**이 확보되어야 하며, 로트번호와 연계 관리한다.
