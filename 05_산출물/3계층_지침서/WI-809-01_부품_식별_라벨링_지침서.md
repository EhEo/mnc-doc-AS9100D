## 1. 목적

본 지침서는 (주)에이로텍에서 취급하는 원자재, 반제품, 완제품에 대한 식별 및 라벨링 방법을 구체적으로 규정하여, AS9100D 8.5.2 요구사항에 따른 추적성을 확보하고 제품의 혼입을 방지함을 목적으로 한다.

## 2. 적용범위

- (주)에이로텍 베트남 사업장에서 취급하는 모든 원자재, 부자재, 반제품, 완제품에 적용한다.
- 수입 검사, 공정 중, 최종 검사, 포장/출하 전 단계에 걸쳐 적용한다.
- 고객 지급품(Customer Property) 및 외주 처리품에도 동일하게 적용한다.

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| 로트(Lot) | 동일 자재, 동일 조건에서 생산된 부품 단위 그룹 |
| 로트번호(Lot No.) | 로트를 고유하게 식별하는 번호 체계 |
| Lot Traveler | 로트와 함께 이동하며 공정 이력을 기록하는 문서 |
| 부품번호(P/N) | 부품을 고유하게 식별하는 번호 |
| SN(Serial Number) | 개별 부품에 부여되는 일련번호 |
| 검사 상태 | 합격/불합격/보류/미검사 등 검사 결과 상태 |

## 7. 라벨 부착 방법

### 7.1 부착 위치

| 대상 | 부착 위치 | 비고 |
|------|----------|------|
| 원자재(봉재/판재) | 자재 끝단 또는 포장재 외면 | 자재 손상 방지 위치 |
| 소형 부품 용기 | 용기 전면 상단 | 쉽게 식별 가능한 위치 |
| 대형 부품 | 비가공면 또는 부착 가능한 태그 | 가공면 부착 금지 |
| 포장 박스 | 박스 외면 2곳(전면, 측면) | 적재 시 가시성 확보 |

### 7.2 고정 방법

| 방법 | 적용 대상 | 주의사항 |
|------|----------|----------|
| 접착 라벨 | 포장재, 용기, 보관함 | 탈락 방지 접착력 확인 |
| 케이블 타이 태그 | 대형/중량물, 봉재 | 태그 재질: 내열/내유성 |
| 와이어 태그 | 고온 공정 전 부품 | 와이어 고정 시 부품 손상 방지 |
| 각인/마킹 | 고객 요구 시 | 도면 지정 위치, 마킹 방법 준수 |

### 7.3 내구성 요건
- 라벨은 해당 공정 환경(온도, 습도, 유류, 세정액)에서 판독 가능해야 한다.
- 표면처리/도금 전 공정 라벨은 내약품성 재질을 사용한다.
- 옥외 보관 자재 라벨은 방수 재질을 사용한다.
- 라벨 인쇄 잉크는 번짐 방지 처리된 것을 사용한다.

## 8. 로트번호 부여 규칙

### 8.1 로트번호 형식

```
YYMMDD-NNN

YY  : 연도 끝 2자리 (예: 26 = 2026년)
MM  : 월 2자리 (예: 04 = 4월)
DD  : 일 2자리 (예: 15 = 15일)
NNN : 당일 순번 3자리 (001부터 시작)
```

**예시**: `260415-001` = 2026년 4월 15일 첫 번째 로트

### 8.2 로트번호 부여 시점

| 시점 | 설명 |
|------|------|
| 자재 입고 시 | 수입검사 합격 후 로트번호 부여 |
| 공정 분리 시 | 1개 로트가 2개 이상으로 분리될 경우 하위 로트 부여 (예: 260415-001A, -001B) |
| 재작업 시 | 원 로트번호에 "-RW" 접미사 추가 (예: 260415-001-RW) |

### 8.3 로트번호 관리대장
- 로트번호 발행 시 로트번호 관리대장(전산 또는 수기)에 등록한다.
- 관리대장 기재 항목: 로트번호, 부품번호, 수량, 발행일, 발행자

## 9. 바코드 라벨 인쇄 및 스캔 방법

### 9.1 바코드 규격
- 1D 바코드: Code 128 형식 사용
- 2D 바코드: QR Code 형식 사용 (고객 요구 시)
- 바코드 내용: 로트번호 + 부품번호 조합

### 9.2 인쇄 방법
1. 바코드 인쇄 프로그램에 로트번호, 부품번호 입력
2. 인쇄 미리보기로 정보 정확성 확인
3. 라벨 프린터로 인쇄 (해상도: 최소 300dpi)
4. 인쇄 후 바코드 스캐너로 판독 테스트 실시
5. 판독 불가 시 재인쇄

### 9.3 스캔 방법
- 공정 투입/완료 시 바코드 스캔으로 이력 등록
- 스캔 실패 시 수기 입력 후 시스템 담당자에게 보고
- 스캔 기록은 MES/ERP 시스템에 자동 저장

## 10. 공정 이동표(Lot Traveler) 기입 요령

### 10.1 Lot Traveler 구성

| 섹션 | 기재 내용 |
|------|----------|
| 헤더 | 부품번호, 로트번호, 수량, 작업지시서 번호, 도면 개정번호 |
| 공정란 | 공정명, 작업일, 작업자, 양품/불량 수량, 비고 |
| 검사란 | 검사 항목, 결과(합격/불합격), 검사자, 검사일 |
| 특이사항란 | 4M 변경, 이상 발생, 재작업 내역 |

### 10.2 기입 규칙
1. 반드시 볼펜(청색 또는 흑색)으로 기재한다. 연필 사용 금지.
2. 오기 시 두 줄 긋고 정정, 정정자 서명 및 일자 기재 (수정액/수정테이프 사용 금지)
3. 각 공정 완료 시 해당 공정란에 즉시 기입한다. 사후 일괄 기입 금지.
4. 공란은 빈칸으로 두지 않고 사선(/)으로 말소한다.
5. Lot Traveler는 해당 로트와 반드시 함께 이동한다.

### 10.3 Lot Traveler 보관
- 완료된 Lot Traveler는 품질 부서에서 수거하여 보관
- 보관 기간: 제품 수명 + 3년 (또는 고객 계약 요구사항에 따름)

## 11. 라벨 손상 및 분실 시 처리

### 11.1 라벨 손상 시
1. 손상 라벨을 제거하지 않고, 새 라벨을 인접 위치에 부착
2. 손상 사유를 Lot Traveler 특이사항란에 기록
3. 바코드 라벨 손상 시 재인쇄 후 판독 테스트 실시

### 11.2 라벨 분실 시
1. 작업을 즉시 중단하고 관리자에게 보고
2. 라벨 재발행 요청서(QF-812-02)를 작성
3. 해당 로트의 Lot Traveler, 작업일보, 시스템 기록을 통해 로트 확인
4. 확인 완료 후 라벨 재발행 및 부착
5. 재발행 이력을 Lot Traveler 특이사항란에 기록

## 12. 식별 불가 자재 처리

### 12.1 식별 불가 판정 기준
- 라벨이 없거나 완전히 훼손되어 판독 불가한 경우
- 라벨 정보와 실물이 일치하지 않는 경우
- 로트번호, 부품번호를 어떠한 기록으로도 확인할 수 없는 경우

### 12.2 처리 절차

```
식별 불가 발견 → 즉시 격리 (적색 태그 부착)
    → 품질 부서 보고
    → 조사 실시 (작업일보, 시스템 기록, 인근 로트 확인)
    → [재식별 가능] → 라벨 재발행 후 복귀
    → [재식별 불가] → 부적합품 처리 절차(QP-815) 적용 → 폐기 또는 고객 협의
```

### 12.3 기록
- 식별 불가 자재 발생 건을 부적합 보고서로 기록
- 재발 방지 대책 수립 및 시행

## 13. 주의사항

1. 라벨이 없는 자재/부품은 절대 사용하거나 다음 공정으로 이동시키지 않는다.
2. 라벨 정보를 임의로 수정하거나 덮어쓰지 않는다.
3. 라벨 부착 시 부품 기능면(가공면, 밀봉면 등)에 직접 부착하지 않는다.
4. 유효기간 관리 자재는 선입선출(FIFO) 원칙을 적용한다.
5. 부적합품 라벨(적색)이 부착된 자재는 지정 격리 구역에만 보관한다.
6. Lot Traveler와 실물의 수량이 일치하지 않을 경우 즉시 작업을 중단하고 조사한다.
7. 바코드 스캔 시 오류가 반복될 경우 장비 점검을 요청한다.
8. 고객 지급품은 고객 라벨을 제거하지 않고, 자사 라벨을 추가 부착한다.
9. 라벨 인쇄 장비는 월 1회 이상 점검하여 인쇄 품질을 유지한다.
10. 본 지침서와 상이한 고객별 식별 요구사항이 있을 경우, 고객 요구사항을 우선 적용한다.
