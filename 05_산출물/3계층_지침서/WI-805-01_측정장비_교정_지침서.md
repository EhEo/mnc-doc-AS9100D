## 1. 목적

본 지침서는 (주)에이로텍(M&C Electronics VINA)에서 사용하는 측정장비 및 계측기의 교정 수행 방법을 구체적으로 규정하여, 측정 결과의 신뢰성을 확보하고 AS9100D 7.1.5 요구사항을 충족함을 목적으로 한다.

## 2. 적용범위

- 본 지침서는 품질에 영향을 미치는 모든 측정장비 및 계측기에 적용한다.
- 생산, 검사, 시험에 사용되는 장비를 포함한다.
- "참조용(Reference Only)"으로 분류된 장비는 교정 대상에서 제외하되, 식별 라벨을 부착한다.

## 4. 용어 정의

| 용어 | 정의 |
|------|------|
| 교정(Calibration) | 측정장비가 표시하는 값과 표준기 값 간의 편차를 확인하고 기록하는 행위 |
| OOT (Out of Tolerance) | 교정 결과 허용 공차를 벗어난 상태 |
| 소급성(Traceability) | 국가/국제표준에 연결된 불절단 교정 체인 |

| MPE | 최대허용오차 (Maximum Permissible Error) |

## 6. 교정 수행 절차

### 6.1 내부교정 vs 외부교정 판단 기준

| 구분 | 내부교정 | 외부교정 |
|------|---------|---------|
| 조건 | 자사 표준기 보유 및 교정 역량 확보 시 | 표준기 미보유 또는 고정밀 장비 |
| 표준기 | ISO/IEC 17025 인정기관에서 교정된 표준기 사용 | 해당 없음 |
| 교정원 | 사내 교정 교육 이수자 (연 1회 이상) | ISO/IEC 17025 인정 교정기관 |
| 대상 장비 | 캘리퍼, 마이크로미터, 높이게이지, 핀게이지 등 | CMM, 저울, 토크렌치, 경도계, 게이지블록 등 |
| 성적서 | 사내 교정 성적서 발행 | 교정기관 성적서 수령 |

### 6.2 교정 전 확인사항

1. 교정 대상 장비의 **관리번호** 확인
2. 장비의 **외관 상태** 점검 (파손, 변형, 오염 여부)
3. 이전 **교정 이력** 및 **교정 만료일** 확인
4. 교정에 사용할 **표준기의 교정 유효기간** 확인
5. 교정 환경 조건 확인:
   - 온도: 20±2℃ (정밀 측정 시)
   - 습도: 45~75% RH
   - 진동, 먼지 등 외부 간섭 요인 배제
6. 장비를 교정 환경에 **최소 2시간 전 안치**하여 온도 안정화

### 6.3 교정 실행 — 내부교정

#### 6.3.1 캘리퍼/마이크로미터 교정

1. 영점 확인: 측정면을 밀착시킨 후 영점 상태 확인
2. 게이지블록을 사용하여 다음 측정점에서 편차 확인:
   - 0~150mm 캘리퍼: 10mm, 50mm, 100mm, 150mm (최소 4점)
   - 0~25mm 마이크로미터: 5mm, 10mm, 15mm, 20mm, 25mm (5점)
3. 각 측정점에서 **3회 반복 측정**하여 평균값 산출
4. 편차가 MPE 이내인지 판정
5. 측정 결과를 사내 교정 성적서에 기록

#### 6.3.2 높이게이지 교정

1. 정반 위에 장비를 안치하고 영점 설정
2. 게이지블록 조합으로 다음 측정점 확인:
   - 0~300mm: 50mm, 100mm, 150mm, 200mm, 300mm (5점)
3. 각 측정점에서 **3회 반복 측정**
4. 편차가 MPE 이내인지 판정

### 6.4 교정 실행 — 외부교정

1. 교정 대상 장비 목록을 작성하여 **품질팀장 승인** 획득
2. ISO/IEC 17025 인정 교정기관에 교정 의뢰
   - 교정기관의 인정서(KOLAS/ILAC 등) 유효기간 확인
3. 장비 발송 시 **충격 방지 포장** 실시
4. 교정기관으로부터 교정 성적서 수령 후 내용 확인

### 6.5 교정 성적서 확인사항

외부 교정 성적서 수령 시 다음 항목을 반드시 확인한다:

- [ ] 교정기관명 및 인정번호
- [ ] 교정 대상 장비 정보 (모델, 시리얼 번호)
- [ ] 교정 일자 및 유효 기간
- [ ] 교정에 사용된 표준기 정보 및 소급성
- [ ] 측정 불확도(Uncertainty) 표기
- [ ] 합격/불합격 판정 (Pass/Fail 기준 적용 시)
- [ ] 교정원 서명

## 7. 교정 상태 라벨 관리

### 7.1 라벨 종류 및 색상

| 라벨 색상 | 상태 | 의미 | 사용 조건 |
|----------|------|------|----------|
| **녹색** | 합격 (CALIBRATED) | 교정 완료, 정상 사용 가능 | MPE 이내 |
| **황색** | 제한 사용 (LIMITED USE) | 특정 범위/조건에서만 사용 가능 | 일부 측정점 OOT |
| **적색** | 불합격 (DO NOT USE) | 사용 금지, 수리/재교정/폐기 필요 | 전체 OOT 또는 파손 |

### 7.2 라벨 기재 항목

모든 교정 라벨에는 다음 정보를 기재한다:

1. 장비 관리번호
2. 교정일
3. 차기 교정 예정일
4. 교정 수행자 (서명 또는 이니셜)
5. 제한 사용 라벨의 경우: 사용 가능 범위 명기

### 7.3 라벨 부착 위치

- 장비 본체의 **잘 보이는 위치**에 부착
- 기존 라벨 위에 중첩 부착 금지 (이전 라벨 제거 후 부착)
- 라벨 부착이 어려운 소형 장비는 **보관 케이스**에 부착

## 8. 교정 이력 기록 방법

### 8.1 기록 양식

교정 이력은 **QF-710-01 교정 이력 관리대장**에 기록한다.

### 8.2 기록 항목

| No. | 기록 항목 | 내용 |
|-----|----------|------|
| 1 | 장비 관리번호 | 자산관리 번호 체계에 따른 고유번호 |
| 2 | 장비명/모델 | 장비의 명칭 및 모델번호 |
| 3 | 시리얼 번호 | 제조사 부여 고유번호 |
| 4 | 사용 부서 | 장비 보유/사용 부서 |
| 5 | 교정 일자 | 교정 수행일 |
| 6 | 차기 교정일 | 교정 주기에 따른 다음 교정 예정일 |
| 7 | 교정 유형 | 내부/외부 |
| 8 | 교정기관명 | 외부 교정 시 기관명 |
| 9 | 교정 성적서 번호 | 성적서 고유번호 |
| 10 | 판정 결과 | 합격/제한/불합격 |
| 11 | 비고 | OOT 내역, 수리 이력 등 |

### 8.3 기록 보관

- 교정 이력 관리대장: 전자 파일(Excel) + 출력본 보관
- 교정 성적서 원본: 파일링 보관 (최소 **10년** 보관)
- 항공우주 제품 관련 교정 기록: **제품 수명 + 5년** 보관

## 9. 교정 만료 장비 처리

### 9.1 즉시 격리

1. 교정 만료가 확인된 장비는 **즉시 사용 중지**한다.
2. 적색 라벨("DO NOT USE")을 부착한다.
3. 해당 장비를 **격리 보관 구역**으로 이동한다.
4. 품질팀에 즉시 보고한다.

### 9.2 소급 조사 (Recall Investigation)

1. 해당 장비로 **마지막 교정 합격일 이후** 측정한 제품을 파악한다.
2. 영향받을 수 있는 제품의 **로트 번호** 및 **고객 출하 여부**를 확인한다.
3. 소급 조사 결과를 기록하고, 필요 시 다음 조치를 수행한다:
   - 재검사 실시
   - 고객 통보 (출하된 경우)
   - 부적합 보고서(NCR) 발행
4. 소급 조사 결과를 **품질팀장에게 보고**한다.

### 9.3 사후 조치

- 교정 가능: 교정 실시 후 합격 시 재사용
- 수리 필요: 수리 후 재교정
- 폐기: 수리 불가 판정 시 폐기 처리 (자산대장 말소)

## 10. OOT 발견 시 조치

### 10.1 OOT 판정

교정 결과 측정값이 **MPE를 초과**하는 경우 OOT로 판정한다.

### 10.2 즉시 조치 절차

```
OOT 판정
  │
  ├─ 1단계: 장비 즉시 사용 중지 → 적색 라벨 부착
  │
  ├─ 2단계: 품질팀장에 보고
  │
  ├─ 3단계: 소급 조사 실시
  │     ├─ 마지막 합격 교정일 ~ OOT 판정일 기간의 측정 제품 파악
  │     ├─ 영향 범위 결정 (로트, 고객, 수량)
  │     └─ 재검사 또는 고객 통보 필요 여부 결정
  │
  ├─ 4단계: 장비 처리 결정
  │     ├─ 조정(Adjustment) 후 재교정 → 합격 시 재사용
  │     ├─ 수리 의뢰 → 수리 후 재교정
  │     └─ 폐기 결정 → 자산대장 말소
  │
  └─ 5단계: 시정조치 보고서(CAR) 발행 (필요 시)
```

### 10.3 기록

- OOT 장비 관리번호, 교정일, OOT 편차량을 기록
- 소급 조사 결과 및 조치 내역을 기록
- 관련 기록을 QF-710-01에 비고란에 기재

## 11. 주의사항

1. **교정 만료 장비를 절대 사용하지 않는다.** 사용 전 반드시 교정 라벨의 유효 기간을 확인한다.
2. 측정장비는 사용 후 **지정된 보관장소**에 보관하며, 충격/오염/습기로부터 보호한다.
3. 내부교정에 사용하는 표준기(게이지블록 등)는 반드시 **외부 인정기관에서 교정**받은 것만 사용한다.
4. 교정 환경(온도, 습도)을 준수하지 않은 교정 결과는 **무효**로 처리한다.
5. 교정 라벨이 훼손되거나 분실된 장비는 **사용을 중지**하고 품질팀에 보고한다.
6. 디지털 장비의 배터리 부족 시 교정 결과에 영향을 줄 수 있으므로, 교정 전 **배터리 상태**를 확인한다.
7. 모든 교정 기록은 AS9100D 요구사항에 따라 **소급성(Traceability)**이 확보되어야 한다.
8. 교정 주기 변경 시 반드시 **품질팀장 승인**을 받고 변경 사유를 기록한다.
9. 외부 교정기관 선정 시 **ISO/IEC 17025 인정 여부**를 반드시 확인한다.
10. 항공우주 제품에 사용되는 측정장비는 일반 제품보다 **엄격한 관리**가 요구되므로, 전용 관리대장을 운영한다.
