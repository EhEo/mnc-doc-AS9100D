## 1. 목적

본 지침서는 (주)에이로텍의 외부공급자(공급업체)에 대한 심사 수행 방법을 구체적으로 규정하여, AS9100D 8.4.1 요구사항에 따라 외부공급자의 품질경영시스템 및 제품 실현 능력을 체계적으로 평가하고, 승인 공급업체 목록(ASL)의 신뢰성을 유지함을 목적으로 한다.

## 2. 적용범위

- (주)에이로텍에 원자재, 부품, 외주공정을 제공하는 모든 외부공급자에 적용한다.
- 항공우주 제품에 직접 사용되는 자재/부품/서비스 공급자를 우선 적용한다.
- 특수공정(열처리, 도금, NDT 등) 외주 공급자에게는 강화된 심사 기준을 적용한다.

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| ASL | Approved Supplier List, 승인 공급업체 목록 |
| SCAR | Supplier Corrective Action Request, 공급자 시정조치 요구서 |
| 심사원 | 공급자 심사를 수행할 자격을 갖춘 자 |
| 부적합(NC) | 요구사항에 적합하지 않은 상태 |
| 관찰사항(OBS) | 부적합은 아니나 개선이 권고되는 사항 |
| 특수공정 | 결과물을 사후 검증만으로 확인할 수 없는 공정 (용접, 열처리, 도금, NDT 등) |

## 5. 심사 유형

### 5.1 심사 유형 분류

| 심사 유형 | 목적 | 시기 | 비고 |
|----------|------|------|------|
| 신규 등재 심사 | 신규 공급자의 ASL 등재 적격성 평가 | ASL 등재 전 | 필수 |
| 정기 심사 | 기존 공급자의 지속적 적합성 확인 | 연 1회 (A등급) / 연 2회 (B등급) | ASL 등급별 차등 |
| 특별 심사 | 중대 품질 문제 발생 시 긴급 평가 | 문제 발생 시 | 품질팀장 승인 |
| 불시 심사 | 공급자 사전 통보 없이 실시하는 심사 | 필요 시 | 계약 조건에 명시 필요 |

### 5.2 심사 면제 조건
- OASIS 등록 공급자 (AS9100/AS9110/AS9120 인증 보유)
- 고객이 지정한 공급자 (단, 수입검사 강화 적용)
- 면제 시에도 납품 품질 실적 모니터링은 지속 실시

## 6. 심사 계획 수립

### 6.1 연간 심사 계획
- 매년 1분기에 연간 공급자 심사 계획을 수립한다.
- 계획 수립 시 고려사항:
  - ASL 등급별 심사 주기
  - 전년도 납품 품질 실적 (불합격률, 클레임 건수)
  - 공급 물량 및 중요도
  - 특수공정 공급자 여부

### 6.2 대상 선정 기준

| 우선순위 | 대상 | 근거 |
|----------|------|------|
| 1순위 | 특수공정 외주처 | AS9100D 8.4.1 강화 요건 |
| 2순위 | C/D등급 공급자 | 개선 필요 공급자 |
| 3순위 | 신규 등재 후보 | ASL 등재 전 필수 심사 |
| 4순위 | B등급 공급자 | 정기 심사 주기 도래 |
| 5순위 | A등급 공급자 | 정기 심사 주기 도래 |

### 6.3 심사 범위 결정
- 심사 범위는 공급 품목, 공정, 해당 QMS 조항을 기준으로 결정한다.
- 신규 등재 심사: 전체 QMS 범위
- 정기 심사: 주요 공정 + 이전 심사 부적합 사항 Follow-up
- 특별 심사: 문제 발생 공정/제품 집중

### 6.4 심사원 배정

| 자격 요건 | 내용 |
|----------|------|
| 심사원 교육 | 내부심사원 교육 수료 (AS9100D 기반) |
| 경력 | 품질 또는 생산 관련 업무 2년 이상 |
| 독립성 | 피심사 공급자와 이해관계 없는 자 |
| 전문성 | 해당 공급자의 공정/제품 분야 이해 |

## 7. 심사 준비

### 7.1 사전 정보 수집
1. 공급자 기본 정보 (회사 규모, 인증 현황, 주요 거래처)
2. 과거 심사 이력 및 부적합 사항
3. 납품 품질 실적 데이터 (불합격률, 납기 준수율, 클레임)
4. 관련 계약서 및 구매 사양서

### 7.2 체크리스트 작성

심사 체크리스트(QF-808-01)는 아래 영역을 포함한다.

| 심사 영역 | 주요 확인 항목 |
|----------|---------------|
| 품질경영시스템(QMS) | 품질방침, 품질목표, 문서관리, 내부심사, 경영검토 |
| 공정 관리 | 공정 흐름도, 작업표준서, 공정 파라미터 관리, 특수공정 인증 |
| 품질 실적 | 불합격률, 재작업률, 고객 클레임, 시정조치 이행률 |
| 납기 관리 | 납기 준수율, 생산 계획, 재고 관리 |
| 특수공정 | 공정 인증(Nadcap 등), 장비 교정, 작업자 자격 |
| 추적성 | 로트 관리, 자재 성적서, 기록 보관 |
| 변경 관리 | 공정/설비/자재 변경 시 통보 절차 |
| 이물질(FOD) 관리 | 작업환경 청결, FOD 방지 절차 |

### 7.3 심사 일정 통보
- 정기 심사: 심사일 최소 2주 전 서면 통보
- 특별 심사: 심사일 최소 3일 전 통보
- 불시 심사: 사전 통보 없음 (계약서에 근거 조항 확인)
- 통보 내용: 심사 목적, 범위, 일정, 심사원, 준비 요청 사항

## 8. 심사 수행

### 8.1 심사 프로세스

```
Opening Meeting → 현장 확인 → 기록 검토 → 인터뷰
    → 시정조치 확인 → 심사 결과 정리 → Closing Meeting
```

### 8.2 Opening Meeting
- 참석자: 심사원, 공급자 경영진/품질 책임자
- 심사 목적, 범위, 일정, 기준 설명
- 안전 사항, 보안 규정, 동행자 안내
- 소요 시간: 약 15~30분

### 8.3 현장 확인
- 공정 흐름에 따라 현장 순회
- 작업 환경(정리정돈, 청결, 안전, FOD 관리) 확인
- 설비 상태, 측정기 교정 상태 확인
- 작업표준서 비치 및 준수 여부 확인
- 제품 식별/추적성 관리 상태 확인
- 부적합품 격리 관리 확인

### 8.4 기록 검토
- 품질 기록(검사 성적서, 교정 기록, 교육 기록)
- 시정조치 기록 (이전 심사 부적합 조치 포함)
- 내부심사 및 경영검토 기록
- 변경 관리 기록
- 고객 클레임 처리 기록

### 8.5 인터뷰
- 관리자 및 현장 작업자 대상 인터뷰 실시
- 품질 방침/목표 인식 수준 확인
- 비상 시 대응 절차 인지 여부 확인
- 변경 관리 프로세스 이해도 확인

### 8.6 시정조치 확인 (Follow-up)
- 이전 심사에서 발행된 부적합 사항의 시정조치 이행 상태 확인
- 시정조치의 효과성 검증 (재발 여부, 데이터 확인)
- 미이행 항목은 심사 보고서에 재기록

### 8.7 Closing Meeting
- 심사 결과 요약 발표
- 부적합 사항 및 관찰사항 설명
- 시정조치 요구 기한 안내
- 공급자 의견 청취 및 합의
- 소요 시간: 약 30분

## 9. 심사 채점 기준

### 9.1 채점 체계

| 점수 범위 | 등급 | 판정 | 조치 |
|----------|------|------|------|
| 90~100점 | A (우수) | 합격 | 정기 심사 주기: 연 1회 |
| 80~89점 | B (양호) | 조건부 합격 | 경미 부적합 시정 후 유지, 정기 심사: 연 2회 |
| 70~79점 | C (보통) | 조건부 합격 | 시정조치 계획 제출 필수, 3개월 내 Follow-up |
| 60~69점 | D (미흡) | 보류 | 중대 시정조치 필요, 거래 축소 검토 |
| 60점 미만 | F (부적격) | 불합격 | ASL 삭제 또는 거래 중단 |

### 9.2 채점 세부 기준

| 심사 영역 | 배점 | 평가 내용 |
|----------|------|----------|
| 품질경영시스템 | 20점 | QMS 구축 및 운영 수준 |
| 공정 관리 | 25점 | 공정 관리 체계, 작업표준, 설비 관리 |
| 품질 실적 | 20점 | 불합격률, 클레임, 시정조치 이행 |
| 납기 관리 | 15점 | 납기 준수율, 생산 계획 적정성 |
| 추적성/기록 관리 | 10점 | 로트 추적, 기록 보관, 문서 관리 |
| 작업 환경/안전 | 10점 | 정리정돈, 안전, FOD 관리 |

### 9.3 부적합 감점 기준

| 부적합 유형 | 감점 |
|------------|------|
| 중대 부적합 (Major NC) | 해당 영역 배점의 50% 이상 감점 |
| 경미 부적합 (Minor NC) | 해당 영역 배점의 20~30% 감점 |
| 관찰사항 (OBS) | 감점 없음 (개선 권고) |

## 10. 심사 보고서 작성

### 10.1 보고서 구성

심사 보고서(QF-808-02)에는 아래 내용을 포함한다.

| 섹션 | 내용 |
|------|------|
| 심사 개요 | 공급자명, 심사일, 심사 유형, 심사원, 심사 범위 |
| 심사 결과 요약 | 총점, 등급, 판정 |
| 부적합 사항 | NC 번호, 유형(Major/Minor), 해당 조항, 부적합 내용, 객관적 증거 |
| 관찰사항 | 개선 권고 사항 |
| 강점 | 공급자의 우수 사례 |
| 약점 | 개선이 필요한 영역 |
| 이전 심사 Follow-up | 이전 부적합 시정조치 이행 상태 |
| 종합 판정 | ASL 유지/조건부/삭제 권고 |
| 첨부 | 체크리스트, 사진, 증빙 자료 |

### 10.2 보고서 제출 기한
- 심사 완료 후 5영업일 이내 심사 보고서 작성 완료
- 품질팀장 검토 및 승인 후 공급자에게 전달

## 11. 사후 관리

### 11.1 시정조치 요구
- 부적합 사항에 대해 SCAR(QF-808-03) 발행
- Major NC: 30일 이내 시정조치 완료 요구
- Minor NC: 60일 이내 시정조치 완료 요구
- 시정조치 내용: 원인 분석, 시정 조치, 재발 방지 대책

### 11.2 Follow-up 심사
- Major NC 발생 공급자: 시정조치 완료 후 3개월 이내 Follow-up 심사 실시
- Follow-up 범위: 부적합 사항 시정조치 이행 및 효과성 확인
- Follow-up 결과에 따라 ASL 등급 재평가

### 11.3 ASL 등급 반영
- 심사 결과를 ASL(QF-808-04)에 반영한다.
- 등급 변경 시 구매 부서 및 관련 부서에 통보한다.
- D등급 이하 공급자는 대체 공급자 확보 계획을 수립한다.

| 심사 등급 | ASL 조치 |
|----------|---------|
| A | ASL 유지, 우대 적용 가능 |
| B | ASL 유지, 모니터링 강화 |
| C | ASL 유지, 시정조치 이행 확인 후 |
| D | ASL 보류, 거래 축소 |
| F | ASL 삭제 또는 거래 중단 |

## 12. 원격 심사(Remote Audit) 수행 방법

### 12.1 원격 심사 적용 조건
- 코로나-19 등 불가항력으로 현장 방문이 불가한 경우
- 지리적으로 원격지에 위치하여 현장 심사 비용이 과도한 경우
- 정기 심사 중 경미한 범위의 Follow-up 심사
- 품질팀장 승인 필요

### 12.2 원격 심사 방법

| 항목 | 방법 |
|------|------|
| 화상 회의 | Zoom, Teams 등 활용 (카메라 필수) |
| 현장 확인 | 공급자 담당자가 카메라로 현장 실시간 중계 |
| 기록 검토 | 사전 문서 송부 또는 화면 공유 |
| 증빙 | 스크린샷, 사진, 동영상 기록 |

### 12.3 원격 심사 제한사항
- 신규 등재 심사는 원격 심사 단독으로 수행 불가 (현장 심사 병행 필수)
- 특수공정 공급자는 현장 심사를 원칙으로 한다.
- 원격 심사 시 확인이 불충분한 항목은 "확인 불가"로 기록하고, 현장 심사 시 재확인

### 12.4 원격 심사 기록
- 화상 회의 녹화본 보관 (공급자 동의 하)
- 심사 보고서에 "원격 심사" 명시
- 확인 제한 항목 별도 기록

## 13. 주의사항

1. 심사 시 객관적 증거(기록, 데이터, 현장 확인)에 근거하여 판정한다. 주관적 판단 금지.
2. 심사원은 피심사 공급자로부터 금품, 접대를 수수하지 않는다.
3. 공급자의 영업 비밀에 해당하는 정보는 비밀 유지 의무를 준수한다.
4. 심사 중 발견된 안전 위험 사항은 즉시 공급자에게 통보한다.
5. 심사 결과는 관련 부서(구매, 품질, 기술) 간 공유하여 공급자 관리에 활용한다.
6. 심사 체크리스트는 AS9100D 개정 또는 고객 요구사항 변경 시 갱신한다.
7. 심사 기록은 최소 7년간 보관한다 (항공우주 규정 요건).
8. 모든 심사원은 연 1회 이상 심사원 역량 강화 교육을 이수한다.
9. 공급자가 심사에 비협조적인 경우 ASL 등급 하향 조정을 검토한다.
10. 고객 또는 인증기관의 공급자 심사 동행 요구 시 적극 협조한다.
