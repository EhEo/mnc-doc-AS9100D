## 1. 목적

본 지침서는 (주)에이로텍(M&C Electronics VINA)에서 생산된 제품의 최종검사(OQC) 및 출하 승인 절차를 구체적으로 규정하여, 고객에게 인도되는 제품의 품질을 최종 확인하고 AS9100D 8.6 요구사항을 충족함을 목적으로 한다.

## 2. 적용범위

- 본 지침서는 출하 전 모든 완성 제품의 최종검사에 적용한다.
- 사내 제조 완성품 및 외주가공 후 자사 출하 제품을 포함한다.
- 시제품(Prototype) 및 양산품 모두 적용한다.

## 5. 최종검사 전 준비

### 5.1 공정 이력 확인

최종검사 착수 전 다음 사항을 확인한다:

1. **Lot Traveler 완결성 확인**
   - 모든 공정의 작업자 서명이 완료되었는지 확인
   - 누락된 공정이 없는지 확인
   - 특수공정(열처리, 도금, 용접 등)의 인증서/성적서 첨부 확인
   - 공정 이관검사 합격 기록 확인

2. **선행 검사 기록 확인**
   - 수입검사(IQC) 합격 기록 확인 (WI-806-01 참조)
   - 공정검사(PQC) 합격 기록 확인 (WI-807-01 참조)
   - 부적합 발생 이력 및 처리 완료 여부 확인
   - 재작업(Rework) 수행 시 재검사 합격 기록 확인

3. **도면 및 사양 확인**
   - 최신 개정 도면 확보 (개정번호 대조)
   - 고객 특별요구사항(Special Requirements) 확인
   - ECN(설계변경통지서) 반영 여부 확인

### 5.2 검사 환경 준비

1. 검사 작업대 청결 상태 확인 (FOD 방지)
2. 조명 조건 확인 (외관검사: 최소 500 lux)
3. 교정된 측정장비 준비 및 교정 유효 기간 확인
4. 검사 지그/치구 상태 확인

## 6. 최종검사 수행

### 6.1 외관검사

| 검사 항목 | 기준 | 방법 |
|----------|------|------|
| 표면 결함 | 한도 견본 기준 (스크래치, 찍힘, 버) | 육안, 확대경 (필요 시 10x) |
| 도금/도장 상태 | 박리, 기포, 변색 없음 | 육안 |
| 마킹/각인 | 품번, 로트번호, 제조일 정확성 | 육안 |
| 이물질(FOD) | 이물질 부착 없음 | 육안, 필요 시 에어건 |
| 부식/산화 | 부식, 산화 흔적 없음 | 육안 |

**외관검사 절차:**
1. 제품을 검사대 위에 놓고 전체 표면을 확인한다.
2. 고객 도면의 외관 기준 또는 한도 견본과 대조한다.
3. 결함 발견 시 결함 유형과 위치를 기록한다.
4. 합격/불합격을 판정한다.

### 6.2 치수검사

1. 도면에 명시된 **모든 치수 항목**을 측정한다.
   - 일반 공차 항목: 샘플링 검사 가능 (AQL 기준)
   - KC (핵심특성) 항목: **100% 전수검사**
2. 기하공차(GD&T) 항목 측정:
   - 위치도, 진원도, 평면도, 직각도 등
   - CMM 사용 시 측정 프로그램 확인
3. 측정 결과를 최종검사 성적서(F-806-01)에 기록한다.

### 6.3 기능시험

해당되는 경우 다음 기능시험을 실시한다:

| 시험 유형 | 적용 제품 | 기준 |
|----------|----------|------|
| 조립 적합성 | 조립 부품 | 결합 부위 간섭 없음, 정상 조립 |
| 전기적 시험 | 전자부품/커넥터 | 저항, 절연, 내전압 규격 충족 |
| 기밀 시험 | 유압/공압 부품 | 누설률 규격 이내 |
| 작동 시험 | 기구부품 | 정상 작동, 반복 시험 |
| 토크 시험 | 체결부품 | 규정 토크 범위 이내 |

### 6.4 포장상태 확인

1. 포장 사양서에 따른 포장 방법 준수 여부 확인
2. 제품 보호 상태 (완충재, 방청 처리, 정전기 방지 등)
3. 라벨 정보 정확성 (품번, 수량, 로트번호, 중량)
4. 항공우주 제품 전용 포장 요구사항 준수 여부

### 6.5 CoC 발행 기준

다음 조건을 **모두** 충족할 때 CoC를 발행한다:

1. 최종검사 전 항목 합격
2. Lot Traveler 전 공정 서명 완료
3. 모든 선행 검사(IQC, PQC) 합격 기록 확인
4. 부적합 이력이 있는 경우, 처리 완료 확인
5. 특수공정 인증서/성적서 첨부 완료
6. 고객 특별요구사항 충족 확인

## 7. 출하 승인 프로세스

### 7.1 표준 출하 승인 흐름

```
최종검사 전 항목 합격
  │
  ├─ 1단계: 검사원이 최종검사 성적서(F-806-01) 작성
  │
  ├─ 2단계: 검사원이 AAM(합격 날인/스탬프) 날인
  │     ├─ 검사 합격 스탬프를 제품 또는 포장에 날인
  │     ├─ Lot Traveler에 최종검사 합격 서명
  │     └─ 검사 날인은 인가된 검사원만 수행
  │
  ├─ 3단계: 품질팀장 출하 승인
  │     ├─ 검사 기록 일체를 검토
  │     ├─ CoC에 승인 서명
  │     └─ 출하 승인 날인
  │
  └─ 4단계: 출하
        ├─ 포장 최종 확인
        ├─ 출하 서류 준비 (CoC, 성적서, 납품서)
        └─ 물류팀에 출하 인도
```

### 7.2 AAM (Authorized Acceptance Media) 관리

| 항목 | 내용 |
|------|------|
| AAM 유형 | 검사 합격 스탬프 (고유번호 부여) |
| 인가자 | 품질팀장이 지정한 검사원 |
| 관리 | 개인별 AAM 스탬프 배정, 타인 사용 금지 |
| 보관 | 사용 후 잠금장치 보관, 분실 시 즉시 보고 |
| 폐기 | 퇴직/이동 시 즉시 회수 및 폐기 기록 |

### 7.3 출하 서류 구성

| No. | 서류 | 필수 여부 | 비고 |
|-----|------|----------|------|
| 1 | CoC (적합성 인증서) | 필수 | 항공우주 제품 필수 |
| 2 | 최종검사 성적서 (F-806-01) | 필수 | 실측 데이터 포함 |
| 3 | 납품서/거래명세서 | 필수 | |
| 4 | FAI 보고서 (AS9102) | 해당 시 | 초도품 납품 시 |
| 5 | 특수공정 성적서 | 해당 시 | 열처리, 도금 등 |
| 6 | 자재증명서 (Mill Cert) | 해당 시 | 고객 요구 시 |
| 7 | SPC 데이터 | 해당 시 | 고객 요구 시 |
| 8 | Source Inspection 기록 | 해당 시 | 고객 입회검사 시 |

## 8. 불합격 시 처리

### 8.1 불합격 판정

최종검사에서 다음 중 하나라도 해당되면 불합격으로 판정한다:

1. 치수 측정값이 도면 공차 초과
2. 외관 결함이 합격 기준 초과
3. 기능시험 요구사항 미달
4. Lot Traveler 불완전 (공정 누락, 서명 미비)
5. 선행 검사 기록 누락 또는 부적합 미처리

### 8.2 처분 결정 흐름

```
최종검사 불합격
  │
  ├─ NCR(부적합 보고서) 발행
  │
  ├─ 품질팀장 주관 처분 검토 회의
  │     ├─ 결함 유형 및 심각도 평가
  │     ├─ 재작업 가능 여부 판단
  │     └─ 고객 영향 분석
  │
  ├─ 처분 결정
  │     ├─ 재작업 (Rework)
  │     │     ├─ 승인된 재작업 절차에 따라 수행
  │     │     ├─ 재작업 후 최종검사 재실시 (전수)
  │     │     └─ 재작업 이력을 Lot Traveler에 기록
  │     │
  │     ├─ 특채 (Concession)
  │     │     ├─ 기술적 검토 (설계팀 확인)
  │     │     ├─ 고객 사전 승인 획득 (항공우주 필수)
  │     │     ├─ 특채 승인서 작성 및 보관
  │     │     └─ CoC에 특채 사항 명기
  │     │
  │     └─ 폐기 (Scrap)
  │           ├─ 폐기 결정서 작성
  │           ├─ 물리적 폐기 실시 (재사용 방지)
  │           └─ 폐기 수량 기록
  │
  └─ 원인 분석 및 시정조치
        ├─ 근본원인 분석 (5Why, 특성요인도 등)
        ├─ 시정조치 실시
        └─ 유효성 확인
```

### 8.3 재작업 후 재검사

1. 재작업은 **승인된 재작업 지시서**에 따라 수행한다.
2. 재작업 완료 후 **최종검사를 전수(100%) 재실시**한다.
3. 재검사 결과를 별도 기록하고, 재작업 이력을 명기한다.
4. 동일 제품의 재작업 횟수는 **최대 2회**로 제한한다.
5. 2회 재작업 후에도 부적합 시 **폐기** 처리한다.

## 9. 고객 요구 특별검사

### 9.1 Source Inspection (고객 입회검사) 대응

| 단계 | 활동 | 담당 |
|------|------|------|
| 1. 사전 통보 | 고객에게 검사 준비 완료 통보 (최소 3영업일 전) | 영업팀 |
| 2. 일정 조율 | 고객 검사원 방문 일정 확정 | 영업팀 + 품질팀 |
| 3. 검사 준비 | 검사 환경 정비, 서류 준비, 장비 확인 | 품질팀 |
| 4. 입회검사 수행 | 고객 검사원 입회 하에 검사 실시 | 품질 검사원 |
| 5. 검사 결과 확인 | 고객 검사원의 합격/불합격 판정 확인 | 품질팀 |
| 6. Buy-off 획득 | 고객 검사원 서명/날인 획득 | 품질팀 |
| 7. 기록 보관 | Source Inspection 기록 보관 | 품질팀 |

### 9.2 Source Inspection 준비 사항

1. **서류 준비**:
   - 최종검사 성적서 (실측 데이터 포함)
   - Lot Traveler (전 공정 기록)
   - 특수공정 인증서
   - 자재증명서 (해당 시)
   - FAI 보고서 (초도품 시)

2. **현장 준비**:
   - 검사 공간 정리 및 청결 유지
   - 해당 제품 및 샘플 준비
   - 교정된 측정장비 배치
   - 도면 및 사양서 비치

3. **인원 준비**:
   - 담당 검사원 배치
   - 통역 지원 (필요 시)
   - 기술 문의 대응 인원 대기

### 9.3 고객 추가 요구사항 대응

- 고객이 별도 검사 항목을 요구하는 경우 **PO 또는 도면**에 명시된 요구사항을 확인한다.
- 추가 시험(파괴시험, 수명시험 등) 요구 시 시험 일정 및 비용을 사전 협의한다.
- 검사 기준 변경 요청 시 **기술적 타당성을 검토**한 후 합의한다.

## 10. 최종검사 성적서 및 CoC 작성 방법

### 10.1 최종검사 성적서 (F-806-01)

#### 기록 항목

| No. | 항목 | 내용 |
|-----|------|------|
| 1 | 검사일자 | 최종검사 수행 일자 |
| 2 | 제품명/품번 | 제품 식별 정보 |
| 3 | 도면번호/개정 | 적용 도면 및 최신 개정 확인 |
| 4 | 로트번호 | 생산 로트 번호 |
| 5 | 생산수량/검사수량 | 전체 수량 / 검사 수량 |

#### 작성 시 유의사항

1. 실측값은 **수기 또는 전자 데이터**로 정확히 기록한다.
2. 측정 단위를 명확히 표기한다 (mm, μm, kg 등).
3. 불합격 항목이 있는 경우 해당 셀을 **적색으로 표시**한다.
4. 성적서 번호를 부여하여 추적성을 확보한다.
5. 수정 시 **이중 취소선 후 정정값 기입**, 정정자 서명을 한다. (수정액 사용 금지)

## 11. 주의사항

1. **최종검사를 완료하지 않은 제품은 절대 출하할 수 없다.** 모든 출하 제품은 검사 합격 확인 후 출하한다.
2. AAM(합격 날인)은 **인가된 검사원**만 사용할 수 있으며, 타인에게 대여하거나 위임할 수 없다.
3. Lot Traveler가 불완전한 상태(공정 누락, 서명 미비)에서는 **최종검사를 개시하지 않는다.**
4. CoC는 **허위 기재를 절대 금지**한다. 허위 CoC 발행은 항공우주 산업에서 중대한 법적 책임을 수반한다.
5. 특채(Concession) 제품은 반드시 **고객 사전 승인**을 획득한 후 출하한다.
6. 재작업 제품은 반드시 **최종검사를 전수(100%) 재실시**한다. 샘플링 검사로 대체할 수 없다.
7. 측정장비는 반드시 **교정 유효기간 내** 장비만 사용한다.
8. Source Inspection 대상 제품은 고객 검사원의 **Buy-off 없이 출하할 수 없다.**
9. 최종검사 성적서의 **수정 시 수정액을 사용하지 않는다.** 이중 취소선 후 정정하고 서명한다.
10. 검사 결과의 **위변조 및 조작은 엄격히 금지**하며, 적발 시 징계 및 법적 조치를 취한다.
11. 포장 상태가 고객 요구사항에 미달하는 경우 **재포장 후 포장상태를 재확인**한다.
12. 모든 최종검사 기록은 **추적성(Traceability)**이 확보되어야 하며, 로트번호 및 CoC 번호와 연계 관리한다.
