## 1. 목적

본 지침서는 (주)에이로텍에서 사용하는 화학물질 중 REACH, RoHS 등 국제 환경 규제에 해당하는 위험물질의 관리 방법을 규정하여, AS9100D 8.1.4(제품 안전) 요구사항을 충족하고 작업자 안전 및 환경 보호를 확보함을 목적으로 한다.

## 2. 적용범위

- (주)에이로텍 베트남 사업장에서 사용, 보관, 취급하는 모든 화학물질에 적용한다.
- 원자재에 포함된 화학물질, 공정 중 사용하는 보조재(세정제, 접착제, 윤활유, 도금액 등) 포함
- 완제품에 잔류할 수 있는 화학물질의 규제 적합성 관리 포함
- 외부공급자로부터 입고되는 화학물질 포함

## 4. 용어정의

| 용어 | 정의 |
|------|------|
| REACH | Registration, Evaluation, Authorization and Restriction of Chemicals. EU 화학물질 등록, 평가, 허가 및 제한 규정 |
| RoHS | Restriction of Hazardous Substances. 전기전자제품 내 유해물질 사용 제한 지침 |
| SVHC | Substance of Very High Concern. 고위험 우려 물질 (ECHA 후보 목록에 등재) |
| SDS | Safety Data Sheet. 물질안전보건자료 (16개 항목 구성) |
| IMDS | International Material Data System. 국제 자재 데이터 시스템 |
| CMRT | Conflict Minerals Reporting Template. 분쟁광물 보고 양식 |
| CAS No. | Chemical Abstracts Service 번호. 화학물질 고유 등록번호 |
| 함유율 | 제품 총 중량 대비 해당 물질의 중량 비율 (ppm 또는 %) |

## 5. 위험물질 목록 관리

### 5.1 자사 사용 화학물질 목록 작성

화학물질 관리대장(QF-815-01)에 아래 항목을 기재하여 관리한다.

| 관리 항목 | 내용 |
|----------|------|
| 물질명 | 상품명 및 화학명 |
| CAS No. | 화학물질 고유 번호 |
| 용도 | 사용 공정 및 용도 |
| 사용량 | 월간 사용량 |
| 공급자 | 화학물질 공급업체 |
| SDS 보유 여부 | 최신 SDS 보유 확인 |
| 규제 해당 여부 | REACH/RoHS/SVHC 해당 여부 |
| 대체물질 검토 | SVHC 해당 시 대체물질 가용성 |

### 5.2 규제물질 해당 여부 확인

| 규제 | 확인 방법 | 주기 |
|------|----------|------|
| RoHS 10대 제한물질 | 납(Pb), 수은(Hg), 카드뮴(Cd), 6가 크롬(Cr6+), PBB, PBDE, DEHP, BBP, DBP, DIBP 함유 여부 | 신규 물질 도입 시 + 연 1회 |
| REACH SVHC | ECHA 후보 목록 대비 확인 (0.1wt% 초과 여부) | ECHA 목록 갱신 시 (연 2회) |
| 고객 제한 물질 | 고객별 제한물질 목록 대비 확인 | 고객 요구 시 |

### 5.3 목록 갱신
- ECHA SVHC 후보 목록 갱신 시 (통상 연 2회, 6월/12월) 화학물질 관리대장 재검토
- 신규 화학물질 도입 전 반드시 규제 해당 여부 사전 확인
- 관련 법규 변경 시 환경안전 담당자가 목록 갱신 및 관련 부서 통보

## 6. 입고 시 확인

### 6.1 SDS 확인
1. 화학물질 입고 시 공급자로부터 최신 SDS를 수령한다.
2. SDS 언어: 영어 또는 베트남어 (필요 시 한국어 번역본 병행)
3. SDS 16개 항목 완비 여부 확인:
   - 2항(위험 유해성), 3항(구성성분), 7항(취급 및 저장), 8항(노출방지), 11항(독성), 15항(법적규제)
4. SDS가 3년 이상 경과한 경우 공급자에게 갱신 요청

### 6.2 RoHS 적합 인증서 확인
- 전기전자 부품/소재 입고 시 RoHS 적합 인증서(또는 성적서) 수령
- 인증서에 10대 제한물질 항목별 함유량(ppm) 명시 여부 확인
- RoHS 기준 초과 시 입고 거부 및 공급자 통보

### 6.3 성분 분석 데이터
- 고객 요구 시 또는 규제 해당 의심 물질에 대해 성분 분석 데이터(ICP, XRF 등) 확인
- 분석 데이터의 시험기관 인정(ISO 17025) 여부 확인
- 분석 데이터 미비 시 자사 또는 외부 시험기관에 분석 의뢰

### 6.4 입고 검증 기록
- 위험물질 입고 검증 체크리스트(QF-815-02) 작성
- SDS, 인증서, 분석 데이터를 화학물질 관리대장에 연결 보관

## 7. 보관 및 취급

### 7.1 전용 보관소 요건

| 요건 | 세부 사항 |
|------|----------|
| 위치 | 생산 구역과 분리된 전용 보관소 |
| 환기 | 강제 환기 설비 (환기 횟수: 시간당 최소 6회) |
| 온도/습도 | SDS 지정 조건 준수 (일반: 15~25°C, 습도 60% 이하) |
| 바닥 | 내약품성 바닥재, 방류턱(Bund) 설치 |
| 소화설비 | 화학물질 종류별 적합한 소화기 비치 |
| 표시 | 위험물 경고 표지판, GHS 라벨 부착 |
| 출입 | 관계자 외 출입 제한, 출입 기록 관리 |

### 7.2 이중 용기
- 액체 화학물질은 이중 용기(Secondary Containment)에 보관
- 이중 용기 용량: 최대 보관 용기의 110% 이상
- 용기 재질: 보관 물질과 화학적 호환성 확인

### 7.3 분리 보관
- 상호 반응 위험이 있는 물질은 분리 보관 (산류/알칼리류/산화제/가연물 분리)
- SDS 7항(취급 및 저장)의 혼합 위험 정보 참조
- 분리 보관 구역 표시 및 물질 목록 게시

### 7.4 취급 시 개인보호구(PPE)

| 물질 종류 | 필수 PPE |
|----------|---------|
| 유기용제 | 방독 마스크, 내화학 장갑, 보안경, 방호복 |
| 산/알칼리 | 전면형 마스크, 내산 장갑, 고글, 앞치마 |
| 분진 발생 물질 | 방진 마스크, 장갑, 보안경 |
| 일반 화학물질 | 니트릴 장갑, 보안경 |

### 7.5 재고 관리
- 선입선출(FIFO) 원칙 적용
- 유효기간 관리 (유효기간 경과 물질은 사용 금지, 폐기 절차 적용)
- 월 1회 재고 실사 및 관리대장 대비 확인

## 8. 공정 중 사용 관리

### 8.1 사용량 관리
- 공정에 필요한 최소량만 반출하여 사용
- 사용량을 기록하여 관리대장에 반영
- 잔량은 원래 보관소에 반납

### 8.2 혼합 금지
- SDS에 명시된 혼합 금지 물질과 접촉하지 않도록 관리
- 용기 간 물질 혼합 금지 (전용 용기 사용)

### 8.3 작업 환경 관리
- 화학물질 사용 구역의 환기 상태 유지
- 작업장 내 SDS 요약본(위험/대응 정보) 비치
- 누출 대응 키트(흡착재, 중화제, 보호구) 비치

## 9. 폐기 절차

### 9.1 폐기 분류
- 폐화학물질을 일반 폐기물과 분리 수거
- 위험 폐기물 분류 기준: 현지 환경법규(베트남 환경보호법, Decree 08/2022/ND-CP) 준수
- 폐기물 종류별 전용 수거 용기 사용, 라벨 부착

### 9.2 위탁 처리
- 허가된 위험 폐기물 처리 업체에 위탁
- 위탁 처리 업체의 허가증 유효 여부 연 1회 확인
- 폐기물 인계 시 인수인계서(Manifest) 작성 및 보관

### 9.3 기록 관리
- 폐기물 종류, 수량, 처리 업체, 처리 일자, 처리 방법을 기록
- 폐기 기록은 최소 5년간 보관
- 관할 환경 당국 보고 요건 준수

## 10. 고객 요구 규제물질 보고

### 10.1 IMDS(International Material Data System) 제출
- 자동차/항공우주 고객 요구 시 IMDS 시스템에 자재 데이터 등록
- 제품에 사용된 모든 물질의 구성 정보 입력
- 공급자로부터 수집한 SDS/성분 데이터를 기반으로 작성

### 10.2 CMRT(Conflict Minerals Reporting Template) 제출
- 고객 요구 시 분쟁광물(주석, 탄탈, 텅스텐, 금 — 3TG) 보고서 작성
- 공급망 내 분쟁광물 원산지 추적
- RMI(Responsible Minerals Initiative) 양식 사용

### 10.3 고객별 제한물질 보고
- 고객이 자체 제한물질 목록(RSL/DSL)을 요구하는 경우 해당 목록 대비 적합성 확인
- 적합성 선언서(Declaration of Conformity) 작성 및 제출
- 부적합 발견 시 즉시 고객에게 통보하고 대체 방안 협의

### 10.4 보고 주기
- SVHC 목록 갱신 시 해당 물질 사용 여부 재검토 및 고객 통보
- 고객 요구에 따른 정기 보고 (통상 연 1회 또는 제품 변경 시)

## 11. 비상 대응

### 11.1 누출 시 대응

```
누출 발견 → 즉시 작업 중단 → 누출 구역 출입 통제
    → PPE 착용 후 누출원 차단
    → 흡착재/중화제로 누출물 수거
    → 오염 구역 제독/세척
    → 비상 대응 기록서(QF-815-03) 작성
    → 관리자/환경안전 담당 보고
```

### 11.2 노출 시 응급 처치
- SDS 4항(응급 처치 요령)에 따라 즉시 조치
- 주요 응급 조치:

| 노출 경로 | 응급 조치 |
|----------|----------|
| 피부 접촉 | 오염 의복 제거, 흐르는 물로 15분 이상 세척 |
| 눈 접촉 | 눈 세척기로 15분 이상 세척, 콘택트 렌즈 제거 |
| 흡입 | 신선한 공기가 있는 곳으로 이동, 호흡 곤란 시 산소 공급 |
| 섭취 | 구토 유발 금지(SDS 지시 확인), 즉시 의료기관 이송 |

### 11.3 보고 체계

```
발견자 → 현장 관리자 → 환경안전 담당 → 공장장
                                    → (중대 사고 시) 관할 당국 신고
```

### 11.4 비상 장비 배치
- 화학물질 보관소 및 사용 구역에 비상 장비 비치:
  - 눈 세척기(Eye Wash Station)
  - 비상 샤워기(Emergency Shower)
  - 누출 대응 키트(Spill Kit)
  - 소화기 (물질별 적합 종류)
  - 응급 처치 키트
- 비상 장비 위치를 작업장 내 표시하고, 월 1회 점검

## 12. 교육 및 훈련

### 12.1 교육 대상 및 주기

| 대상 | 교육 내용 | 주기 |
|------|----------|------|
| 화학물질 취급자 | SDS 읽는 법, PPE 사용법, 누출 대응, 보관 규정 | 연 1회 + 신규 배치 시 |
| 관리 감독자 | 규제물질 관리, 보고 체계, 비상 대응 지휘 | 연 1회 |
| 전 사원 | 위험물질 기본 인식, 비상 대피 | 연 1회 |

### 12.2 훈련
- 누출 대응 모의 훈련: 연 1회 이상 실시
- 훈련 결과 기록 및 개선사항 반영

## 13. 주의사항

1. SDS가 없는 화학물질은 사용을 금지한다.
2. 화학물질 용기에서 라벨을 제거하거나 훼손하지 않는다.
3. 식음료 용기에 화학물질을 보관하지 않는다.
4. 화학물질 보관소에서 음식물 섭취를 금지한다.
5. 화학물질 취급 후 반드시 손을 세척한다.
6. 유효기간이 경과한 화학물질은 즉시 격리하고 폐기 절차에 따라 처리한다.
7. 신규 화학물질 도입 시 반드시 환경안전 담당자의 사전 승인을 받는다.
8. SVHC 해당 물질은 가능한 대체물질로 전환을 추진한다.
9. 규제물질 관련 고객 문의에 대해 5영업일 이내에 회신한다.
10. 본 지침서에 명시되지 않은 화학물질 관련 사안은 환경안전 담당자에게 문의한다.
11. 위험물질 관련 법규 변경 사항을 지속적으로 모니터링하고 반영한다.
12. 모든 화학물질 관련 기록은 최소 보관 기간을 준수하여 관리한다.
