# M&C AS9100D 인증 준비 프로젝트

**회사**: M&C (베트남 사업장) | **목표 인증**: AS9100D Rev D | **인증 완료 목표**: 2026년 9월 중순

---

## 문서 위험등급 안내

Document Master List의 모든 문서에는 **위험등급**이 부여되어 있습니다.
위험등급은 **"해당 문서가 누락되거나 부적절하게 작성될 경우 AS9100D 인증에 미치는 영향도"** 를 기준으로 분류됩니다.

### 프로젝트 관리 관점에서의 실질적 의미

```text
🔴 고위험 → AS9100D 전문 컨설턴트 지원 + 5/22까지 완료 필수
              (14건 중 12건이 P1 — 인증 직접 위협)

🟡 중위험 → 내부 작성 가능, 교차 검토 및 부서 간 협력 필요
              (집중 관리 시 일정 내 완료 가능)

🟢 저위험 → 기존 ISO/IATF 베트남어 문서 번역·변환으로 처리 가능
              (일정 내 완료 가능)
```
