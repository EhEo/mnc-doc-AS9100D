## Project Overview

**M&C AS9100D Rev D 인증 준비 프로젝트** - 중소 항공우주 제조기업의 AS9100D 신규 인증 취득을 위한 문서 관리 저장소. 기준일 2026-03-24, 목표 인증 완료 2026년 9월 중순.

- **인증 규격**: AS9100D Rev D (항공우주 품질경영시스템, ISO 9001 기반 확장)
- **회사**: M&C (베트남 사업장 운영)
- **크리티컬 패스**: 갭분석 → 문서화(9주) → 내부심사 → 경영검토 → Stage 1 → Stage 2 → 인증서 발급

## Architecture & Key Concepts

### 문서 계층 구조 (4-Tier)
- **1계층**: 품질매뉴얼 1건 (QM-001) - AS9100D 4~10 전 조항 포괄
- **2계층**: 절차서 30건 (QP-xxx) - 조항별 운영 절차
- **3계층**: 지침서 15건 (WI-xxx) + 양식/기록 21건 (QF-xxx)
- 문서번호 체계: `QP-{조항}{순번}` (예: QP-801 = 8.1 운용 기획)

### AS9100D 항공우주 특수 요구사항 (고위험 14건 - 최우선)
ISO 9001에는 없는, 항공우주 고유 요건으로 반드시 별도 문서화 필요:
- **8.1.2** 운영 리스크 관리 (FMEA 등)
- **8.1.3** 형상관리 (Configuration Management, CCB)
- **8.1.4** 제품/서비스 안전
- **8.3** 설계/개발 (DVP&R)
- **8.4.1** 외부공급자/ASL (승인 공급업체 목록)
- **8.5.1.2** FAI (First Article Inspection, AS9102 준거)
- **8.5.2** 식별/추적성 (로트 추적)
- **FOD** 이물질 손상 방지

### 기존 문서 활용
`ISO. IATF 문서/` 폴더에 기존 ISO 9001/IATF 16949 문서가 베트남어로 보관됨. AS9100D 절차서 작성 시 재활용 가능한 기반 문서. 폴더명 번호(1~46)는 기존 분류 번호이며 AS9100D 문서번호 체계(QP-xxx)와는 별도.

## Working with This Repository

### 문서 편집 규칙
- 관리 문서(01~04 폴더)는 모두 **Markdown 형식**
- 문서 현황 변경 시 `02_문서관리/Document_Master_List.md`의 상태 컬럼 동시 업데이트
- CAR 조치 완료 시 `03_실행계획/CAR_목록_시정조치계획.md`의 상태 컬럼 업데이트
- `04_대시보드/`는 다른 문서들의 통합 뷰 → 개별 문서 변경 후 정합성 유지 필요

### 문서 간 연결 관계
```
Document_Master_List.md (67건 문서 상태)
    ↕ 동기화
CAR_목록_시정조치계획.md (17건 부적합 → 절차서 작성과 연동)
    ↕ 연동
내부심사_계획서.md (심사 대상 = 작성 완료된 절차서)
    ↕ 입력
경영검토_보고서_초안.md (내부심사 결과 반영)
    ↕ 통합
AS9100D_마스터_대시보드.md (전체 현황 집계)
```
